## <a name="unclassified-streams"></a><span data-ttu-id="9ee77-244">Fluxos não classificados</span><span class="sxs-lookup"><span data-stu-id="9ee77-244">Unclassified Streams</span></span>

<span data-ttu-id="9ee77-245">No CQD, um  fluxo é marcado como Não classificado quando a conectividade ICE (Estabelecimento de Conectividade Interativa) falha ou quando todas as métricas necessárias para calcular a classificação de fluxo não são relatadas.</span><span class="sxs-lookup"><span data-stu-id="9ee77-245">In CQD, a stream is marked _Unclassified_ when Interactive Connectivity Establishment (ICE) connectivity fails or when all the metrics required to compute the stream classification are not reported.</span></span>

<span data-ttu-id="9ee77-246">Para verificar se há falhas de conectividade ICE, examine as dimensões "Ice de primeira conectividade" e "Ice de segunda conectividade" em busca de um valor "FALHOU".</span><span class="sxs-lookup"><span data-stu-id="9ee77-246">To check for ICE connectivity failures, examine the dimensions "First Connectivity Ice" and "Second Connectivity Ice" for a "FAILED" value.</span></span> <span data-ttu-id="9ee77-247">Se um dos valores indicar uma falha, o fluxo será marcado como _Unclassified_.</span><span class="sxs-lookup"><span data-stu-id="9ee77-247">If either value indicates a failure, the stream is marked as _Unclassified_.</span></span>

<span data-ttu-id="9ee77-248">Se a conectividade ICE  tiver sido bem-sucedida para um fluxo não classificado, o fluxo provavelmente será considerado _Não classificado_ porque as métricas de fluxo de teclas não foram relatadas.</span><span class="sxs-lookup"><span data-stu-id="9ee77-248">If ICE connectivity succeeded for an _Unclassified_ stream, the stream is likely considered _Unclassified_ because key stream metrics were not reported.</span></span> <span data-ttu-id="9ee77-249">Existem algumas razões pelas quais essas métricas podem não ser relatadas:</span><span class="sxs-lookup"><span data-stu-id="9ee77-249">There are a few reasons these metrics may not be reported:</span></span>

- <span data-ttu-id="9ee77-250">**Relatórios de QoE não foram recebidos** — As métricas usadas para classificação são relatadas em um relatório de QoE enviado no final de uma chamada.</span><span class="sxs-lookup"><span data-stu-id="9ee77-250">**QoE reports were not received** — The metrics used for classification are reported in a QoE report sent at the end of a call.</span></span> <span data-ttu-id="9ee77-251">Se esse relatório não for produzido (por exemplo, porque alguns pontos de extremidade de terceiros podem não enviar QoE) ou não puder ser enviado (por exemplo, devido a uma interrupção de rede), o CQD não poderá classificar o fluxo.</span><span class="sxs-lookup"><span data-stu-id="9ee77-251">If this report is not produced (for example, because some third-party endpoints may not send QoE) or could not be sent (for example, because of a network outage), CQD is unable to classify the stream.</span></span>

- <span data-ttu-id="9ee77-255">**Chamadas curtas** — chamadas curtas podem não ter atividade de mídia suficiente para calcular métricas de fluxo de chaves.</span><span class="sxs-lookup"><span data-stu-id="9ee77-255">**Short calls** — Short calls may not have enough media activity to compute key stream metrics.</span></span> <span data-ttu-id="9ee77-256">Sem essas métricas, o CQD não consegue classificar o fluxo.</span><span class="sxs-lookup"><span data-stu-id="9ee77-256">Without these metrics, CQD is unable to classify the stream.</span></span>

  - <span data-ttu-id="9ee77-261">Um cenário comum de baixa utilização de pacotes ocorre quando um participante ins participa de uma reunião para ouvir o apresentador, mas nunca fala (o microfone é mudo para a maioria da chamada).</span><span class="sxs-lookup"><span data-stu-id="9ee77-261">A common low packet utilization scenario occurs when an attendee joins a meeting to listen to the presenter, but never speaks (the microphone is muted for most of the call).</span></span> <span data-ttu-id="9ee77-262">Aqui, o fluxo de áudio de entrada para o cliente tem alta utilização de pacotes enquanto o fluxo de áudio de saída do cliente tem pouca ou nenhuma utilização de pacote.</span><span class="sxs-lookup"><span data-stu-id="9ee77-262">Here, the audio stream inbound to the client has high packet utilization while the audio stream outbound from the client has little to no packet utilization.</span></span> <span data-ttu-id="9ee77-263">A duração do fluxo pode ser de uma hora ou mais, mas _a_ utilização do pacote no fluxo do cliente para o servidor é baixa desde que o microfone foi silenciado e um fluxo não classificado resulta.</span><span class="sxs-lookup"><span data-stu-id="9ee77-263">The duration of the stream may be an hour or longer but the packet utilization on the stream from the client to the server is low since the microphone was muted, and an _Unclassified_ stream results.</span></span>
