# <a name="getting-started-with-microsoft-teams-rooms-premium-managed-service"></a>Introdução ao Salas do Microsoft Teams Premium gerenciado pelo usuário

O Salas do Microsoft Teams Premium gerenciados reduz a complexidade para administradores de Salas do Teams fornecendo operações de sala aprimoradas que aprimoram a produtividade da reunião na sala.

## <a name="obtain-and-assign-a-license"></a>Obter e atribuir uma licença

Salas do Microsoft Teams de recursos exigem uma Salas do Microsoft Teams Premium para trabalhar com o serviço Salas do Microsoft Teams Premium gerenciado pelo usuário.

Para atribuir uma licença, execute as seguintes etapas:

1. Faça logon [Centro de administração do Microsoft 365](https://admin.microsoft.com) como administrador.

    > [!NOTE]
    > Às vezes, esse tipo de administrador é conhecido como **administrador global**. Consulte Visão geral do [Centro de administração do Microsoft 365](/microsoft-365/business-video/admin-center-overview).

2. Escolha **Licenças > cobrança no** painel de navegação esquerdo.
3. Selecione a licença **Salas do Teams premium** apropriada para o uso pretendido (licenças pagas e de avaliação estão disponíveis). Consulte [soluções de licenciamento para dispositivos de comunicações](rooms-licensing.md) compartilhadas para obter uma comparação entre Salas do Microsoft Teams Padrão e Premium SKUs.
4. Na página **Contas de cobrança** , selecione **Atribuir licenças**. A **página Atribuir licenças aos** usuários é exibida.
5. Insira as contas de **Salas do Microsoft Teams** existentes para **Salas do Teams Premium** licença. Para obter mais informações sobre como configurar contas novas ou Salas do Microsoft Teams existentes, consulte Criar contas de recursos para salas e dispositivos [Teams compartilhados](with-office-365.md).
6. Depois que o envio da conta for concluído, selecione **Atribuir** e **, em seguida, Fechar**. As **Salas do Microsoft Teams Premium** foram atribuídas **a Salas do Microsoft Teams contas**.

Agora você está pronto para registrar as contas de recursos e os dispositivos Salas do Microsoft Teams certificados associados no serviço Salas do Microsoft Teams Premium gerenciado pelo usuário. Para obter mais informações, [consulte Registrar um dispositivo no Salas do Microsoft Teams Premium gerenciado](enrolling-mtrp-managed-service.md).
