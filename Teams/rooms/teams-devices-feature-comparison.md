# <a name="teams-devices-for-shared-spaces-feature-comparison-between-windows-and-android"></a>Teams dispositivos para comparação de recursos de espaços compartilhados entre Windows e Android 
Salas do Microsoft Teams experiências são projetadas com os clientes em mente. Cada cliente tem necessidades exclusivas, e é por isso que a Microsoft oferece uma série de opções, incluindo Salas do Teams no Windows e no Android. 

A funcionalidade principal está disponível em ambas as plataformas para que os usuários finais sempre possam ter ótimas experiências de áudio e vídeo, usar uma participação de um toque para participar de reuniões e acessar recursos inclusivos, como legendas ao vivo e levantar a mão. 

À medida que a Microsoft traz novos recursos para Teams e para Salas do Teams, nos esforçamos para trazê-los para todas as plataformas, mas os recursos podem ser rolados em momentos diferentes devido a vários fatores, incluindo coisas como viabilidade técnica e feedback do cliente para cada plataforma. Isso significa que você pode ver alguns recursos sendo Windows antes que eles venham para a plataforma Android. 

Para ajudar a orientá-lo sobre quais recursos estão disponíveis em diferentes plataformas, você pode consultar o gráfico abaixo, que será atualizado periodicamente. Nem todos Salas do Teams recursos serão incluídos nesta lista, portanto, continue a consultar o [roteiro](https://www.microsoft.com/en-us/microsoft-365/roadmap) Microsoft 365 e representantes de vendas da Microsoft para obter detalhes adicionais.
