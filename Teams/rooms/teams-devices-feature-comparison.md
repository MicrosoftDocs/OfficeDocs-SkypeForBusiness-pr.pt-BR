# <a name="teams-devices-for-shared-spaces-feature-comparison-between-windows-and-android"></a>Teams para comparação de recursos de espaços compartilhados entre Windows e Android 
Salas do Microsoft Teams experiências são projetadas com os clientes em mente. Cada cliente tem necessidades exclusivas, por isso a Microsoft oferece uma variedade de opções, incluindo Salas do Teams no Windows e Android. 

A funcionalidade principal está disponível em ambas as plataformas para que os usuários finais sempre possam ter ótimas experiências de áudio e vídeo, usar uma junção de um toque para ingressar em reuniões e acessar recursos inclusivos, como legendas ao vivo e levantar a mão. 

À medida que a Microsoft traz novos recursos para o Teams e para o Salas do Teams, nos esforçamos para trazê-los para todas as plataformas, mas os recursos podem ser distribuídos em momentos diferentes devido a vários fatores, incluindo itens como viabilidade técnica e comentários dos clientes para cada plataforma. Isso significa que você pode ver alguns recursos sendo implantados Windows antes que eles venham para a Android plataforma. 

Para ajudar a orientar você sobre quais recursos estão disponíveis em diferentes plataformas, você pode consultar o gráfico abaixo, que será atualizado periodicamente. Nem todos Salas do Teams recursos serão incluídos nesta lista, portanto, continue [a](https://www.microsoft.com/en-us/microsoft-365/roadmap) consultar o roteiro Microsoft 365 e os representantes de vendas da Microsoft para obter detalhes adicionais.
