# <a name="phone-number-management-for-canada"></a>Gerenciamento de números de telefone para o Canadá

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone no Canadá para o Microsoft Teams.

Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).

| **O que você deseja fazer?** | **Execute estas etapas e baixe os formulários corretos** |

|Obter números de telefone transferidos para nós.  <br/> | Baixe um destes formulários: <br/> [LOA (Carta de Autorização) para o Canadá (números de usuário e serviço) (v.3.2) (en-ca)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.3.2)-(en-ca).pdf?raw=true) <br/> [Carta de Autorização (LOA) para o Canadá (números de usuário e de serviço) (v.1.0)(fr-ca)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.1.0)-(fr-ca).pdf?raw=true) <br/> [LOA (Carta de Autorização) para o Canadá (números de chamada gratuita) (v.3.2) (en-ca)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.3.2)-(en-ca).pdf?raw=true) <br/>[Carta de Autorização (LOA) para o Canadá (números de chamada gratuita) (v.1.1)(fr-ca)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/Letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.1.1)-(fr-ca).pdf?raw=true)<br/> <br>**Observação:** Se você estiver tentando obter mais de 999 números de telefone de assinante ou usuário, precisará preencher uma LOA (Carta de Autorização) e enviá-la para nós. Para configurar um PIN, vá [aqui](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier).  <br/> |
