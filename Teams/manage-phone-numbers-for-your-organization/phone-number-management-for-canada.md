# <a name="phone-number-management-for-canada"></a>Gerenciamento de números de telefone para o Canadá

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone no Canadá para o Microsoft Teams.

Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).

|Obter números de telefone transferidos para nós.  <br/> | Baixe um destes formulários: <br/> [Carta de Autorização (LOA) para o Canadá (números de usuário e serviço) (v.3.2) (en-ca)](https://download.microsoft.com/download/7/5/2/7525e2cb-d496-4133-87a9-bac3ee4a3216/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.3.2)-(en-ca).pdf) <br/> [Carta de Autorização (LOA) para o Canadá (números de usuário e de serviço) (v.1.0)(fr-ca)](https://download.microsoft.com/download/b/1/a/b1ada279-0711-4144-bc6c-d0f5541c27ef/letter-of-authorization-(loa)-for-canada-(user-and-service-numbers)-(v.1.0)-(fr-ca).pdf)<br/> [Carta de Autorização (LOA) para o Canadá (números gratuitos) (v.3.2) (en-ca)](https://download.microsoft.com/download/1/7/1/1712dc2a-6b36-4b77-8588-fc33e80f2228/letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.3.2)-(en-ca).pdf ) <br/>[Carta de Autorização (LOA) para o Canadá (números de chamada gratuita) (v.1.1)(fr-ca)](https://download.microsoft.com/download/1/7/1/1712dc2a-6b36-4b77-8588-fc33e80f2228/letter-of-authorization-(loa)-for-canada-(toll-free-numbers)-(v.3.2)-(en-ca).pdf)<br/> <br>**Observação:** Se você estiver tentando obter mais de 999 números de telefone de assinante ou usuário, precisará preencher uma Carta de Autorização (LOA) e enviá-la para nós.
