# <a name="phone-number-management-for-japan"></a>Gerenciamento de número de telefone para Japão

> [!IMPORTANT]
> No momento, o uso do SoftBank para chamadas no Japão está disponível apenas para os pioneiros (EA).

No Japão, fizemos a parceria com a SoftBank para compra e provisionamento de planos de chamadas para clientes com um endereço de cobrança no Japão. O SoftBank venderá seus planos de chamadas, fornecerá números de usuário para o Japão e também fornecerá o suporte para o gerenciamento de número de telefone, mas os números de serviço e licenças para videoconferências, atendedores automáticos e filas de chamadas.  ainda será vendido e gerenciado pelo Office 365.

Para obter mais informações sobre as [chamadas Softbank para o Office 365](https://aka.ms/SoftBankVoicePlan).

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone no Japão para Microsoft Teams.

Para obter mais informações, consulte [gerenciar números de telefone para sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Execute estas etapas e baixe os formulários corretos**|
|:-----|:-----|
|Obtenha novos números de telefone de usuário para planos de chamadas no Japão.   <br/> |Entre em contato com [Softbank chamadas para o Office 365](https://aka.ms/SoftBankVoicePlan) para obter novos números de telefone para seus usuários. Em seguida, eles adicionarão esses números à sua organização para que você possa atribuí-los aos usuários. <br/>
|Obtenha novos números de telefone de serviço para conferências de áudio, atendedores automáticos e filas de chamadas no Japão.  <br/> |[Entre em contato conosco](mailto:ptnapac@microsoft.com).|
|Obter números de telefone transferidos para o Office 365 para usuários do Japão.  <br/> |Entre em contato com [Softbank chamada para o Office 365](https://aka.ms/SoftBankVoicePlan) para transferir números de telefone. Em seguida, eles adicionarão esses números à sua organização.  <br/> |
|Obter números de telefone transferidos para o Office 365 para conferências de áudio, atendedores automáticos e filas de chamadas no Japão.  |[Entre em contato conosco](mailto:ptnapac@microsoft.com) |
|Alterar o endereçamento de emergência atribuído aos usuários que têm um número de telefone de um plano de chamadas. |Entre em contato com [Softbank chamada para o Office 365](https://aka.ms/SoftBankVoicePlan) para fazer alterações de endereço de emergência. Eles podem atualizar essas informações para você.|
|Alterar um número de usuário para um número de serviço. |Não tem suporte no momento ao usar chamadas SoftBank para o Office 365.
