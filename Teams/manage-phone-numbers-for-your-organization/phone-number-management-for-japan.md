# <a name="phone-number-management-for-japan"></a>Gerenciamento de números de telefone para o Japão

No Japão, fizemos uma parceria com a SoftBank para a compra e o provisionamento de Planos de Chamada para clientes com um endereço de cobrança no Japão. O SoftBank venderá seus Planos de Chamada, fornecerá números de usuário para o Japão e fornecerá suporte para o gerenciamento de números de telefone. No entanto, números de serviço e licenças para audioconferência, atendimento automático e filas de chamada ainda serão vendidos e gerenciados pelo Microsoft 365 e pelo Office 365.

Para obter mais informações sobre a [Chamada do SoftBank.](https://aka.ms/SoftBankVoicePlan)

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone no Japão para o Microsoft Teams.

Para obter mais informações, consulte [Gerenciar números de telefone para sua organização.](manage-phone-numbers-for-your-organization.md)
  
|**O que você deseja fazer?**|**Execute estas etapas e baixe os formulários corretos**|
|:-----|:-----|
|Obter novos números de telefone de usuário para Planos de Chamada no Japão.|Entre em contato [com a Chamada Do SoftBank](https://aka.ms/SoftBankVoicePlan) para ver novos números de telefone para seus usuários. Em seguida, eles adicionarão esses números à sua organização para que você possa atribuí-los aos usuários. <br/>
|Obter novos números de telefone de serviço para audioconferência, atender automaticamente e filas de chamadas no Japão.|[Entre em contato conosco.](mailto:ptnapac@microsoft.com)|
|Obter números de telefone transferidos para o Microsoft 365 ou o Office 365 para **usuários** no Japão.|Entre em contato com [a Chamada Do SoftBank](https://aka.ms/SoftBankVoicePlan) para transferir números de telefone do usuário. Em seguida, eles adicionarão esses números à sua organização.  <br/> |
|Obter números de telefone transferidos para o Microsoft 365 ou o Office 365 para **filas de chamadas** no Japão.|Entre em contato com [a Chamada Do SoftBank](https://aka.ms/SoftBankVoicePlan) para transferir números de telefone do serviço. Em seguida, eles adicionarão esses números à sua organização.|
|Altere o endereço de emergência atribuído aos usuários que têm um número de telefone do Plano de Chamada.|Entre em contato com [a Chamada Do SoftBank](https://aka.ms/SoftBankVoicePlan) para fazer alterações de endereço de emergência. Eles podem atualizar essas informações para você.|
|Altere um número de usuário para um número de serviço. |No momento, não há suporte ao usar a Chamada Do SoftBank.|
|Fazer atualizações de serviço de diretório para números de fila de chamada ou de atendimento automático|Para adicionar ou alterar os números da fila de chamada ou do atendimento automático, entre em contato com o service [desk PSTN.](contact-pstn-service-desk.md) |
