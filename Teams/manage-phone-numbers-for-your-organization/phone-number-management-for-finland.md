# <a name="phone-number-management-for-finland"></a>Telefone gerenciamento de números para a Finlândia

[!INCLUDE [preview-feature](../includes/preview-feature.md)]

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone na Finlândia para Microsoft Teams.
  
Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Baixar os formulários corretos**|
|:-----|:-----|
|Obter novos números de telefone de usuário ou serviço que não estão listados no Microsoft Teams de administração.<br/> |Na Finlândia, você pode obter números de telefone para sua organização usando o Microsoft Teams de administração. Você também pode enviar uma nova solicitação de número de telefone baixando o formulário e contate o [Service Desk TNS para obter instruções.](contact-tns-service-desk.md)<br/><br/>Nova solicitação de número de telefone para a Finlândia (v.1.0) (en-fi)]().<br/><br/> Para que os números de telefone possam ser atribuídos a você, você deve baixar o arquivo Conheça seu cliente para a Finlândia [(v.1.0) (fi-FI)](https://download.microsoft.com/download/d/5/3/d5393c6b-eb6e-45e3-975e-d8a501571366/new-phone-number-request-for-finland-(v1.0)-(fi-FI).pdf) Conheça seu formulário de cliente, conclua-o na íntegra e envie-o para nós.<br/><br/>Os novos pedidos podem levar de duas a três semanas para preencher esse mercado.   |
|Obter números de telefone transferidos para nós.| Baixe este formulário. <br/>[Carta de Autorização (LOA) para a Finlândia (todos os números) (v.1.0) (en-fi)](https://download.microsoft.com/download/a/6/8/a68d6c80-daf5-4d40-ba6e-d0f99db1041b/letter-of-authorization-(loa)-for-finland-(all-numbers)-(v1.0)-(en-fi).pdf) <br/> <br/>Preencha-o e [entre em contato com o Service Desk TNS para obter instruções](contact-tns-service-desk.md). <br/><br/>A transferência de números pode levar de duas a três semanas para ser concluída neste mercado. <br/><br/> Esses números não podem ser usados para conferência no momento.   |
|Publicar/não publicar números de telefone no diretório telefônico nacional.  <br/> |[Entre em contato com o Service Desk TNS](contact-tns-service-desk.md) para obter instruções sobre a lista de números de telefone que você deseja publicar/não publicar. <br/> |
|Altere o número de um usuário para um número de serviço.  <br/> |[Entre em contato com o Service Desk TNS](contact-tns-service-desk.md) para obter instruções sobre a lista de números de usuário que você deseja converter em números de serviço. <br/> |
|Fazer atualizações de serviço de diretório para chamar fila ou números de atendimento automático|Para adicionar ou alterar números de fila de chamada ou atendimento automático, entre em contato com o service [desk TNS para obter instruções.](contact-tns-service-desk.md) |
