# <a name="phone-number-management-for-australia"></a>Gerenciamento de números de telefone para a Austrália

Na Austrália, fizemos uma parceria com a Telstra para a compra e o provisionamento de Planos de Chamada para clientes com um endereço de cobrança na Austrália. A Telstra venderá seus Planos de Chamada e fornecerá números de usuário australianos. A Telstra fornece suporte para o gerenciamento de números de telefone para os números de usuário e de serviço para os atendimentos automáticos e filas de chamadas. No entanto, os números de serviço e as licenças de Audioconferência ainda serão vendidos e gerenciados pelo Microsoft 365 e pelo Office 365.

Para obter mais informações sobre a [Chamada Telstra.](https://aka.ms/TelstraVoicePlan)

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone na Austrália para o Microsoft Teams.

Para obter mais informações, consulte [Gerenciar números de telefone para sua organização.](manage-phone-numbers-for-your-organization.md)
