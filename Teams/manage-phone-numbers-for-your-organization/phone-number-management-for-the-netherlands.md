# <a name="phone-number-management-for-the-netherlands"></a>Gerenciamento de números de telefone para os Países Baixos

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone na Holanda para o Microsoft Teams.
  
Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Baixar os formulários corretos**|
|:-----|:-----|
|Obtenha novos números de telefone de usuário ou serviço que não estão listados no centro de administração do Microsoft Teams.   <br/> | Baixe um desses formulários.</br> [Nova solicitação de número de telefone para os Países Baixos (v.5.0) (nl-nl)](https://download.microsoft.com/download/d/5/9/d5937e19-e9f7-4a08-9631-e5d456068726/new-phone-number-request-for-the-netherlands-(v.5.0)-(nl-nl).pdf) <br/>  [Nova solicitação de número de telefone para os Países Baixos (v.5.0) (en-us)](https://download.microsoft.com/download/9/f/4/9f42be90-1977-4ba7-887a-3772757e12fa/new-phone-number-request-for-the-netherlands-(v.5.0)-(en-us).pdf)<br/><br/>  Preencha-o e [entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md).  <br/> |
|Faça com que os números de telefone sejam transferidos para nós.  <br/> |Baixe um destes formulários: <br/> [LOA (Carta de Autorização) para os Países Baixos (todos os números) (v.6.0) (nl-nl)](https://download.microsoft.com/download/c/7/0/c70c4dcd-f23a-4935-aa31-fc13aba33ee5/LOA-NL-GEO-NL.pdf) <br/> <br/> [Carta de Autorização (LOA) para os Países Baixos (todos os números) (v.6.0) (en-en)](https://download.microsoft.com/download/c/7/0/c70c4dcd-f23a-4935-aa31-fc13aba33ee5/LOA-NL-GEO-EN.pdf) <br/> <br/> [LOA (Carta de Autorização de Término Antecipado) para os Países Baixos (todos os números) (v.6.0) (nl-nl)](https://download.microsoft.com/download/c/7/0/c70c4dcd-f23a-4935-aa31-fc13aba33ee5/LOA-NL-ERT-NL.pdf) <br/> <br/> [LOA (Carta de Autorização de Término Antecipado) para os Países Baixos (todos os números) (v.6.0) (en-en)](https://download.microsoft.com/download/c/7/0/c70c4dcd-f23a-4935-aa31-fc13aba33ee5/LOA-NL-ERT-EN.pdf) <br/><br/>  Preencha-o e [entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md).  <br/> |
|Publicar/publicar números de telefone no diretório telefônico nacional.  <br/> |[Entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md) sobre a lista de números de telefone que você deseja publicar/desmarque. <br/> |
|Altere o número de um usuário para um número de serviço.  <br/> |[Entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md) sobre a lista de números de usuário que você deseja converter em números de serviço. <br/> |
