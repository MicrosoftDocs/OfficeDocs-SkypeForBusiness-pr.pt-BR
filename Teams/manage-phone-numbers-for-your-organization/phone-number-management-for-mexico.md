# <a name="phone-number-management-for-mexico"></a>Gerenciamento de números de telefone para o México

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone no México para o Microsoft Teams.
  
Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Baixe os formulários corretos e os envie para nós**|
|:-----|:-----|
|Obtenha novos números de telefone de usuário ou serviço que não estão listados no Centro de administração do Microsoft Teams.   <br/> |Baixe um desses formulários. <br/>  [Nova solicitação de número de telefone para o México (es-mx)](https://download.microsoft.com/download/0/0/8/008e6ad0-1b1c-4b2c-949d-9b885f6f3650/new-phone-number-request-for-mexico-(v.1.0)-(es-mx).pdf) <br/>  [Nova solicitação de número de telefone para o México (en-us)](https://download.microsoft.com/download/0/2/6/02657735-678d-44cb-b2ed-36a66407150e/new-phone-number-request-for-mexico-(v.1.0)-(en-us).pdf)<br/>  <br/>Preencha-o e [entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md).<br/><br/> Você também deve preencher um contrato de três partes, assinar e entrar em contato com o [TNS Service Desk para obter instruções](contact-tns-service-desk.md). Esse formulário será enviado para você pela nossa equipe do TNS Service Desk depois que você fazer um pedido para obter novos números de telefone. <br/><br/>|
|Obter números de telefone transferidos para nós.  <br/> | Baixe este formulário. <br/>[Carta de Autorização (LOA) para o México (es-mx)](https://download.microsoft.com/download/a/7/b/a7bc578f-218b-46f6-a192-0df5b6abc653/LOA-MX-GEO-ES.pdf) <br/> <br/>Preencha-o e [entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md). <br/><br/> Você também deve preencher um contrato de três partes, assinar e entrar em contato com o [TNS Service Desk para obter instruções](contact-tns-service-desk.md). Esse formulário será enviado para você pela nossa equipe de suporte ao serviço TNS depois que você fazer um pedido para transferir números de telefone.<br/><br/>|
|Publicar/cancelar a publicação de números de telefone no diretório de telefone nacional.  <br/> |[Entre em contato com o Service Desk do TNS](contact-tns-service-desk.md) para obter instruções sobre a lista de números de telefone que você deseja publicar/cancelar a publicação.      <br/> |
|Altere o número de um usuário para um número de serviço.  <br/> |[Entre em contato com o Service Desk do TNS](contact-tns-service-desk.md) para obter instruções sobre a lista de números de usuário que você deseja converter em números de serviço. <br/> |
|Faça atualizações do serviço de diretório para chamar números de fila ou atendedor automático. |Para adicionar ou alterar os números da fila de chamadas ou do atendedor automático, entre em contato [com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md). |
