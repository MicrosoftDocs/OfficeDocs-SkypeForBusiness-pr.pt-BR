# <a name="phone-number-management-for-spain"></a>Gerenciamento de números de telefone para a Espanha

Use esta tabela para descobrir como obter e gerenciar números de telefone do Skype for Business Online e do Microsoft Teams para a Espanha. 
  
Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Baixar os formulários corretos**|
|:-----|:-----|
|Obtenha o novo usuário ou números de telefone de serviço que não estão listados no Skype para centro de administração de negócios.   <br/> | Baixe este formulário.</br> [Nova solicitação de número de telefone da Espanha (v.4.0)(es-es)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-spain-(v.4.0)-(es-es).pdf?raw=true) </br> [Nova solicitação de número de telefone da Espanha (v.4.0)(en-us)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-spain-(v.4.0)-(en-us).pdf?raw=true) <br/>    Preencha o formulário e [o envie para nós](mailto:ptneu@microsoft.com).  <br/> |
|Obtenha números de telefone transferidos para nós.  <br/> |Baixe o formulário [Carta de Autorização (LOA) para a Espanha (todos os números) (v.1.0)(es-es)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-spain-(all-numbers)-(v.1.0)-(es-es).pdf?raw=true), preencha-o e [o envie para nós](mailto:ptneu@microsoft.com). <br/> |
|Publicar/cancelar a publicação de números de telefone no diretório nacional de telefone.  <br/> |[Nos envie](mailto:ptneu@microsoft.com) a lista de números de telefone que você deseja publicar/cancelar a publicação. <br/> |
|Altere o número de um usuário para um número de serviço.  <br/> |[Nos envie](mailto:ptneu@microsoft.com) a lista de números de usuário que você deseja converter em números de serviço. <br/> |
