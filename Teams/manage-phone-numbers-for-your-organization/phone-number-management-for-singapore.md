# <a name="phone-number-management-for-singapore"></a>Gerenciamento de números de telefone para Cingapura

[!INCLUDE [preview-feature](../includes/preview-feature.md)]

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone em Cingapura para o Microsoft Teams.
  
Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Baixar os formulários corretos**|
|:-----|:-----|
|Obtenha novos números de telefone de usuário ou serviço que não estão listados no centro de administração do Microsoft Teams. |Em Cingapura, você pode obter números de telefone para sua organização usando o centro de administração do Microsoft Teams. Você também pode enviar uma nova solicitação de número de telefone baixando o formulário e [entrando em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md).<br/><br/>Antes que os números de telefone possam ser atribuídos a você, você deve baixar um dos seguintes formulários de cliente:<ul><li>[Conheça seu cliente Singapura - Inglês en-US](https://download.microsoft.com/download/e/9/5/e959053a-4b35-49f9-a23e-7085575c46f2/KYC-SG-EN.pdf)</li><li>[Conheça seu cliente Cingapura - Chinês v1.0 zn-CN](https://download.microsoft.com/download/1/4/7/147324ac-06a5-4c7b-a38b-7efa0166fdf7/know-your-customer-for-singapore-(v1.0)-(zh-CN).pdf)</li><li>[Conheça seu cliente para Cingapura - Tamil v1.0 ta-IN](https://download.microsoft.com/download/b/4/8/b48457c1-cef1-4bc7-ae44-71c5302e5eb2/know-your-customer-for-singapore-(v1.0)-(ta-IN).pdf)</li><li>[Conheça seu cliente Singapura - Malay v1.0 ms-MY](https://download.microsoft.com/download/f/5/9/f597a754-9478-4d5b-88af-a2a2fe5bbd62/know-your-customer-for-singapore-(v1.0)-(ms-MY).pdf)</li></ul>Conclua-o na íntegra e [entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md).<br/><br/>Novos pedidos podem levar de duas a três semanas para preencher este mercado.  |
|Faça com que os números de telefone sejam transferidos para nós.  <br/> | Baixe este formulário. <br/>[LOA (Carta de Autorização) para Cingapura (Números de Usuário e Serviço)(en-sg)v2](<https://download.microsoft.com/download/e/9/5/e959053a-4b35-49f9-a23e-7085575c46f2/LOA-SG-GEO-EN.pdf>)<br/> <br/>Preencha-o e [entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md). <br/><br/>A transferência de números pode levar de uma a seis semanas para ser concluída neste mercado, dependendo da complexidade da porta. <br/><br/>Os números geográficos podem ser portados como tipo de número de serviço (para filas de chamada & Atendimento Automático, pois esses números não podem ser usados para conferência neste momento) ou tipo de número de assinante. Observe que o serviço de portabilidade para número gratuito (0800) não tem suporte no momento. |
|Publicar/publicar números de telefone no diretório telefônico nacional.  <br/> |[Entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md) sobre a lista de números de telefone que você deseja publicar/desmarque. <br/> |
|Altere o número de um usuário para um número de serviço.  <br/> |[Entre em contato com o Service Desk do TNS para obter instruções](contact-tns-service-desk.md) sobre a lista de números de usuário que você deseja converter em números de serviço. <br/> |
|Fazer atualizações de serviço de diretório para chamar números de fila ou de atendimento automático|Para adicionar ou alterar a fila de chamadas ou números de atendimento automático, entre em contato com [o Service Desk do TNS para obter instruções](contact-tns-service-desk.md).|
