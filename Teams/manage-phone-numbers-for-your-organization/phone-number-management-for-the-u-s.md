# <a name="phone-number-management-for-the-united-states"></a>Gerenciamento de números de telefone para os Estados Unidos

Use esta tabela para encontrar informações sobre como obter e gerenciar números de telefone nos Estados Unidos (incluindo Porto Rico) para o Microsoft Teams.

Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).
  
| O que você deseja fazer? | Execute estas etapas e baixe os formulários corretos |
|:-----|:-----|
|Obter novos números de telefone de usuário ou serviço que não estão listados no Centro de administração do Microsoft Teams.  <br/> |Baixe este formulário.<br/>  [Nova solicitação de número de telefone para os Estados Unidos (v.5.0) (en-us)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-the-united-states-(v.5.0)-(en-us).pdf?raw=true) <br/><br/> Preencha o formulário e [o envie para nós](mailto:gcstnmsd@microsoft.com). |
|Obter números de telefone transferidos para nós.  <br/> | Baixe um destes formulários: <br/> [Carta de Autorização (LOA) para os Estados Unidos (números de usuário e serviço) (v.3.3)(en-us)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(user-and-service-numbers)-(v.3.3)-(en-us).pdf?raw=true)<br/>[Carta de Autorização (LOA) para os Estados Unidos (números de tarifa gratuita) (v.3.3)(en-us)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(toll-free-numbers)-(v.3.3)-(en-us).pdf?raw=true)<br/><br/>**Observação** Se você estiver tentando obter mais de 999 números de telefone de assinante ou usuário, precisará preencher uma LOA (Carta de Autorização) e enviá-la para nós. Consulte [Transferir números de telefone para o Teams](../phone-number-calling-plans/transfer-phone-numbers-to-teams.md) para obter mais informações.          <br/>  <br/>   Preencha o formulário e [o envie para nós](mailto:ptn@microsoft.com).  <br/> |
|Alterar o número de um usuário para um número de serviço ou um número de serviço para o número de um usuário  <br/> |[Envie-nos](mailto:ptn@microsoft.com) seu PIN e a lista de números que você deseja converter. Para configurar um PIN, consulte [aqui.](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier)  <br/> |
|Transferir números de um Microsoft 365 ou do Office 365 para outro.  <br/> |[Envie-nos](mailto:ptn@microsoft.com) a lista de números de telefone de usuário (assinantes) que você deseja transferir junto com o PIN da organização de origem cujos números você deseja transferir. Para configurar um PIN, consulte [aqui.](../phone-number-calling-plans/port-order-overview.md#can-i-port-out-my-numbers-from-teams-to-a-different-phone-service-provider-or-carrier)  <br/> |
|Fazer atualizações de serviço de diretório para números de fila de chamada ou de atendimento automático|Para adicionar ou alterar os números da fila de chamada ou do atendimento automático, entre em contato com o service [desk PSTN.](contact-pstn-service-desk.md) |
