# <a name="whats-the-status-of-your-port-orders"></a>Qual é o status dos seus pedidos de portabilidade?

Para ver o status do pedido de portabilidade, no painel de navegação esquerdo do Centro de administração do Microsoft Teams,  >  vá para > pedidos de Porta de Voz e clique em Histórico **de pedidos**. Cada status do pedido de portabilidade é listado na **coluna Status** . Veja [quanto tempo leva para os números de portabilidade](../phone-number-calling-plans/port-order-overview.md#how-long-does-it-take-to-port-numbers) para saber mais sobre o processo de pedido. 

A tabela a seguir lista os status do pedido de portabilidade, bem como as ações que você pode executar, se necessário.

|**Status**|**Você pode ver o pedido?**|**Você pode editar o pedido?**|**Você pode cancelar o pedido?**|**Você pode excluir o pedido?**|**Descrição**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Processamento** <br/> |Sim  <br/> |Não  <br/> |Sim  <br/> |Não  <br/> |O administrador criou o pedido e ele foi recebido pela Microsoft.  <br/> |
|**Contatando a operadora** <br/> |Sim  <br/> |Não  <br/> |Sim  <br/> |Não  <br/> |O pedido foi recebido e aprovado pela Microsoft e estamos trabalhando com a operadora perdedora para aprová-lo.  <br/> |
|**Transferência aprovada** <br/> |Sim  <br/> |Não  <br/> |Sim  <br/> |Não  <br/> |O pedido foi aceito pela transportadora perdedora e a data do COMPROMISSO (FOC) foi definida.  <br/> |
|**Transferência pendente** <br/> |Sim  <br/> |Não  <br/> |Não  <br/> |Não  <br/> |A transferência está a menos de 24 horas de distância, portanto, o pedido não pode mais ser editado ou cancelado.  <br/> |
|**Error** <br/> |Não  <br/> |Sim  <br/> |Sim  <br/> |Sim (neste momento, você não poderá excluir a ordem de portabilidade se houver um erro. O pedido de portabilidade precisa ser criado novamente ou você precisa entrar em contato com o [Service Desk do TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).  <br/> |A transportadora perdedora rejeitou a ordem.  <br/> |
|**Concluído** <br/> |Sim  <br/> |Não  <br/> |Não  <br/> |Não  <br/> |Os números foram transferidos com êxito.  <br/> |
|**Cancelado** <br/> |Não  <br/> |Sim  <br/> |Não  <br/> |Não  <br/> |O administrador cancelou o pedido.  <br/> |

Para obter instruções passo a passo completas, consulte [Transferir números de telefone para o Teams](transfer-phone-numbers-to-teams.md).

Se precisar de ajuda ou se precisar obter mais números de telefone, entre em contato com o [Service Desk do TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).
