# <a name="manage-the-whiteboard-in-microsoft-teams"></a>Gerenciar o quadro de Microsoft Teams

Microsoft Whiteboard é uma tela digital de forma livre onde as pessoas, o conteúdo e as ideias se reúnem. A integração do quadro de Microsoft Teams reuniões é alimentada pelo aplicativo Web do Quadro de Teams, que permite que Teams participantes da reunião desenhem, esboçam e escrevam juntos em uma tela digital compartilhada.

Os usuários podem compartilhar um quadro de opções para torná-lo disponível para todos os participantes em uma Teams reunião. Esse mesmo quadro de comunicação está disponível simultaneamente em todos os aplicativos de quadro de Windows 10, iOS e aplicativo Web.

Para ativar ou desativar o aplicativo whiteboard, consulte [Enable Microsoft Whiteboard for your organization](https://support.office.com/article/1caaa2e2-5c18-4bdf-b878-2d98f1da4b24). Lembre-se de que essa configuração habilita ou desabilita o Quadro de Trabalho para toda a sua organização e não apenas para Teams.

## <a name="whiteboard-on-surface-hub"></a>Quadro de Surface Hub

Antes de tentar usar Microsoft Whiteboard, certifique-se de que o aplicativo whiteboard está instalado em seu Surface Hub dispositivo. Se o quadro de Surface Hub estiver instalado, vá para o aplicativo Microsoft Store e Microsoft Whiteboard [.](https://www.microsoft.com/p/microsoft-whiteboard/9mspc6mp8fm4?activetab=pivot:overviewtab) Para obter mais informações, consulte [Enable Microsoft Whiteboard on Surface Hub](https://support.office.com/article/enable-microsoft-whiteboard-on-surface-hub-b5df4539-f735-42ff-b22a-0f5e21be7627).
