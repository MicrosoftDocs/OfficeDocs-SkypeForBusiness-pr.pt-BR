# <a name="manage-the-whiteboard-in-microsoft-teams"></a>Gerenciar o quadro de dados no Microsoft Teams

O Quadro de Branco da Microsoft é uma tela digital de forma livre onde as pessoas, o conteúdo e as ideias se reúnem. A integração do quadro de trabalho nas reuniões do Microsoft Teams é alimentada pelo aplicativo Web do Quadro de Trabalho, que permite que os participantes da reunião do Teams desenhem, esboçam e escrevam juntos em uma tela digital compartilhada.

Os usuários podem compartilhar um quadro de opções para torná-lo disponível para todos os participantes em uma reunião do Teams. Esse mesmo quadro de comunicação está disponível simultaneamente em todos os aplicativos de quadro de comunicação no Windows 10, iOS e no aplicativo Web.

Para ativar ou desativar o aplicativo whiteboard, consulte [Enable Microsoft Whiteboard for your organization](https://support.office.com/article/1caaa2e2-5c18-4bdf-b878-2d98f1da4b24). Lembre-se de que essa configuração habilita ou desabilita o Quadro de Trabalho para toda a sua organização e não apenas para o Teams.

## <a name="whiteboard-on-surface-hub"></a>Quadro de trabalho no Surface Hub

Antes de tentar usar o Microsoft Whiteboard, certifique-se de que o aplicativo whiteboard está instalado em seu dispositivo Surface Hub. Se o Quadro de Branco não estiver instalado, no dispositivo Surface Hub, vá para o aplicativo da Microsoft Store e obter [o Microsoft Whiteboard](https://www.microsoft.com/p/microsoft-whiteboard/9mspc6mp8fm4?activetab=pivot:overviewtab). Para obter mais informações, consulte [Enable Microsoft Whiteboard on Surface Hub](https://support.office.com/article/enable-microsoft-whiteboard-on-surface-hub-b5df4539-f735-42ff-b22a-0f5e21be7627).
