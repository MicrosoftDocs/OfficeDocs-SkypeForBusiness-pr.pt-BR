# <a name="manage-the-whiteboard-in-microsoft-teams"></a>Gerenciar o Quadro de Comunicações no Microsoft Teams

O Microsoft Whiteboard é uma tela digital de forma livre em que as pessoas, o conteúdo e as ideias se reúnem. A integração do Quadro de Comunicações em reuniões do Microsoft Teams é alimentada pelo aplicativo Web Whiteboard, que permite que os participantes da reunião do Teams desenhem, esboçam e escrevam juntos em uma tela digital compartilhada.

Os usuários podem compartilhar um quadro de comunicações para dispo-lo para todos os participantes em uma reunião do Teams. Esse mesmo quadro de comunicações está disponível simultaneamente em todos os aplicativos do Whiteboard no Windows 10, no iOS e no aplicativo Web.

Para ativar ou desativar o aplicativo Whiteboard, consulte [Habilitar o Microsoft Whiteboard para sua organização](https://support.office.com/article/1caaa2e2-5c18-4bdf-b878-2d98f1da4b24). Tenha em mente que essa configuração habilitará ou desabilitará o Quadro de Comunicações para toda a sua organização e não apenas para o Teams.

## <a name="whiteboard-on-surface-hub"></a>Quadro de comunicações no Surface Hub

Antes de tentar usar o Microsoft Whiteboard, verifique se o aplicativo Whiteboard está instalado em seu dispositivo Surface Hub. Se o Whiteboard não estiver instalado, no dispositivo Surface Hub, acesse o aplicativo da Microsoft Store e obtenha o [Microsoft Whiteboard](https://www.microsoft.com/p/microsoft-whiteboard/9mspc6mp8fm4?activetab=pivot:overviewtab). Para obter mais informações, consulte [Habilitar o Microsoft Whiteboard no Surface Hub](https://support.office.com/article/enable-microsoft-whiteboard-on-surface-hub-b5df4539-f735-42ff-b22a-0f5e21be7627).
