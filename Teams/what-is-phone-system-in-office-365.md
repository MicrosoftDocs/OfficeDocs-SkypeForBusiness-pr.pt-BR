# <a name="what-is-phone-system"></a>O que é o Sistema de Telefonia

O Sistema de Telefonia é a tecnologia da Microsoft para habilitar o controle de chamadas e recursos de PBX (Private Branch Exchange) na nuvem do Microsoft 365 com o Microsoft Teams.

O Sistema de Telefonia funciona com clientes do Teams e dispositivos certificados. O Sistema de Telefonia permite que você substitua seu sistema PBX existente por um conjunto de recursos fornecidos diretamente do Microsoft 365.

As chamadas entre usuários em sua organização são tratadas internamente no Sistema de Telefonia e nunca vão para a Rede Telefônica Pública Comuada (PSTN) – removendo, assim, os custos de longa distância em chamadas internas. 

Para fazer chamadas externas, o Sistema de Telefonia fornece opções de complemento para se conectar ao PSTN. Para obter mais informações sobre soluções de voz e opções de conectividade PSTN, consulte Planejar sua solução de voz do [Teams](cloud-voice-landing-page.md) e [Conectar-se ao PSTN](#connect-to-the-public-switched-telephone-network-pstn).

O Sistema de Telefonia está disponível em todo o mundo, exceto Macau. 

## <a name="phone-system-features"></a>Recursos do Sistema de Telefonia

Com o Sistema de Telefonia, os usuários podem usar o Teams para fazer e receber chamadas, transferir chamadas e ativar ou desativar mudo de chamadas. Os usuários do Sistema de Telefonia podem clicar em um nome no catálogo de endereços e fazer chamadas do Teams para essa pessoa. Para fazer e receber chamadas, os usuários do Sistema de Telefonia podem usar seus dispositivos móveis, um fone de ouvido com um laptop ou computador ou um dos muitos telefones IP que funcionam com o Teams. 

Os administradores do Sistema de Telefonia podem gerenciar opções de chamada e configurações no mesmo console usado para mensagens, colaboração e assim por diante.

Para obter mais informações sobre os recursos do Sistema de Telefonia, confira Aqui está [o que você obtém com o Sistema de Telefonia](here-s-what-you-get-with-phone-system.md).
  

## <a name="connect-to-the-public-switched-telephone-network-pstn"></a>Conectar-se à PSTN (Rede Telefônica Pública Comunada)
  
Para chamadas externas, o Sistema de Telefonia pode ser conectado ao PSTN de uma das várias maneiras:
  
- Comprar um Plano de Chamadas da Microsoft (doméstica ou doméstica e internacional). O Plano de Chamadas da Microsoft é uma solução totalmente na nuvem com a Microsoft como sua operadora PSTN. Para obter mais informações, consulte [Sistema de Telefonia e Planos de Chamadas](calling-plan-landing-page.md).

- Use sua infraestrutura de telefonia existente para conectividade PSTN local.

  Você pode conectar sua infraestrutura de telefonia local ao Sistema de Telefonia usando a Chamada de Conexão do Operador ou o Roteamento Direto. 

Para obter mais informações sobre todas as opções de conectividade PSTN, consulte [as opções de conectividade PSTN](pstn-connectivity.md).


## <a name="phone-system-with-services"></a>Sistema de Telefonia com serviços

O Sistema de Telefonia pode ser usado para serviços e caixa postal, como:

- **Atendedores automáticos** – Os atendedores automáticos podem ser usados para criar um sistema de menus para sua organização que permite que chamadores externos e internos se movam pelo sistema para localizar e fazer ou transferir chamadas para usuários ou departamentos da empresa em sua organização. Veja [o que são atendedores automáticos de nuvem?](what-are-phone-system-auto-attendants.md).

- **Filas de chamadas** – As saudações da fila de chamadas podem ser usadas quando alguém liga para um número de telefone para sua organização. Essas saudações incluem a capacidade de colocar automaticamente as chamadas em espera e pesquisar o próximo agente de chamada disponível para lidar com a chamada. As pessoas em espera também podem ouvir música enquanto estão em espera. Você pode criar uma ou várias filas de chamadas para sua organização. Confira [Criar uma fila de chamadas na nuvem](create-a-phone-system-call-queue.md).

- **Caixa** postal – Caixa postal na Nuvem é configurada e provisionada automaticamente para todos os usuários do Teams. Confira [Configurar a Caixa postal na Nuvem](set-up-phone-system-voicemail.md).

Para obter mais informações sobre recursos, [consulte Aqui está o que você obtém com o Sistema de Telefonia](here-s-what-you-get-with-phone-system.md). Se você estiver pronto para começar, consulte [Configurar o Sistema de Telefonia em sua organização](setting-up-your-phone-system.md).
