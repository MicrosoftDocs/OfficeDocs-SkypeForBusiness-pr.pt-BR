# <a name="location-of-data-in-microsoft-teams"></a>Localização de dados no Microsoft Teams

Os dados no Microsoft Teams residem na região geográfica associada ao seu locatário do Office 365. Atualmente, equipes suporta as regiões da Austrália, Canadá, França, Índia, Japão, Reino Unido, Américas, APAC e EMEA. 

> [!IMPORTANT]
> As equipes atualmente oferece residência de dados na Austrália, Canadá, França, Índia, Japão e Reino Unido para os novos locatários apenas. Um novo locatário é definido como aquele que não teve nenhum usuário sequer entrando no Microsoft Teams. Inquilinos existentes da Austrália, Índia e Japão continuarão a suas equipes dos dados armazenadas na região APAC. Inquilinos existentes no Canadá continuará tem seus dados armazenados em Américas. França e Reino Unido terão seus dados armazenados na região EMEA.

Para saber mais sobre o lançamento do Índia e Reino Unido residência de dados para equipes, leia a postagem do blog de Ansuman Acharya, [equipes da Microsoft que inicia a Índia dados residência, outros geos em breve](https://techcommunity.microsoft.com/t5/Microsoft-Teams-Blog/Microsoft-Teams-Launches-Australia-and-Japan-Data-Residency/ba-p/237827). 

Para obter mais informações em residência de dados do Canadá para equipes, leia a postagem do blog de Varun Sagar, [residência de dados do Microsoft equipes inicia no Canadá, Austrália e Japão em breve](https://techcommunity.microsoft.com/t5/Microsoft-Teams-Blog/Microsoft-Teams-Launches-Canada-Data-Residency-Australia-and/ba-p/227178). 

Para saber mais sobre o lançamento do Austrália e Japão residência de dados para equipes, leia a postagem do blog de Varun Sagar, [Microsoft equipes inicia Austrália e Japão dados residência](https://go.microsoft.com/fwlink/?linkid=867773). 

Para saber mais sobre o lançamento de residência de dados da França para equipes, leia a postagem do blog de Varun Sagar, [Microsoft equipes inicia França dados residência](https://techcommunity.microsoft.com/t5/Microsoft-Teams-Blog/Microsoft-Teams-launches-France-Data-Residency/ba-p/364466). 

Para ver qual região armazena dados para seu locatário, vá para o [Centro de administração do Microsoft 365](https://portal.office.com/adminportal/home) > **configurações** > **perfil da organização**. Role para baixo até **Local dos dados**.
