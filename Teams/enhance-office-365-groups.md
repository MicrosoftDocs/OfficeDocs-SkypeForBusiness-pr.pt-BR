<a name="enhance-existing-microsoft-365-groups-with-microsoft-teams"></a>Aprimorar grupos existentes do Microsoft 365 com o Microsoft Teams
=======================================================

Os usuários do Microsoft Teams podem aprimorar um Grupo do Microsoft 365 existente com a funcionalidade do Teams. Ao procurar aprimorar um Grupo público do Microsoft 365, os usuários podem fazer isso se o número de membros for igual ou menor que 5.000.

Para isso, os usuários deverão seguir o fluxo de criar uma nova equipe a partir do cliente do Microsoft Teams. Selecione **Criar do** grupo do Microsoft  >  **365** e, em seguida, escolha o grupo existente que você deseja aprimorar com o Teams. Os membros do grupo existente serão adicionados como membros da equipe automaticamente.

> [!IMPORTANT]
> Somente os proprietários do Grupo do Microsoft 365 têm permissão para aprimorar um grupo existente com o Teams, e esse grupo deve conter menos de 5.000 usuários. As equipes que já foram aprimoradas não aparecerão na lista.
>
>A opção de criar uma equipe a partir de um Grupo do Microsoft 365 não estará disponível, a menos que essas condições tenham sido atendidas.

Os usuários também podem convidar uma lista de distribuição para uma equipe, e os membros dessa lista de distribuição também serão incluídos na equipe. Esse é um complemento único e as alterações posteriores na associação ao grupo na lista de distribuição não serão replicadas ou sincronizadas com o Teams.

![Captura de tela da opção para criar uma equipe a partir de um Grupo do Microsoft 365.](media/Enhance_Existing_Office_365_groups_with_Microsoft_Teams_image2.png)

Você também pode adicionar grupos de segurança habilitados para e-mail como membros de uma equipe. Mas, se você adicionar mais membros ao grupo de segurança, esses membros não serão incluídos automaticamente na equipe. Você precisa adicionar os novos membros separadamente ou adicionar novamente o grupo de segurança na equipe. (Se você adicionar novamente o grupo de segurança, remover as duplicidades garante que os membros sejam incluídos apenas uma vez).

Há dois tipos de configurações de privacidade com grupos do Microsoft 365, **públicos e privados.** Enquanto ambos os tipos de grupo podem ser habilitados para o Teams, há uma pequena diferença quando se trata de autoatenditar.

-   Os usuários podem pesquisar grupos públicos e podem ingressar sozinhos sem precisar da aprovação do proprietário da equipe.

-   Os grupos privados não aparecem nas pesquisas e os usuários não podem entrar a menos que o proprietário da equipe os inclua como membros.

Ao criar uma nova equipe no Teams, um proprietário de um grupo particular existente do Microsoft 365 tem uma opção para usar a associação ao grupo do Microsoft 365 para criar a equipe. Os usuários podem adicionar seus arquivos existentes do SharePoint e do OneNote acrescentando uma guia para o SharePoint e mesclando os arquivos do OneNote.
