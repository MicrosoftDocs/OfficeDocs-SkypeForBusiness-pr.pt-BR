# <a name="whats-new-for-operator-connect-operators"></a>Novidades para operadores de conexão de operador

Este artigo descreve as novidades do Operator Connect. Verifique com frequência se há atualizações.

## <a name="dr-to-oc-migration-api"></a>DR para a API de Migração de OC

Em 1º de julho de 2022, a Microsoft publicou uma API para operadores em que eles podem migrar os números do Roteamento Direto para o Operator Connect. 

Para obter mais informações, consulte [Operator Connect Swagger](https://operatorconnect.microsoft.com/api/swagger#tag/TnMigration-(New)).

## <a name="number-status"></a>Status do Número

Em 1º de julho de 2022, a Microsoft publicou uma API para operadores que é usada para fornecer informações de status do operador sobre os números fornecidos em relação ao locatário consentido.

Para obter mais informações, consulte [Operator Connect Swagger](https://operatorconnect.microsoft.com/api/swagger#operation/NumberStatus_GetNumberStatus).
