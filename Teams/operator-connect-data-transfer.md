# <a name="data-transfers-between-carriers-and-microsoft-for-operator-connect"></a>Transferências de dados entre operadoras e Microsoft para operadores Conexão

## <a name="provisioned-and-assigned-numbers"></a>Números provisionados e atribuídos

Para garantir o bom funcionamento do programa Conexão operador, a Microsoft informará a cada operadora participante, por meio de uma API, qual dos números de telefone da operadora foram atribuídos no centro de administração do Teams, bem como o URI SIP (Identificadores de Recurso Uniformes do Protocolo de Iniciação de Sessão) desse número de telefone.

## <a name="call-detail-records-and-quality-data"></a>Registros de detalhes de chamada e dados de qualidade

A Microsoft fornecerá registros de detalhes de chamada e dados de qualidade para cada operadora participante por meio de uma API. Isso é necessário para que a Microsoft e a operadora realizem a solução de problemas e assegurem o bom funcionamento do serviço.

## <a name="call-duration-data"></a>Dados de Duração da Chamada

Cada operadora fornecerá à Microsoft, por meio de uma API, a duração das chamadas feitas por seus clientes usando Microsoft Teams. A Microsoft usará esses registros para fornecer aos clientes informações de uso e garantir que nossos registros sejam consistentes com os da operadora. A Microsoft armazenará e processará esses registros em conformidade com as leis e regulamentos aplicáveis, bem como a Declaração de Privacidade da Microsoft.
