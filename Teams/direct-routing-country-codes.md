# <a name="direct-routing-media-path-country-codes"></a><span data-ttu-id="1b6eb-103">Código de caminho de mídia de roteamento direto</span><span class="sxs-lookup"><span data-stu-id="1b6eb-103">Direct Routing media path country codes</span></span>

<span data-ttu-id="1b6eb-104">Ao escolher um caminho de roteamento para mídia, o roteamento direto, por padrão, sempre atribui um datacenter com base no endereço IP público do controlador de borda de sessão (SBC) e sempre seleciona o caminho mais próximo ao Datacenter do SBC.</span><span class="sxs-lookup"><span data-stu-id="1b6eb-104">When choosing a routing path for media, Direct Routing, by default, always assigns a datacenter based on the public IP address of the Session Border Controller (SBC), and always selects the path closest to the SBC datacenter.</span></span>

<span data-ttu-id="1b6eb-105">No entanto, em alguns casos o caminho de mídia padrão pode não ser o caminho de mídia ideal; por exemplo, um IP público de uma faixa de Estados Unidos pode ser atribuído a um SBC localizado na Europa.</span><span class="sxs-lookup"><span data-stu-id="1b6eb-105">However, in some cases the default media path might not be the optimal media path; for example, a public IP from a United States range might be assigned to an SBC located in Europe.</span></span> 

<span data-ttu-id="1b6eb-106">Usando o parâmetro-MediaRelayRoutingLocationOverride com os cmdlets New-CsOnlinePSTNGateway e Set-CsOnlinePSTNGateway, você pode especificar a região preferida para o tráfego de mídia.</span><span class="sxs-lookup"><span data-stu-id="1b6eb-106">By using the -MediaRelayRoutingLocationOverride parameter with the New-CsOnlinePSTNGateway and Set-CsOnlinePSTNGateway cmdlets, you can specify the preferred region for media traffic.</span></span> <span data-ttu-id="1b6eb-107">Por exemplo, o comando a seguir especifica que a região preferida é a Alemanha:</span><span class="sxs-lookup"><span data-stu-id="1b6eb-107">For example, the following command specifies that the preferred region is Germany:</span></span>

<span data-ttu-id="1b6eb-108">Set-CSOnlinePSTNGateway-Identity sbc1.contoso.com – MediaRelayRoutingLocationOverride DE</span><span class="sxs-lookup"><span data-stu-id="1b6eb-108">Set-CSOnlinePSTNGateway -Identity sbc1.contoso.com –MediaRelayRoutingLocationOverride DE</span></span> 

<span data-ttu-id="1b6eb-109">Observe que a Microsoft recomenda a configuração desse parâmetro apenas se os logs de chamada indicar claramente que a atribuição padrão do datacenter para o caminho de mídia não use o caminho mais próximo ao Datacenter do SBC.</span><span class="sxs-lookup"><span data-stu-id="1b6eb-109">Note that Microsoft only recommends setting this parameter if the call logs clearly indicate that the default assignment of the datacenter for the media path does not use the path closest to the SBC datacenter.</span></span>
