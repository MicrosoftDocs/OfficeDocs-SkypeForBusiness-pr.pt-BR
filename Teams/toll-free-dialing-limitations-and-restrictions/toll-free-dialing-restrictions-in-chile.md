|**Detalhes**|**Descrição**|
|:-----|:-----|
|A chamada gratuita está disponível?  <br/> |Sim  <br/> |
|Formato (s) de discagem  <br/> | 123 XXXX XXXX <br/>  188 800 XXX XXX <br/> |
|Código ISO de país/região  <br/> |CL  <br/> |
|Ilha/território abrangido  <br/> |Não aplicável  <br/> |
