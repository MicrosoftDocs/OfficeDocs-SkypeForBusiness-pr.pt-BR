# <a name="toll-free-dialing-restrictions-in-panama"></a>Restrições de discagem de chamada gratuita no Panamá

[Encontre outro país ou região](../toll-free-dialing-limitations-and-restrictions.md), ou aprender mais sobre as limitações de discagem gratuita e restrições.


|**Detalhes**|**Descrição**|
|:-----|:-----|
|É gratuita chamar disponível?  <br/> |Sim  <br/> |
|Discagem format(s)  <br/> | 00-800-XXX-XXXX <br/> |
|Código ISO de país/região  <br/> |PA  <br/> |
|Ilha/território abordado  <br/> |Não aplicável  <br/> |
|Ilha/território não coberto  <br/> |Não aplicável  <br/> |
|Disponibilidade de linha fixa/com fio  <br/> |Somente pela C &amp; W.  <br/> |
|Disponibilidade com a rede móvel  <br/> |Não  <br/> |
|Comentários, restrições e problemas de acesso  <br/> |O cliente deve discar 011 (o internacional interurbano Panamá código) antes do número de chamada gratuito.  <br/> |
