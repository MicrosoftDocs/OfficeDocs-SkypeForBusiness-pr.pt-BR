# <a name="toll-free-dialing-restrictions-in-panama"></a>Restrições de discagem de chamada gratuita no Panamá

[Encontre outro país ou região](../toll-free-dialing-limitations-and-restrictions.md)ou saiba mais sobre limitações e restrições de discagem gratuita.


|**Detalhes**|**Descrição**|
|:-----|:-----|
|A chamada gratuita está disponível?  <br/> |Sim  <br/> |
|Formatos de discagem  <br/> | 00-800-XXX-XXXX <br/> |
|Código ISO de país/região  <br/> |PA  <br/> |
|Ilha/território coberto  <br/> |Não aplicável  <br/> |
|Ilha/território não coberto  <br/> |Não aplicável  <br/> |
|Disponibilidade de linha com fio/fixa  <br/> |Somente pela C &amp; W.  <br/> |
|Disponibilidade com a rede móvel  <br/> |Não  <br/> |
|Comentários, restrições e problemas de acesso  <br/> |O cliente deve discar 011 (o código internacional de longa distância do Panamá) antes do número gratuito.  <br/> |
