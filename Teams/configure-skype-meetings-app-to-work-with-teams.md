<a name="configure-the-skype-meetings-app-to-work-with-teams"></a>Configurar o aplicativo de Reuniões do Skype para trabalhar com o Teams
===================================================

Depois que um usuário é atualizado para o Microsoft Teams, os administradores podem usar o Centro de administração do Microsoft Teams para especificar o aplicativo preferencial que os usuários usarão para ingressar em reuniões do Skype for Business.

Para especificar o aplicativo Skype for Meetings como o aplicativo preferencial:

1. Entre no centro de administração do Microsoft Teams.
2. No painel esquerdo, em **Configurações de** toda a organização, selecione **Atualização do Teams**.
3. Na página de atualização do Teams, em **Preferências** de aplicativo, selecione **Skype Meetings App** no aplicativo Preferencial para que os usuários participem da lista de reuniões do Skype for **Business.**

    ![Escolha Aplicativo preferencial para os usuários ingressarem em reuniões do Skype for Business](media/teams-configure-skype-meetings-app-to-work-with-teams-image1.png)

## <a name="known-limitations"></a>Limitações conhecidas

Os usuários que usam o Aplicativo de Reuniões do Skype com o Teams estão sujeitos às seguintes limitações:

- Os usuários não têm opção para alterar o dispositivo de vídeo.
- Depois que um usuário é atualizado para o Teams, se o usuário estiver em uma reunião usando o Aplicativo de Reuniões do Skype e receber uma chamada no Teams, a reunião no Skype Meetings App não será colocada em espera. Em vez disso, o usuário está conectado às duas chamadas.
