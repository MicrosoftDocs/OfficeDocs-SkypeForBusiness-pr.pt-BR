<a name="install-microsoft-teams-using-msi-with-sccm"></a>Instalar o Microsoft Teams usando MSI (com SCCM)
===========================================

Microsoft forneceu equipes para selecionar usuários ou máquinas de implantar um arquivo MSI para os administradores em massa. Os administradores podem usar esse arquivo para implantar remotamente equipes para que os usuários não precisam baixar manualmente o aplicativo de equipes. System Center Configuration Manager (SCCM) é a principal ferramenta usada com arquivos MSI para implantações gerenciadas.
