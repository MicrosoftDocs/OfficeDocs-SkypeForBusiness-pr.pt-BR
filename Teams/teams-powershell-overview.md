# <a name="microsoft-teams-powershell-overview"></a>Microsoft Teams visão geral do PowerShell

Microsoft Teams módulo do PowerShell é um conjunto de cmdlets para gerenciar Teams diretamente da linha de comando do PowerShell. O PowerShell fornece recursos avançados para automação que podem ser aproveitados para gerenciar sua Teams de trabalho.  

Você pode usar o Microsoft Teams do PowerShell usando um dos seguintes métodos: 

- Instale o Microsoft Teams do PowerShell por [meio Galeria do PowerShell](https://www.powershellgallery.com/packages/MicrosoftTeams) (opção recomendada). 
- Instale o módulo Microsoft Teams PowerShell por meio do [Download Manual](https://www.powershellgallery.com/packages/MicrosoftTeams). 

> [!NOTE]
> Microsoft Teams versões de módulo do PowerShell anteriores à 4.x.x estão se desativando. Atualize para a versão mais recente. Mais detalhes no [Teams PowerShell – Versões com suporte](teams-powershell-supported-versions.md).

## <a name="features"></a>Recursos 

O Microsoft Teams módulo do PowerShell inclui os seguintes recursos: 

- Usa um único módulo para gerenciar todos os aspectos Teams administração, incluindo usuários, equipes, políticas e configuração.  
- Dá suporte a mecanismos de autenticação, como tokens de acesso e credenciais. 

##  <a name="data-collection"></a>Coleta de dados 

Microsoft Teams configurações padrão do módulo do PowerShell coletam dados de telemetria. A Microsoft agrega dados coletados para identificar padrões de uso e problemas comuns, como cmdlets com baixo sucesso, para nos ajudar a priorizar nosso trabalho para melhorar a experiência do Teams PowerShell. Microsoft Teams módulo do PowerShell não coleta dados pessoais ou privados. Para obter mais detalhes, consulte a Política [de Privacidade da Microsoft](https://privacy.microsoft.com/privacystatement).
