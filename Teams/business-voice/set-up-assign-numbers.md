# <a name="step-5-assign-teams-phone-system-phone-numbers-to-your-users"></a>Etapa 5: Atribuir números de telefone do Sistema de Telefonia do Teams aos usuários

Antes que os usuários possam usar o Teams para fazer ou receber chamadas telefônicas de ou para linhas telefônicas regulares, você precisa atribuir números de telefone a eles. Nos clientes do Microsoft Teams, o número de telefone que você atribui a um usuário é listado no teclado de discagem quando o usuário clica em **Chamadas**. Execute as etapas a seguir para cada usuário que precisa de um número de telefone.

> [!NOTE]
> Se você não vir nenhum número de telefone, aguarde. Pode levar várias horas para que seus novos números de telefone se tornem disponíveis no Teams.
