## <a name="how-does-guest-access-compare-to-federation-external-access"></a><span data-ttu-id="1446b-128">Como o acesso de convidado se compara à Federação (acesso externo)?</span><span class="sxs-lookup"><span data-stu-id="1446b-128">How does guest access compare to federation (external access)?</span></span>

<span data-ttu-id="1446b-129">Acesso de convidado difere de acesso externo (também conhecido como federação) da seguinte maneira:</span><span class="sxs-lookup"><span data-stu-id="1446b-129">Guest access differs from external access (also known as federation) as follows:</span></span>

-   <span data-ttu-id="1446b-130">Com acesso de convidado, um usuário convidado possui acesso para uma equipe de equipes específico que o usuário é convidado para e pode acessar os recursos de equipe específica como canais e arquivos.</span><span class="sxs-lookup"><span data-stu-id="1446b-130">With guest access, a guest user has access to a specific Teams team that the user is invited into, and can access team-specific resources such as channels and files.</span></span> <span data-ttu-id="1446b-131">O usuário convidado também pode conversar com e membros da equipe de chamada.</span><span class="sxs-lookup"><span data-stu-id="1446b-131">The guest user can also chat with and call members of the team.</span></span>

-   <span data-ttu-id="1446b-132">Com federação ou acesso externo, um usuário externo pode ter chats de 1:1 e chamadas com um usuário de outra organização por meio de equipes ou Skype para negócios.</span><span class="sxs-lookup"><span data-stu-id="1446b-132">With federation or external access, an external user can have 1:1 chats and calls with a user in another organization by using either Teams or Skype for Business.</span></span> <span data-ttu-id="1446b-133">Os usuários federados não podem acessar quaisquer recursos da equipe.</span><span class="sxs-lookup"><span data-stu-id="1446b-133">The federated user cannot access any team resources.</span></span> <span data-ttu-id="1446b-134">Comparado ao acesso de convidado, acesso externo permite a comunicação entre organização (se permitido pela políticas) sem qualquer convite explícito, mas é limitado ao chat e chamar somente.</span><span class="sxs-lookup"><span data-stu-id="1446b-134">Compared to guest access, external access allows cross organization communication (if permitted by policies) without an explicit invitation, but is limited to chat and call only.</span></span>
