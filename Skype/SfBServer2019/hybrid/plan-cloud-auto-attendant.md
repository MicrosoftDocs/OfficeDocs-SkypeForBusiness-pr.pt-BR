# <a name="plan-cloud-auto-attendant"></a>Planejar o atendedor automático de nuvem

Skype para Business Server 2019 implementações híbridas usam atendentes automáticos da caixa postal nuvem nuvem apenas e não integram com o Exchange Online.

Usando o Skype para servidor de negócios 2019 com [Atendedores automáticos da telefone sistema](/SkypeForBusiness/what-is-phone-system-in-office-365/what-are-phone-system-auto-attendants.md) ainda não está disponível.

Se você for um usuário atual do Exchange Server 2013 ou 2016 do Exchange Server, você poderá continuar a usá-los à medida que implementa Skype para Business Server 2019. UM do Exchange, incluindo a funcionalidade de atendedor automático, está sendo desativado no Exchange 2019.

Funcionalidade semelhante existe no sistema telefônico e Skype para Business Server 2019 interação com o sistema telefônico é aplicação em uma base de recurso por recurso. Planeje apropriadamente.
