# <a name="plan-for-your-skype-for-business-server-2019-deployment"></a>Planejar sua implantação Skype for Business Server 2019
 
**Resumo:** Explore esses links para saber como desenvolver designs conceituais, lógicos e físicos para Skype for Business Server recursos, servidores e topologias de 2019.
  
Skype for Business Server oferece mensagens instantâneas (IM), presença, conferência, vídeo e soluções de telefonia que podem dar suporte aos requisitos de colaboração no nível da empresa. Este tópico fornece links para informações sobre como planejar uma implantação Skype for Business Server bem-sucedida. 
  
Os redatores atualizam os artigos continuamente, quando novas informações tornam-se disponíveis e os usuários fornecem feedback. Esta página será atualizada com frequência, portanto, verifique novamente com frequência.
