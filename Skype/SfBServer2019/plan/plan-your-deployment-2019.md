# <a name="plan-for-your-skype-for-business-server-2019-deployment"></a>Planejar sua implantação do Skype for Business Server 2019
 
**Resumo:** Explore esses links para saber como desenvolver designs conceituais, lógicos e físicos para configurar recursos, servidores e topologias do Skype for Business Server 2019.
  
O Skype for Business Server oferece soluções de mensagens instantâneas (IM), presença, conferência, vídeo e telefonia que podem dar suporte a requisitos de colaboração de nível empresarial. Este tópico fornece links para informações sobre como planejar uma implantação bem-sucedida do Skype for Business Server. 
  
Os redatores atualizam os artigos continuamente, quando novas informações tornam-se disponíveis e os usuários fornecem feedback. Esta página será atualizada com frequência, portanto, verifique novamente com frequência.
