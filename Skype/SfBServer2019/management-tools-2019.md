# <a name="skype-for-business-server-2019-management-tools"></a>Ferramentas de gerenciamento do Skype for Business Server 2019
 
**Resumo:** Saiba mais sobre as ferramentas de gerenciamento de serviço no Skype for Business Server 2019.
  
O Skype for Business Server 2019 oferece soluções de mensagens instantâneas (IM), presença, conferência e telefonia que dão suporte a requisitos de colaboração no nível empresarial. As ferramentas para gerenciar esses serviços são flexíveis e poderosas.
