# <a name="skype-for-business-server-2019-management-tools"></a>Skype para ferramentas de gerenciamento de servidor 2019 de negócios
 
**Resumo:** Saiba mais sobre as ferramentas de gerenciamento do serviço do Skype para Business Server 2019.
  
Skype para Business Server 2019 oferece mensagens instantâneas (IM), presença, conferência e soluções de telefonia que oferecem suporte a requisitos de colaboração de nível empresarial. As ferramentas para gerenciar esses serviços são flexíveis e poderosas.
