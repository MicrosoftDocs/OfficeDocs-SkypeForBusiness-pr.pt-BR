# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Fase 4: mover usuários de teste para o pool piloto

Você pode mover um único usuário ou grupos de usuários para sua nova implantação do Microsoft Skype for Business Server 2019 usando os dois métodos a seguir: painel de controle do Skype for Business Server e o Shell de gerenciamento do Skype for Business Server. Os tópicos desta seção descrevem as tarefas que você deve concluir durante a implantação piloto, bem como antes de mover sua implantação do Skype for Business Server 2019 de uma implantação piloto para uma implantação no nível de produção.
