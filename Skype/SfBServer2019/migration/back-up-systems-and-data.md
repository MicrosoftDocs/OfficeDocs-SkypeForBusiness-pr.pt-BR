# <a name="back-up-systems-and-data"></a>Fazer backup de dados e sistemas

Antes de começar a migração, realize um backup completo do sistema e documentos existentes no sistema, incluindo um inventário das contas de usuário que estão hospedados em cada pool, de forma que você pode reverter se for necessário. Várias ferramentas e programas estão disponíveis para backup e restauração de sistemas, configurações e dados.

Para obter detalhes e procedimentos, consulte backup and restore procedimentos para a sua versão.
