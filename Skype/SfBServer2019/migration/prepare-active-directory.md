# <a name="prepare-active-directory-for-skype-for-business-server"></a>Preparar o Active Directory para Skype for Business Server

Antes de implantar o Skype for Business Server 2019 em um estado de coexistência, você deve executar algumas tarefas adicionais do Active Directory para configurar o esquema, a floresta e o domínio para Skype for Business Server 2019. As extensões de esquema adicionam as classes e atributos do Active Directory exigidos pelo Skype for Business Server 2019. <!-- For additional information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>Para preparar o Active Directory para Skype for Business Server 2019

1. No servidor Skype for Business Server front-end 2019, execute Skype for Business Server configuração 2019.
    
2. Selecione **Preparar Active Directory**.
  
3. Conclua as etapas de 1 a 5 no assistente.
