# <a name="prepare-active-directory-for-skype-for-business-server"></a>Preparar o Active Directory para Skype para Business Server

Antes de implantar o Skype para negócios 2019 de servidor em um estado de coexistência, você deve executar algumas tarefas adicionais do Active Directory para configurar o esquema, floresta e domínio do Skype para Business Server 2019. As extensões de esquema adicionam as classes do Active Directory e atributos que são exigidos pelo Skype para Business Server 2019. <!-- For additional information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>Para preparar o Active Directory para o Skype para Business Server 2019

1. No Skype para Business Server 2019 servidor Front-End, execute Skype para a instalação do Business Server 2019.
    
2. Selecione **preparar o Active Directory**.
  
3. Conclua as etapas 1 a 5 no assistente.
