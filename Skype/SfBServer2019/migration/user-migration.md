# <a name="user-migration"></a>Migração de usuários

Uma prática recomendada geralmente aceita em migrações de é criar vários usuários de teste e usá-los para conduzir testes de sistemas. Depois de ter movido com êxito e testado essas contas, você deve identificar um grupo de usuários de produção piloto e mova as contas e execute testes de validação neles. Quando você obtiver resultados satisfatórios, você pode mover o restante de seus usuários para a nova implantação.
  
Para obter informações adicionais sobre habilitação de usuários para Skype para Business Server 2019, consulte o Skype para Business Server 2019 documentação implantando.
