# <a name="configure-clients-for-migration"></a>Configurar clientes para migração

Este tópico contém as etapas de implantação de cliente recomendadas que devem ser tomadas antes da migração para o Skype for Business Server 2019. Essas alterações de configuração devem ser feitas no Lync Server 2013 ou no Skype for Business Server 2015 (a versão **da**qual você está migrando).
  
### <a name="to-configure-clients-before-migration"></a>Para configurar clientes antes da migração

1. Implante as atualizações de servidor, cliente e dispositivo (hotfixes) mais recentes para a sua instalação existente.
    
2. Na versão anterior do Skype for Business Server, use a filtragem de versão do cliente para permitir que apenas os clientes com as atualizações mais recentes sejam instaladas.
