# <a name="configure-clients-for-migration"></a>Configurar clientes para migração

Este tópico contém as etapas de implantação de cliente recomendado que você deve tomar antes de migrar para o Skype para Business Server 2019. Essas alterações de configuração devem ser feitas no Lync Server 2013 ou Skype para Business Server 2015 (a versão que você está migrando **de**).
  
### <a name="to-configure-clients-before-migration"></a>Para configurar clientes antes da migração

1. Implante as atualizações mais recentes servidor, cliente e dispositivo (hotfixes) para sua instalação existente.
    
2. Na versão anterior do Skype para Business Server, use a filtragem de versão do cliente para permitir apenas clientes com as atualizações mais recentes instaladas.
