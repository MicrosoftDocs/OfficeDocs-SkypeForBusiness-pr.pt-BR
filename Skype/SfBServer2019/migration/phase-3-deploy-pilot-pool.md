# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Fase 3: Implantar Skype para o pool piloto Business Server 2019

Esta seção aborda as etapas necessárias para implantar um pool piloto do Skype para Business Server 2019. A implantação do Skype para Business Server 2019 requer usando o construtor de topologias para definir sua topologia e os componentes que você deseja implantar, Preparando seu ambiente para a implantação do Skype para componentes de negócios Server 2019, publicar sua topologia projetar o primeiro servidor Front-End e, em seguida, instalação e configuração Skype para Business Server 2019 do software para os componentes para sua implantação. Quando concluir, sua Skype para implantação do pool piloto Business Server 2019 coexistirá com um pool herdado existente.
