# <a name="view-current-users-in-legacy-pool"></a>Exibir usuários atuais no pool herdado

Antes de aprender as várias maneiras de mover os usuários entre pools, primeiro é necessário determinar quais usuários existem no pool herdado. A coluna **pool de registradores** identifica os usuários que estão configurados para o pool herdado. Estes são os usuários de teste que iremos mover para o pool do Skype for Business Server 2019.
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>Para ver a lista de usuários no pool herdado

1. Faça logon no servidor de front-end herdado com uma conta que seja membro do grupo RTCUniversalServerAdmins ou membro da função administrativa CsAdministrator ou CsUserAdministrator.
    
2. Abra o **painel de controle do Skype for Business Server**.
    
3. Clique em **usuários**, em **Pesquisar**e em **Localizar**.
