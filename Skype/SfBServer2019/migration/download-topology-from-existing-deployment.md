# <a name="download-topology-from-existing-deployment"></a>Baixar a topologia da implantação existente

Ao criar um pool Skype for Business Server 2019, você usará o Armazenamento de Gerenciamento Central associado à instalação herdada. Quando você inicia o Construtor de Topologias no primeiro uso e sessões de edição subsequentes, você é solicitado a usar o local onde deseja que o Construtor de Topologias carregue o documento de configuração atual. Como você já tem uma topologia definida e estabeleceu o armazenamento de Gerenciamento Central, você deve optar por baixar uma topologia de uma implantação existente. O Construtor de Topologia lerá o banco de dados e recuperará a definição atual. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Para baixar uma topologia de uma implantação existente

1. Abra o Assistente Skype for Business Server implantação.
    
2. Na página **Skype for Business Server 2019 - Assistente** de Implantação, clique em Instalar Ferramentas **Administrativas.**
    
3. Iniciar Construtor de Topologias: clique em **Iniciar,** em Todos os **Programas,** em **Microsoft Skype for Business Server 2019** e em Skype for Business Server **Construtor de Topologias.**
    
4. Selecione **Download da topologia de uma implantação existente**.
  
5. Escolha um nome de arquivo e salve a topologia com o tipo de arquivo .tbxml padrão.
    
6. Expanda o nó Skype for Business Server para revelar as várias funções de servidor na implantação.
