# Movendo o provedor de audioconferência de um usuário para Microsoft

> [!IMPORTANT]
> Este artigo foi traduzido por um sistema de tradução automática, leia o aviso de isenção de responsabilidade.  
  
Para usar audioconferência no Office 365 com o Skype for Business e Teams Microsoft, os usuários em sua organização precisa de uma licença de ** Conferência de áudio** atribuída a eles e seu provedor de audioconferência devem ser definidos para a Microsoft. Consulte[Experimentar ou comprar conferência de áudio no Office 365](try-or-purchase-audio-conferencing-in-office-365.md) para obter mais informações sobre o licenciamento e quanto custa.
  
## Para mover o provedor de audioconferência de um usuário para Microsoft

Se uma licença de **Conferência de áudio** é atribuída a um usuário que não tem um provedor de audioconferência, provedor do usuário será automaticamente definido para a Microsoft e você não precisa fazer mais nada. Se o usuário já tiver um provedor de conferência de áudio, você deve alterar o provedor do usuário para Microsoft depois de atribuir a elas uma licença de **Conferência de áudio**.
  
Para definir a Microsoft como o provedor de audioconferência para um usuário com uma licença de **Conferência de áudio** atribuído, mas está usando outro provedor de conferência de áudio, faça o seguinte:
  
1. Entre no Office 365 com sua conta corporativa ou de estudante.
    
2. Vá para o **Centro de administração do Office 365** > **Skype for Business**.
    
3. Do **Skype para o Centro de administração de negócios**, vá para **a conferência de áudio**.
    
4. Se você vir uma faixa notificando que há usuários que têm uma **Conferência de áudio** licença atribuída, mas não tiver o Microsoft definido como seu provedor de audioconferência ainda, clique em **clique aqui para movê-los**. Se você não vir a faixa de opções, no **Skype para o Centro de administração de negócios** em **usuários** e, em seguida, selecione o filtro de **usuários prontos para ser movida para a conferência de áudio**.
    
5. Selecione os usuários que você deseja mover e, em seguida, no painel ação, clique em **Editar**.
    
6. Selecione **Microsoft** na lista **nome do provedor**.
    
    > [!NOTE]
    > Quando o provedor de conferência de áudio de um usuário é alterado para a Microsoft, as informações de conferência de áudio do usuário serão alterado. Se você precisar manter essas informações, registre-la em outro antes de alterar o provedor de qualquer um dos usuários. 
  
7. Clique em **Salvar**.
    
## O que mais devo saber?

- Se você selecionar o usuário na lista, você pode exibir as informações de conferência de áudio padrão do usuário, mas você não será capaz de ver o PIN de conferência de áudio. Você pode redefinir a PIN de um usuário audioconferência clicando em **Redefinir**.
    
- Se você quiser alterar as configurações de conferência de áudio para um usuário, você pode selecionar o usuário na lista e clique em **Editar** e faça suas alterações na página **Propriedades**.
