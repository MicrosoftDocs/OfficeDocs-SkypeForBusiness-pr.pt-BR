|**Detalhes**|**Descrição**|
|:-----|:-----|
|É gratuita chamar disponível?  <br/> |Sim  <br/> |
|Discagem format(s)  <br/> | 123 XXXX XXXX <br/>  188 800 XXX XXX <br/> |
|Código ISO de país/região  <br/> |CL  <br/> |
|Ilha/território abordado  <br/> |Não se aplica  <br/> |
|Ilha/território não coberto  <br/> |Sem acesso na Ilha da Páscoa.  <br/> |
|Disponibilidade de linha fixa/com fio  <br/> |Parcial. Veja os comentários.  <br/> |
|Disponibilidade de rede móvel  <br/> |Parcial. Veja os comentários.  <br/> |
|Comentários, restrições e problemas de acesso  <br/> |O formato de número 188 800 XXX XXX é acessível a partir da rede Telefonica. O formato de número XXXX XXXX 123 é acessível a partir da rede Entel somente. Acesso móvel está acessível para ambos os formatos de número.  <br/> |
