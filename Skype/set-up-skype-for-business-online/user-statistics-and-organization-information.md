# <a name="user-statistics-and-organization-information"></a>Informações estatísticas e a organização do usuário

O Skype para Business dashboard fornece um instantâneo rápido de como o serviço está configurado para a sua empresa. Se você ainda não tiver configurado Skype for Business, consulte [Configurar Skype para Business Online](set-up-skype-for-business-online.md) para obter instruções detalhadas.
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>Como alterar Skype para Business Online estatísticas de usuário

|||
|:-----|:-----|
|**Propriedade** <br/> |**Como alterá-la** <br/> |
|Skype para usuários corporativos Online  <br/> |Para adicionar ou impedir que pessoas usando o Skype para o aplicativo de negócios, consulte [atribuir ou remover licenças para o Office 365 para empresas](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) ou [Remover licenças de usuários no Office 365 para empresas](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Para obter uma lista completa do que fazer quando um usuário deixa o seu negócio, consulte [Remover um antigo funcionário do Office 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Usuários habilitados para conferência discada  <br/> |Para permitir que as pessoas para a conferência discada, consulte [Configurar conferências de áudio para Skype para equipes da Microsoft e de negócios](../audio-conferencing-in-office-365/set-up-audio-conferencing.md).  <br/> |
|Usuários na nuvem  <br/> |Esse é o número do Skype para usuários corporativos Online que possuem contas do Office 365.  <br/> Para adicionar pessoas à sua assinatura do Office 365 (ou seja, criar contas do Office 365 para eles), consulte [Adicionar usuários individualmente ou em massa para o Office 365 - ajuda de Admin](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Os usuários sincronizados e hospedagem online  <br/> |Esse é o número do Skype para usuários corporativos Online que são sincronizados em um local no Active Directory.  <br/> Para configurar o seu local no Active Directory, consulte [Configurar a sincronização de diretório para o Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
   
> [!NOTE]
> Se você vir que indisponível listado, isso significa que não há nenhuma informação ou ocorreram erros. 
  
## <a name="how-to-change-information-about-your-organization"></a>Como alterar as informações sobre sua organização

|||
|:-----|:-----|
|**Propriedade** <br/> |**Como alterá-la** <br/> |
|Nome da organização  <br/> |Este é o nome que você inseriu quando você assinou para o Office 365. Para alterá-lo, consulte [alterar sua organização endereço, contato técnico e muito mais](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|ID da organização  <br/> |Você não pode alterar sua ID de organização.  <br/> |
|Organização criada  <br/> |Você não pode alterar esse valor.  <br/> |
|Sincronização do Active Directory em  <br/> |Trata-se a sua organização está sendo sincronizada para um Skype para a organização Business Online que está no local.  <br/> Para configurar o seu local no Active Directory, consulte [Configurar a sincronização de diretório para o Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Domínios de organização  <br/> |Quando você configura inicialmente o Office 365, esses são os domínios que foram registrados.  <br/> Para adicionar um domínio, consulte [Adicionar vários domínios ao Office 365](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207).  <br/> Para remover um domínio, consulte [Remover um domínio do Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c).  <br/> |
