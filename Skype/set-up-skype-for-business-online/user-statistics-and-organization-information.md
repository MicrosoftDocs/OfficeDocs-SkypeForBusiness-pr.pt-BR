# Estatísticas de usuário e informações da organização

> [!IMPORTANT]
> Este artigo foi traduzido por um sistema de tradução automática, leia o [aviso de isenção de responsabilidade](3e55c5a2-baad-4573-9ad9-37d0c42dcb53.md#MT_Footer). Para sua referência, veja a versão em inglês deste artigo [aqui](https://support.office.com/en-us/article/3e55c5a2-baad-4573-9ad9-37d0c42dcb53). 
  
O painel de controle Skype for Business oferece uma visão rápida de como o serviço está configurado para sua empresa. Se você ainda não configurou Skype for Business, consulte [Configurar o Skype for Business Online](set-up-skype-for-business-online.md) para instruções passo a passo.
  
## Como alterar as estatísticas de usuário do Skype for Business Online

|||
|:-----|:-----|
|**Propriedade** <br/> |**Como alterá-la** <br/> |
|Usuários do Skype for Business Online  <br/> |Para adicionar pessoas ou impedir que elas usem o aplicativo Skype for Business, consulte [Atribuir ou remover licenças de assinatura no Office 365](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) ou[Remove licenses from users in Office 365 for business](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Para obter uma lista completa do que fazer quando um usuário deixa sua empresa, consulte [Remover um ex-funcionário do Office 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Usuários habilitados para conferência discada  <br/> |Para permitir que pessoas para conferência discada, consulte [Configurar conferência de áudio para o Skype for Business e Teams da Microsoft](../audio-conferencing-in-office-365/set-up-audio-conferencing-for-skype-for-business-and-microsoft-teams.md).  <br/> |
|Usuários na nuvem  <br/> |Este é o número de usuários do Skype for Business Online que têm conta para o Office 365.  <br/> Para adicionar pessoas à sua assinatura do Office 365 (isto é, criar contas do Office 365 para eles), consulte [Adicionar usuários individualmente ou em lote ao Office 365 - Ajuda para administradores](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Usuários sincronizados e hospedados online  <br/> |Este é o número de usuários do Skype for Businnes Online que estão sincronizados de um Active Directory local.  <br/> Para configurar seu Active Directory local, consulte [Configurar a sincronização de diretório no Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
   
> [!NOTE]
> Se Indisponível aparecer listado, significa que não há informações ou havia erros. 
  
## Como alterar informações sobre sua organização

|||
|:-----|:-----|
|**Propriedade** <br/> |**Como alterá-la** <br/> |
|Nome da organização  <br/> |Este é o nome que você inseriu quando se cadastrou no Office 365. Para alterá-lo, consulte [Alterar o endereço da sua organização, contato técnico e mais](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|OrganizationID  <br/> |Você não pode alterar o ID da Organização  <br/> |
|Organização criada  <br/> |Você não pode alterar este valor.  <br/> |
|Sincronização do Active Directory ativada  <br/> |Este é se sua organização está sendo sincronizada a uma organização Skype for Business Online local.  <br/> Para configurar seu Active Directory local, consulte [Configurar a sincronização de diretório no Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Domínios da organização  <br/> |Quando você configurou Office 365 inicialmente, estes foram os domínios registrados.  <br/> Para adicionar um domínio, consulte [Como adicionar outros domínios ao Office 365](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207).  <br/> Para remove rum domínio, consulte [Remover um domínio do Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c).  <br/> |
