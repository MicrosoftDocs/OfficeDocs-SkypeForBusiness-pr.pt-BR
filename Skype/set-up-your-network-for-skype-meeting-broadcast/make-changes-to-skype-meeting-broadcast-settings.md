# Fazer alterações nas configurações da Transmissão de Reunião do Skype para a sua organização

> [!IMPORTANT]
> Este artigo foi traduzido por um sistema de tradução automática, leia o aviso de isenção de responsabilidade.  
  
Você pode habilitar a Transmissão de Reunião do Skype e fazer alterações nas configurações e políticas para essas reuniões.
  
- **Habilitar a Transmissão de Reunião do Skype** Habilita a Transmissão de Reunião do Skype. Após habilitar a Transmissão de Reunião do Skype, você precisa[Configurar a rede para a Transmissão de Reunião do Skype](set-up-your-network-for-skype-meeting-broadcast.md). Siga esta etapa se quiser realizar webinars ou outras transmissões para pessoas de fora de sua empresa. 
    
- **Recursos de habilitar visualização de transmissão de reunião de Skype para minha organização** O Skype para programas de clientes de negócios fornecer acesso antecipado a novos produtos e recursos. Isso dá a sua organização uma espiada no qual está chegando e a oportunidade de testar os novos recursos no seu ambiente e fazer comentários antes de nós libere compilações de produto para o público em geral.[Skype para visualização de negócios](https://www.skypepreview.com/)
    
- **Permitir organizadores agendar reuniões anônimos** Isso permite que os organizadores faça legendas codificadas ou tradução para transmitir reuniões.
    
- **Permitir transmitir reuniões a serem registrados** Isso permite que as reuniões que você precisa ser registrado pelo apresentador ou organizador.
    
- **URL para participantes de suporte técnico** Insira um link para os participantes de transmissão de reunião usar se precisar de ajuda se conectando ou participar de uma reunião de transmissão.
