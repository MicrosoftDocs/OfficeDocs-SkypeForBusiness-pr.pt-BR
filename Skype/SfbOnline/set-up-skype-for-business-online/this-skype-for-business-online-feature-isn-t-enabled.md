# <a name="this-skype-for-business-online-feature-isnt-enabled"></a>Este recurso do Skype for Business Online não está habilitado

Parece que o usuário não está licenciado para esse recurso.
  
|**Possível causa**|**Ação sugerida**|
|:-----|:-----|
|Você tem licenças disponíveis, mas ainda não a tiver ligado para esse usuário.  <br/> |Vá para o Centro de administração do **Microsoft 365**  >  **Usuários ativos** para licenciar este usuário. <br/> |
|O usuário está em um local onde esse recurso não está disponível.  <br/> |Se possível, vá para o Centro de administração do **Microsoft 365** Usuários ativos e atribua esse usuário a um local onde  >   o recurso está disponível. <br/> |
|O recurso não foi ligado para sua organização.  <br/> |Vá para o Centro de administração do **Microsoft 365**  >  **Skype for Business** e a ligue o recurso para toda a organização. <br/> |
