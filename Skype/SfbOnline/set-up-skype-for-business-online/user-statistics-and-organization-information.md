# <a name="user-statistics-and-organization-information"></a>Estatísticas de usuários e informações da organização

O painel do Skype for Business fornece um instantâneo de como o serviço está configurado para a sua empresa. Se você ainda não configurou o Skype for Business, veja [Configurar o Skype for Business Online](set-up-skype-for-business-online.md) para obter instruções passo a passo.
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>Como alterar as estatísticas de usuário do Skype for Business Online

|||
|:-----|:-----|
|**Propriedade** <br/> |**Como alterá-la** <br/> |
|Usuários do Skype for Business Online  <br/> |Para adicionar ou impedir que as pessoas usem o aplicativo Skype for Business, consulte Atribuir ou remover licenças do [Microsoft 365](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) para empresas ou Remover licenças de usuários no [Microsoft 365 para empresas.](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1)  <br/> Para ver uma lista completa do que fazer quando um usuário sair da sua empresa, consulte [Remover um ex-funcionário do Microsoft 365.](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1)  <br/> |
|Usuários habilitados para Audioconferência  <br/> |Para habilitar pessoas para Audioconferência, consulte Experimentar ou [comprar Audioconferência no Microsoft 365 ou no Office 365.](../audio-conferencing-in-office-365/try-or-purchase-audio-conferencing-in-office-365.md)  <br/> |
|Usuários na nuvem  <br/> |Esse é o número de usuários do Skype for Business Online que têm contas do Microsoft 365 ou do Office 365.  <br/> Para adicionar pessoas à sua assinatura do Microsoft 365 ou do Office 365 (ou seja, criar contas do Microsoft 365 ou office 365 para elas), consulte Adicionar usuários individualmente ou em massa ao [Office 365 –](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec)Ajuda para Administradores.  <br/> |
|Usuários sincronizados ou hospedados online  <br/> |Esse é o número de usuários híbridos criados em uma implantação local do Skype for Business Server, mas sincronizados com o Microsoft 365 ou o Office 365.  <br/> Para configurar a conectividade híbrida do Skype for Business, veja [Implantar a conectividade híbrida entre o Skype for Business Server e o Skype for Business Online](https://technet.microsoft.com/library/jj204669.aspx).  <br/> |
   
> [!NOTE]
> Se for exibido Não Disponível, não há informações ou ocorreram erros. 
  
## <a name="how-to-change-information-about-your-organization"></a>Como alterar as informações de sua organização

|||
|:-----|:-----|
|**Propriedade** <br/> |**Como alterá-la** <br/> |
|Nome da organização  <br/> |Esse é o nome que você inscreveu quando se inscreveu no Microsoft 365 ou no Office 365. Para alterá-lo, veja [Alterar o endereço, contato técnico e outras informações de sua organização](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|ID da organização  <br/> |Você não pode alterar o ID da Organização  <br/> |
|Data de criação da organização  <br/> |Você não pode alterar este valor.  <br/> |
|Sincronização do Active Directory ativada  <br/> |Mostra se a sua organização está sincronizada com uma organização do Skype for Business Online local.  <br/> Para configurar o Active Directory local, consulte Configurar a sincronização de diretório do [Microsoft 365 ou do Office 365.](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846)  <br/> |
|Domínios da organização  <br/> |Quando você configura inicialmente o Microsoft 365 ou o Office 365, esses são os domínios que foram registrados.  <br/> Para adicionar um domínio, consulte [Adicionar vários domínios ao Microsoft 365 ou ao Office 365.](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207)  <br/> Para remover um domínio, consulte [Remover um domínio do Microsoft 365 ou do Office 365.](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c)  <br/> |
