# <a name="customize-meeting-invitations"></a>Personalizar convites de reunião

Como um administrador, você pode garantir que todos os Skype para convites de reunião de negócios incluir algumas informações úteis, os usuários talvez precisem, como links para seu site de suporte e aviso de isenção legal, logotipo de sua empresa e outro texto que você deseja incluir. 
  
## <a name="tips-for-creating-a-logo-for-meeting-invitations"></a>Dicas para criar um logotipo para convites de reunião
<a name="__top"> </a>

1. Crie uma imagem que é não ultrapassa 188 pixels de largura por 30 pixels de altura (é muito pequeno).
    
2. Salve a imagem no formato JPG.
    
3. Armazene a imagem em um local central que todos na sua empresa podem acessar, como um compartilhamento de rede, OneDrive compartilhado para a pasta de negócios, pasta de recados, etc.
    
    Agora você pode adicioná-lo dos convites para reuniões. Consulte as próximas etapas.
    
## <a name="customize-your-meeting-invitations"></a>Personalizar seus convites de reunião
<a name="__top"> </a>

![logotipo-sfb-30x30.png](../images/sfb-logo-30x30.png) **usando o Skype para centro de administração de negócios**

1. Entre no Office 365 com sua conta corporativa ou de estudante.
    
2. A partir do Centro de administração do Office 365, escolha **Admin centrais** > **Skype para negócios**.
    
3. No Skype para centro de administração de negócios, escolha **reuniões online**. 
    
4. Preencha os seguintes campos:
    
    ![Aqui está o que ele pode parecer com quando você concluir o formulário para personalizar seus convites de reunião.](../images/b0a7c3c6-0d86-41c6-b116-331143bbe398.png) 

   **URL do logotipo:** Insira a URL onde o logotipo de sua está armazenado. Lembre-se de que ele deve estar no formato JPG! 
 
   **URL da Ajuda:** Se sua organização tem um site de suporte que você deseja que as pessoas para ir para se eles tiverem problemas, insira a URL aqui. 

   **URL legal:** Se sua organização tem um site legal que você quiser que as pessoas para ir para se eles tiverem preocupações legais, insira a URL aqui.
    
   **Texto do rodapé:** Insira uma nota de rodapé somente texto que será apropriada para seus usuários.  
  
   
5. Escolha **Salvar**.
    
6. Aguarde uma hora ou mais para que as alterações obter preenchido em todos os serviços do Office 365. Em seguida, vá para o Outlook e agendar uma reunião do Skype para ver a aparência de suas alterações. 
    
    ![Aqui é um convite personalizado semelhante ao seguinte, sem o gráfico](../images/ebb5c03c-c23d-4da7-97f1-9b13e26a6cf8.png)
