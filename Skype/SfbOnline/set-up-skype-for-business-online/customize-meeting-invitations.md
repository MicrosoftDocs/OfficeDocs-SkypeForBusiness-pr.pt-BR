# <a name="customize-meeting-invitations"></a>Personalizar convites para reuniões

Como administrador, você pode garantir que todos os convites de reunião do Skype for Business incluam algumas informações úteis que seus usuários podem precisar, como links para seu site de suporte e aviso de isenção de responsabilidade, o logotipo da sua empresa e outros textos que você deseja incluir. 
  
## <a name="tips-for-creating-a-logo-for-meeting-invitations"></a>Dicas para criar um logotipo para convites de reuniões
<a name="__top"> </a>

1. Crie uma imagem com no mínimo 188 pixels de largura por 30 pixels de altura (é bem pequena).
    
2. Salve a imagem no formato JPG.
    
3. Armazene a imagem em um local central que todos em sua empresa possam acessar, como um compartilhamento de rede, pasta compartilhada do OneDrive for Business, Dropbox etc.
    
    Agora você pode adicioná-la aos seus convites para reuniões. Veja os próximos passos.
    
## <a name="customize-your-meeting-invitations"></a>Personalize seus convites para reuniões
<a name="__top"> </a>

![Um ícone mostrando o logotipo do Skype for Business](../images/sfb-logo-30x30.png) **Usando o centro de administração do Skype for Business**

1. Entre com sua conta do trabalho ou da escola.
    
2. No centro de administração, escolha **Centros de administração do** Skype for  >  **Business.**
    
3. No Centro de administração do Skype for Business, escolha **reuniões online.** 
    
4. Preencha as caixas a seguir:
    
    ![Veja como será a aparência quando você concluir o formulário para personalizar seus convites de reunião.](../images/b0a7c3c6-0d86-41c6-b116-331143bbe398.png) 

   **URL do logotipo:** Insira a URL onde seu logotipo está armazenado. Lembre-se de que ele deve estar no formato JPG! 
 
   **URL da Ajuda:** Se sua organização tiver um site de suporte para o que você deseja que as pessoas acessem se elas têm problemas, insira a URL aqui. 

   **URL legal:** Se sua organização tiver um site legal para o que você deseja que as pessoas acessem se elas têm problemas legais, insira a URL aqui.
    
   **Texto do rodapé:** Insira uma nota de rodapé somente texto que será apropriada para os usuários.  
  
   
5. Escolha **Salvar**.
    
6. Aguarde cerca de uma hora para que as alterações se preencham em todos os serviços do Microsoft 365 ou do Office 365. Em seguida, vá para o Outlook e agende uma reunião do Skype para ver a aparência das alterações. 
    
    ![Veja como é a aparência de um convite personalizado, sem o gráfico](../images/ebb5c03c-c23d-4da7-97f1-9b13e26a6cf8.png)
