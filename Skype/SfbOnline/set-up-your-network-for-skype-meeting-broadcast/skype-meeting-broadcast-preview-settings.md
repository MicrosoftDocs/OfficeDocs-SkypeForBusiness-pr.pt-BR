# <a name="skype-meeting-broadcast-preview-settings"></a>Configurações de Visualização da Transmissão de Reunião do Skype

O programa de visualização do cliente do Skype for Business oferece acesso antecipado a novos produtos e recursos. Ele permite que sua organização dê uma olhada no que está por vir, teste os novos recursos em seu próprio ambiente e dê comentários antes de lançarmos builds de produtos para o público em geral. Para saber mais, consulte [a visualização do Skype for Business.](https://www.skypepreview.com/)
