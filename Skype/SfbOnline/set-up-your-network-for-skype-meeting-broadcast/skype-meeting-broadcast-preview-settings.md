# <a name="skype-meeting-broadcast-preview-settings"></a>Configurações de visualização da transmissão de reunião do Skype

O programa de visualização do cliente do Skype for Business oferece acesso antecipado a novos produtos e recursos. Ele permite que sua organização tenha uma espiação do que está chegando, testar os novos recursos em seu próprio ambiente e enviar comentários antes de lançarmos compilações de produtos para o público em geral. Para saber mais, confira [visualização do Skype for Business](https://www.skypepreview.com/).
