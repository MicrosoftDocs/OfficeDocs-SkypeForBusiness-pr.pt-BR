# <a name="download-a-letter-of-authorization-loa"></a>Baixar uma Carta de Autorização (LOA) 

A **Carta de Autorização (LOA)** nos dá permissão para solicitar, em seu nome, a transferência dos números existentes de outro provedor de serviços, em caso de controvérsias. Use uma LOA se você tiver números de serviço para pontes de audioconferência, atendedores automáticos ou outros números de serviço, números de chamada gratuita ou tiver mais de 999 números de telefone de usuário (assinantes) que precisam ser transferidos para o Skype for Business (ou se não quiser usar o **Assistente de Portabilidade Numérica Local**. Você deve [Enviar uma solicitação de serviço personalizado manualmente](manually-submit-a-custom-service-request.md) com essa LOA para obter números de telefone transferidos de seu provedor de serviços atual.
  
Se você tiver números de serviço para pontes de audioconferência, atendedores automáticos ou outros números de serviço, números de chamada gratuita ou tiver mais de 999 números de telefone de usuário (assinantes) que precisam ser transferidos para o Skype for Business, envie manualmente um pedido de portabilidade.
  
Se não tem certeza sobre os tipos de números de telefone cuja transferência você solicitará, veja [Diferentes tipos de números de telefone usados para Planos de Chamadas](different-kinds-of-phone-numbers-used-for-calling-plans.md).
  
> [!IMPORTANT]
> [!IMPORTANTE] Ao fazer a portabilidade manual de números de telefone usando uma LOA, **selecione o tipo correto de número de telefone**. É necessário **enviar pedidos de portabilidade separados** para cada tipo de número de telefone que você queira transferir.
