# <a name="phone-number-management-for-belgium"></a>Gerenciamento de números de telefone para a Bélgica

Use esta tabela para descobrir como obter e gerenciar números de telefone do Skype for Business Online e do Microsoft Teams para a Bélgica. 
  
Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Baixe os formulários corretos e os envie para nós**|
|:-----|:-----|
|Obtenha novos números de telefone do usuário que não estão listados no Skype para centro de administração de negócios.   <br/> |Na Bélgica, não é possível obter números de telefone para a sua organização usando o Centro de administração do Skype for Business. Você deve enviar uma solicitação de novo número de telefone. <br/><br/>Baixe um destes formulários: <br/> [Solicitação de novo número de telefone para a Bélgica (números geográficos) (v.3.0)(de-be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(de-be).pdf) <br/> [Solicitação de novo número de telefone para a Bélgica (números geográficos) (v.3.0)(fr-be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(fr-be).pdf) <br/> [Solicitação de novo número de telefone para a Bélgica (números geográficos) (v.3.0)(nl-be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(nl-be).pdf) <br/>  <br/>Preencha o formulário e [o envie para nós](mailto:ptneu@microsoft.com).  <br/> 
|Obtenha novos números de telefone do serviço que não estão listados no Skype para centro de administração de negócios. <br/> |<br/>Baixe um destes formulários:<br/>[Nova solicitação de número de telefone para Bélgica (serviço-números) (v. 1.2) (de-ser)](../../downloads/new-number-request-forms/New-phone-number-request-for-belgium-(service-numbers)-(v.1.2)-(de-be).pdf)<br/>[Nova solicitação de número de telefone para Bélgica (serviço-números) (v. 1.2) (nl-ser)](../../downloads/new-number-request-forms/New-phone-number-request-for-belgium-(service-numbers)-(v.1.2)-(nl-be).pdf) <br/> [Nova solicitação de número de telefone para Bélgica (serviço-números) (v. 1.2) (fr-ser)](../../downloads/new-number-request-forms/New-phone-number-request-for-belgium-(service-numbers)-(v.1.2)-(fr-be).pdf)<br/><br/>Preencha o formulário e [o envie para nós](mailto:ptneu@microsoft.com).
|Você precisa obter números de telefone transferidos para nós.  <br/> | Baixe uma dessas formas. <br/> <br/> Para números de telefone do usuário: <br/>[Letra de autorização (LOA) para a Bélgica (números geográficas) (v.1.0) (de-ser)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(de-be).pdf) <br/> [Letra de autorização (LOA) para a Bélgica (números geográficas) (v.1.0) (fr-ser)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(fr-be).pdf) <br/> [Letra de autorização (LOA) para a Bélgica (números geográficas) (v.1.0) (nl-ser)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(nl-be).pdf)  <br/>  <br/><br/> Para números para ligações gratuitas: <br/>[Letra de autorização (LOA) para a Bélgica (números para ligações gratuitas) (v.1.0) (de-ser)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(toll-free-numbers)-(v.1.0)-(de-be).pdf) <br/> [Letra de autorização (LOA) para a Bélgica (números para ligações gratuitas) (v.1.0) (fr-ser)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(toll-free-numbers)-(v.1.0)-(fr-be).pdf) <br/> [Letra de autorização (LOA) para a Bélgica (números para ligações gratuitas) (v.1.0) (nl-ser)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(toll-free-numbers)-(v.1.0)-(nl-be).pdf) <br/> <br/>   Preencha o formulário e [o envie para nós](mailto:ptneu@microsoft.com). <br/> |
|Você precisa publicar/cancelar a publicação de números de telefone na lista telefônica nacional.  <br/> |[Nos envie](mailto:ptneu@microsoft.com) a lista de números de telefone que você deseja publicar/cancelar a publicação.      <br/> |
|Você precisa alterar o número de um usuário para um número de serviço.  <br/> |[Nos envie](mailto:ptneu@microsoft.com) a lista de números de usuário que você deseja converter em números de serviço. <br/> |
