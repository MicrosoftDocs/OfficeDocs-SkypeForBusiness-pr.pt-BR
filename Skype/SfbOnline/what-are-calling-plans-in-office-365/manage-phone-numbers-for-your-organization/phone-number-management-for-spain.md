# <a name="phone-number-management-for-spain"></a>Gerenciamento de números de telefone para a Espanha

Use esta tabela para descobrir como obter e gerenciar números de telefone do Skype for Business Online e do Microsoft Teams para a Espanha. 
  
Para obter mais informações, consulte [Gerenciar os números de telefone de sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Baixar os formulários corretos**|
|:-----|:-----|
|Você precisa obter números de usuário que não estejam listados no Centro de administração do Skype for Business.  <br/> | Baixe a [Solicitação de novo número de telefone para a Espanha (números geográficos) (v.3.0)(es-es)](../../downloads/new-number-request-forms/new-phone-number-request-for-spain-(geographic-numbers)-(v.3.0)-(es-es).pdf) <br/><br/>    Preencha o formulário e [o envie para nós](mailto:ptneu@microsoft.com).  <br/> |
|Você precisa obter números de telefone transferidos para nós.  <br/> |Baixe o formulário [Carta de Autorização (LOA) para a Espanha (todos os números) (v.1.0)(es-es)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-spain-(all-numbers)-(v.1.0)-(es-es).pdf), preencha-o e [o envie para nós](mailto:ptneu@microsoft.com). <br/> |
|Você precisa publicar/cancelar a publicação de números de telefone na lista telefônica nacional.  <br/> |[Nos envie](mailto:ptneu@microsoft.com) a lista de números de telefone que você deseja publicar/cancelar a publicação. <br/> |
|Você precisa alterar o número de um usuário para um número de serviço.  <br/> |[Nos envie](mailto:ptneu@microsoft.com ) a lista de números de usuário que você deseja converter em números de serviço. <br/> |
