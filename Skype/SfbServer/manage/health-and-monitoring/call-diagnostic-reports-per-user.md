# <a name="call-diagnostic-reports-per-user-in-skype-for-business-server"></a>Chamar Diagnostic Reports (per user) no Skype para Business Server
 
**Resumo:** Saiba mais sobre os por usuário chamada relatórios de diagnóstico usada no Skype para Business Server.
  
Os Relatórios de Diagnóstico de Chamada oferecem informações por usuário sobre falhas em sessões ponto a ponto e de conferência. Neste momento, há apenas um relatório.
  
## <a name="in-this-section"></a>Nesta seção

 [Relatório de atividades do usuário no Skype para Business Server](user-activity-report.md) Fornece informações sobre atividades ponto a ponto e de conferência para cada um dos seus usuários.
