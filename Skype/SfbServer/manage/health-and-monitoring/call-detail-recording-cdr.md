# <a name="call-detail-recording-cdr-in-skype-for-business-server"></a>Registro de detalhes das chamadas (CDR) no Skype for Business Server
 
**Resumo:** Saiba mais sobre os registros cdr (registro de detalhes das chamadas) usados no Skype for Business Server.
  
O CDR (registro de detalhes das chamadas) registra informações de uso e diagnóstico sobre atividades ponto a ponto, incluindo serviço de mensagens instantâneas, chamadas VoIP, compartilhamento de aplicativos, transferência de arquivos e reuniões. Os dados de uso podem ser utilizados para calcular o retorno sobre o investimento, enquanto os dados de diagnóstico podem ser usados para solucionar problemas de atividades ponto a ponto e reuniões. Ao instalar o Skype for Business Server, você também instalará um conjunto predefinido de definições de configuração globais para CDR. Use os tópicos desta seção para configurar o CDR.
