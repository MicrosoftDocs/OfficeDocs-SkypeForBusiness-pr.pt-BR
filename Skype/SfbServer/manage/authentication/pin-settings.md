Use os procedimentos nas seções a seguir para gerenciar PINs no Skype para Business Server.
  
Você pode gerenciar Skype para políticas de PIN de servidor de negócios do Skype para painel de controle do Business Server ou Skype do Shell de gerenciamento do servidor de negócios. Use os procedimentos a seguir para configurar políticas de PIN para sua organização.
  
- [Exibir informações de política PIN no Skype para Business Server](view-pin-policy-information.md)
    
- [Criar uma nova política PIN no Skype para Business Server](create-a-new-pin-policy.md)

- [Atribuir uma política de PIN por usuário no Skype para Business Server](assign-a-per-user-pin-policy.md)
    
Use os procedimentos a seguir para gerenciar o PIN de conferência discada dos usuários da seção de **usuários** do Skype para painel de controle do servidor de negócios.
  
- [Definir discada um usuário PIN no Skype para Business Server](set-a-user-s-dial-in-conferencing-pin.md)
    
- [Exibir usuário informações de PIN no Skype para Business Server](view-user-pin-information.md)
