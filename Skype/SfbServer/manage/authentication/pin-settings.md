- [Atribuir uma política de PIN por usuário no Skype para Business Server 2015](assign-a-per-user-pin-policy.md)
    
Use os procedimentos a seguir para gerenciar o PIN de conferência discada dos usuários da seção de **usuários** do Skype para painel de controle do servidor de negócios.
  
- [Definir discada um usuário PIN no Skype para Business Server 2015](set-a-user-s-dial-in-conferencing-pin.md)
