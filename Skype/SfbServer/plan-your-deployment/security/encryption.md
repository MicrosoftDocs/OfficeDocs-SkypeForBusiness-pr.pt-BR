## <a name="media-encryption"></a><span data-ttu-id="c3b33-138">Criptografia de mídia</span><span class="sxs-lookup"><span data-stu-id="c3b33-138">Media Encryption</span></span>

<span data-ttu-id="c3b33-p106">O tráfego de mídia é criptografado usando SRTP (Secure RTP), um perfil do protocolo RTP que fornece confidencialidade, autenticação e proteção contra ataques de repetição para o tráfego RTP. Media flowing in both directions between the Mediation Server and a media gateway is optionally encrypted and recommended.
