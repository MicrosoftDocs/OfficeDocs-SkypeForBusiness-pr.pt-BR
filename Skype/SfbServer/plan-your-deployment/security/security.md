# <a name="plan-for-security-in-skype-for-business-server"></a>Planejar a segurança no Skype for Business Server 
 
Conteúdo de segurança do Skype for Business Server. 
  
Esse conteúdo está relacionado à segurança do Skype for Business Server. Segurança é um tópico bastante amplo. A segurança é feita entre todos os recursos do Skype for Business Server, além de bancos de dados, serviços e hardware que compõem o ecossistema. Este conteúdo é um suplemento para tópicos específicos de segurança sobre o Skype for Business Server.
