<span data-ttu-id="7fbee-p113">Para fazer essa configuração, crie dois endereços de GeoDNS. Um endereço de GeoDNS é usado para acesso interno e resolve no FQDN interno da Web ou endereço IP do balanceador de carga dos dois pools. O outro endereço GeoDNS é usado para acesso externo e resolve no FQDN externo da Web ou endereço IP do balanceador de carga dos dois pools. The following is an example for the Meet simple URL, using the FQDNs for the pools.</span></span> 
  
```
Meet-int.geolb.contoso.com
     Pool1InternalWebFQDN.contoso.com
     Pool2InternalWebFQDN.contoso.com

```

     Pool1ExternalWebFQDN.contoso.com
     Pool2ExternalWebFQDN.contoso.com
