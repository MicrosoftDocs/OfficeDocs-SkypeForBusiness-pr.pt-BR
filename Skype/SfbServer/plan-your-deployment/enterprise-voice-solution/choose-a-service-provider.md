# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server-2015"></a>Escolher um fornecedor de serviço E9-1-1 no Skype for Business Server 2015
 
Escolhendo o seu provedor de serviços para uma implantação do E9-1-1 em Skype para Business Server Enterprise Voice.
  
As E9-1-1 serviço provedor rotas as chamadas efetuadas do Skype para Business Server para o correto pública Safety atendimento PSAP (ponto) com base nas informações de local contidas na chamada. 
  
Para suportar o E9-1-1 como parte de um Skype para implantação Business Server, você deve obter o E9-1-1 qualificado de serviço de roteamento de um programa de interoperabilidade aberta do Lync o provedor de serviços E9-1-1. Escolha o provedor que melhor atende às suas necessidades organizacionais.
