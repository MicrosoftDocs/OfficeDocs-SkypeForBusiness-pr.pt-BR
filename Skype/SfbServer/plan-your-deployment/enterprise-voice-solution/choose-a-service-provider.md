# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>Escolha um provedor de serviços E9-1-1 para o Skype for Business Server
 
Escolher seu provedor de serviços para uma implantação do E9-1-1 no Skype for Business Server Enterprise Voice.
  
O provedor de serviços E9-1-1 encaminha as chamadas de emergência originadas do Skype for Business Server para o ponto de resposta de segurança pública (PSAP) correto, com base nas informações de local contidas na chamada. 
  
Para dar suporte ao E9-1-1 como parte de uma implantação do Skype for Business Server, você deve obter o serviço de roteamento E9-1-1 de um provedor de serviços qualificado do programa de interoperabilidade do Lync Open E9-1-1. Escolha o provedor que melhor atende às suas necessidades organizacionais.

Para ver a lista de serviços de roteamento do E9-1-1 para o Lync, confira [Serviços E-911 qualificados para Lync](https://docs.microsoft.com/skypeforbusiness/lync-cert/e-911-service-providers).
