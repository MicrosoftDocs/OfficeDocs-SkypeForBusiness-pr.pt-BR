# <a name="define-your-requirements-for-emergency-calls-in-skype-for-business-server"></a>Definir seus requisitos para chamadas de emergência em Skype for Business Server
 
Resume as etapas necessárias para habilitação do E9-1-1 no Skype for Business Server Enterprise Voice, dependendo se você tem um provedor de serviço E9-1-1 de tronco SIP ou um gateway ELIN.
  
Antes de iniciar uma implantação Skype for Business Server E9-1-1, você deve primeiro poder responder às perguntas detalhadas nas seções a seguir. O planejamento necessário dependerá do tipo de solução E9-1-1 que você optar por implantar: um provedor de serviços E9-1-1 de tronco SIP ou um gateway ELIN (número de identificação de local de emergência). A tabela a seguir identifica as seções nesta lista de trabalho de planejamento que você precisará revisar para cada uma dessas soluções.
