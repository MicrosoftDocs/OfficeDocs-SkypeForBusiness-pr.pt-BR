# <a name="plan-for-clients-and-devices"></a>Planejar clientes e dispositivos
 
**Resumo:** Revisão dos clientes com suporte e aplicativos Skype para negócios.
  
A força de trabalho atual está constantemente em movimento. Os funcionários precisam se comunicar e colaborar – seja trabalhando no escritório da matriz, nas filiais regionais, em casa ou em viagens. Skype para Business Server 2015 oferece suporte a essas necessidades através de uma coleção de interfaces de cliente que poderá ser implantado para os usuários da organização. Um planejamento elaborado garante que os funcionários obtêm o que precisam e que o Skype para negócios está disponível para acessá-los, onde quer que estejam ser. 
  
## <a name="available-clients"></a>Clientes disponíveis

Skype para Business Server 2015 oferece suporte a vários tipos de clientes, incluindo o software cliente instalado no computador, clientes baseados na web e clientes para dispositivos móveis. Cada um deles é apresentado nesta seção. Se você tiver usado anteriormente uma combinação de clientes do Lync, observe que há não oferece suporte a [clientes herdados](clients-and-devices.md#Legacy) que são incompatíveis com o Skype para negócios. Atualizações são feitas com frequência, portanto, verifique este tópico regularmente para obter as últimas informações sobre o cliente.
  
### <a name="skype-for-business-2016"></a>Skype para 2016 de negócios

Skype para negócios 2016 é o cliente recomendado de completos para Skype para Business Server 2015. Para obter uma descrição dos novos recursos, consulte [What's new in Skype para 2016 de negócios](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) . Suporte ao recurso de cliente é detalhado na [comparação de recursos do cliente de Desktop do Skype para negócios](desktop-feature-comparison.md)e documentação de usuário está em [Ajuda do Skype para negócios](https://support.office.com/en-US/Skype-for-business). Esse cliente é incluído quando um usuário instala o Office 365.
  
Um cliente básico, com menos recursos, grátis e compatível também está disponível. As duas versões estão disponíveis para download em [Baixar Skype para a empresa em todos os dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). As diferenças entre clientes completo e Basic são descritas na seção [limitações do cliente básico](desktop-feature-comparison.md#Full-Basic) .
  
### <a name="skype-for-business-2015"></a>Skype for Business 2015

Skype para negócios 2015 é o cliente recomendado de completos para Skype para Business Server 2015. O Skype para interface de usuário de negócios foram totalmente reprojetado e inclui recursos recentemente integrados, como o Monitor de chamada, integração de diretórios Skype, emoticons e muito mais. Para ver um resumo das alterações, consulte [Lync agora é Skype para negócios — consulte What's new](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612). Suporte ao recurso de cliente é detalhado na [comparação de recursos do cliente de Desktop do Skype para negócios](desktop-feature-comparison.md)e documentação de usuário está em [Ajuda do Skype para negócios](https://support.office.com/en-US/Skype-for-business). Esse cliente é incluído quando um usuário instala o Office 365.
  
Um cliente básico, com menos recursos, grátis e compatível também está disponível. As duas versões estão disponíveis para download em [Baixar Skype para a empresa em todos os dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). As diferenças entre clientes completo e Basic são descritas na seção [limitações do cliente básico](desktop-feature-comparison.md#Full-Basic) .
  
### <a name="skype-for-business-on-mac"></a>Skype for Business para Mac

O cliente do [Skype for Business no Mac](https://www.microsoft.com/en-us/download/details.aspx?id=54108) está disponível para download e é fornecido com o Office 365 para Mac. Consulte [Skype para negócios nos requisitos de cliente do Mac](mac-requirements.md) , para revisar os pré-requisitos.
  
### <a name="skype-for-business-for-mobile-devices"></a>Skype para negócios para dispositivos móveis

Clientes estão disponíveis para o Windows Phone, iPhone/iPad e Android. Os usuários podem conectá-los em [Baixar Skype para a empresa em todos os dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Suporte para estes clientes a funcionalidade é detalhada na [comparação de recursos do cliente móvel para Skype para negócios](mobile-feature-comparison.md).
  
### <a name="online-meeting-add-in-for-skype-for-business"></a>Reunião online Add-in para Skype para negócios

O suplemento de Reunião Online para Skype para a empresa oferece suporte a gerenciamento de reuniões de dentro do cliente de mensagens e colaboração Microsoft Outlook. O suplemento de Reunião Online para Skype para software comercial instala automaticamente com Skype para negócios.
  
### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype para negócios Web App e o aplicativo de reuniões do Skype

Skype para negócios Web App é o cliente de plug-in de navegador de conferência baseado na web para Skype para reuniões Business Server 2015. Os participantes de reunião têm acesso a todos os recursos de colaboração e compartilhamento e aos controles de reunião do apresentador. Esse cliente também permite que os convidados de fora da sua organização que não têm um cliente regular para participar de reuniões criados pelo seu Skype para usuários comerciais. 
  
Se Skype for Business não está instalado no computador de um usuário e o usuário clica em um link da reunião em uma solicitação de reunião, Skype para negócios Web App será instalado e abra. Você também pode configurar a página de ingresso na reunião para permitir que usuários ingressem em reuniões usando versões anteriores de clientes. Skype para negócios Web App é o cliente preferencial para participantes fora da sua organização. Com o Skype para negócios Web App, nenhuma instalação de cliente local será necessária, embora os recursos de áudio, vídeos e compartilhamento requerem a instalação de um plug-in durante o primeiro uso. Consulte o [plano para clientes de reuniões (Web App e reuniões App)](meetings-clients.md) para obter os requisitos usar esse cliente.
  
O aplicativo de reuniões Skype é um aplicativo do Windows 10 direto que usa um navegador durante a instalação apenas e é executado como um aplicativo nativo do Windows 10. Ele requer a configuração de servidor específica para habilitar a distribuição do servidor Front-End. Como Skype para negócios Web App, esse cliente também permite que convidados de fora da sua organização que não têm um cliente regular para participar de reuniões criados pelo seu Skype para usuários comerciais. 
  
### <a name="skype-for-business-web-scheduler"></a>Skype para Agendador da Web de negócios

Skype para negócios Web Scheduler é baseado na web, ferramenta de gerenciamento e agendamento para usuários que não têm acesso ao Microsoft Outlook, ou que estão em um sistema operacional não baseado no Windows de reunião. Com Skype para negócios Web Scheduler, os usuários podem criar novas reuniões, modificar reuniões existentes e enviar convites usando seu programa de e-mail preferido.
  
Documentação do [Skype para negócios Web Scheduler](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) fornece mais detalhes.
  
### <a name="vdi-plugins"></a>Plug-ins VDI

Um ambiente VDI (Virtual Desktop Infrastructure) é usado em algumas organizações onde questões de segurança e conformidade são especialmente sensíveis. Usando o Skype for Business com completos de áudio e vídeo em uma conexão como essa requerem cargas intenso de áudio e vídeo de processamento no cliente hospedados em uma área de trabalho virtual. Software adicional de plug-in de VDI está disponível que libera o que o processamento de máquina de local do usuário final e reduz a carga da área de trabalho virtual. Para obter detalhes sobre como usar esses plug-ins, consulte [Planejar Skype para os negócios em ambientes de VDI](vdi-environments.md) .
  
### <a name="skype-room-systems-v2"></a>Skype Room Systems versão 2

Sistemas de sala Skype v2 é solução conferência mais recente da Microsoft que usa um Skype familiar para interface de negócios e é facilmente implantada e gerenciado, aproveitando equipamento existente como painéis LCD para facilitar a instalação. Sistemas de sala Skype v2 usa um aplicativo desenvolvido especificamente de UWP executando em um 4 de Surface Pro ou Surface Pro em um modo de console (uma vez implantado o aplicativo UWP é o único aplicativo que serão executados no dispositivo) e ele requer sua própria conta de dispositivo em sua Skype para a implementação de negócios. O software é atualizado via Windows Store e Windows Update. Para obter detalhes sobre como usar esses consoles de sala em sua implantação, consulte [Planejar v2 Skype sistemas de sala](skype-room-systems-v2-0.md) .
  
### <a name="skype-for-business-on-surface-hub"></a>Skype for Business no Hub superfície

Microsoft Surface Hub é um dispositivo de produtividade all in one que é destinado para debate, colaboração e apresentações. Ele tem seu próprio iteração do Skype para cliente corporativos, documentado no [guia de administração do Microsoft Surface Hub](https://docs.microsoft.com/en-us/surface-hub/).
  
## <a name="choosing-your-organizations-preferred-client"></a>Como escolher o cliente preferencial para sua organização
<a name="BK_client_choose"> </a>

Se sua organização comprou as licenças adequadas, escolha o cliente completo; caso contrário, o cliente básico.  
  
Os usuários podem instalar o cliente para si próprios do [Skype baixar para a empresa em todos os dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Se alguns dos seus usuários têm Macs, esses usuários terão um conjunto de recursos diferentes, conforme descrito nas seções anteriores. O cliente também é instalado quando os usuários instalam o Office 365.
  
Alguns recursos disponíveis com o Skype para Business Server 2015 não estão disponíveis no Skype para Business Online, consulte [Online ou híbrida limitações de conta de usuário](desktop-feature-comparison.md#Online-Hybrid) para obter informações específicas. Skype para administradores de negócios Online talvez queira consulte [Skype para negócios Online Service Description](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) para obter informações sobre os planos de diferentes disponíveis para eles.
  
 Antes de implantar ou atualizar para o Skype para os negócios, verifique quais clientes já estão em uso na sua organização. Use a [comparação de recursos do cliente de Desktop do Skype for Business](desktop-feature-comparison.md) para entender o impacto de suporte de recurso sobre esses clientes. Isso pode ajudar você a comunicar alterações aos usuários, a acompanhar o processo de implantação e a compreender totalmente os benefícios da atualização para o cliente mais recente.
  
## <a name="ways-to-deploy-the-client-to-your-users"></a>Maneiras de implantar o cliente em seus usuários
<a name="BK_User_Deploy"> </a>

Os instaladores do cliente estão disponíveis para os instaladores do tipo MSI e Clique para Executar. O Skype para a estratégia de cliente-sustainment de negócios pode afetar suas escolhas, portanto, você deve compreender os seguintes pontos: 
  
- Em geral Skype para negócios não adiciona novos recursos aos clientes lançados anteriormente
    
- Em geral Skype para negócios não pretenda no envio novos recursos no Skype para negócios MSI após sua versão inicial. Os aprimoramentos do MSI entre as versões serão basicamente qualidade/segurança em sua natureza. 
    
- O Skype melhor e mais recente para a experiência do cliente de negócios será encontrada no Skype para negócios 2016 click-to-run instalador. 
    
Você pode fazer uma implantação personalizada do cliente conforme descrito na [instalação do cliente de personalizar o Windows no Skype para Business Server 2015](../../deploy/deploy-clients/customize-windows-client-installation.md). Métodos de instalação são descritos em mais detalhes em [implantar clientes para Skype para Business Server 2015](../../deploy/deploy-clients/deploy-clients.md)
  
### <a name="click-to-run"></a>Clique para Executar

Click-to-run é um Microsoft streaming e a tecnologia de virtualização que você pode usar para instalar e atualizar os produtos do Office incluindo Skype para negócios. Esses recursos de streaming e virtualização são baseados nas tecnologias do Microsoft Application Virtualization (App-V). A tecnologia Clique para Executar tem as seguintes vantagens: 
  
- A instalação com streaming do pacote do Office resulta em uma instalação rápida.
    
- Atualizações e patches integrados.
    
- Ocupa menos espaço em disco.
    
- Licenciamento com base no usuário: cinco instalações por usuário.
    
- Personalizável por meio do XML Editor para a instalação de programas independentes.
    
Convém usar a [Ferramenta de implantação do Office 2016](https://www.microsoft.com/en-us/download/details.aspx?id=49117) para esse tipo de instalador.
  
Tanto as versões de cliente básico e completo (com a opção de versões de 32 e 64 bits) estão disponíveis usando um instalador click-to-run, os usuários podem baixar em [Baixar Skype para a empresa em todos os dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). 
  
### <a name="msi"></a>MSI

O MSI é um método de instalação mais tradicional. Ele permite que você instalar manualmente as atualizações e patches, uso licenciamento por volume e ativação e é personalizável por meio da [Ferramenta de personalização do Office](http://www.microsoft.com/en-us/download/details.aspx?id=49030). Você pode distribuir clientes aplicando as Políticas de Grupo, usando o System Center Configuration Manager ou usando uma ferramenta de terceiros.
