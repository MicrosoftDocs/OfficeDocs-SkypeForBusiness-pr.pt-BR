# <a name="deployment-process-overview-for-integrating-on-premises-unified-messaging-and-skype-for-business"></a>Visão geral do processo de implantação para integração de Unificação de Mensagens local e Skype for Business
 
**Resumo:** Analise este tópico durante o planejamento para integrar o Skype para 2015 do servidor de negócios com o Exchange 2013.
  
 Se você deseja integrar o Exchange Unified Messaging (UM) com Skype para Business Server 2015, você deve executar as tarefas descritas neste tópico. Além disso, certifique-se de que você analise os planejamento e implantação práticas recomendadas descritas em [Planejar a integração de Unificação de mensagens do Exchange no Skype para negócios](unified-messaging.md). Este tópico pressupõe que você implantou Skype para negócios 2015 de servidor com um servidor de mediação colocado e que você ter habilitado os usuários para Skype para Business Server 2015, mas talvez não tenha realizado todas as etapas de implantação e configuração para habilitar Enterprise Voz, conforme descrito em [Implantar o Enterprise Voice no Skype para Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) na documentação de implantação.
