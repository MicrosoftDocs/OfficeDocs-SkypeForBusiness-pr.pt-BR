# <a name="plan-for-unified-contact-store-in-skype-for-business-server-2015"></a>Planejamento de repositório unificado de contatos no Skype for Business Server 2015
 
**Resumo:** Analise este tópico durante o planejamento para integrar o Skype para 2015 do servidor de negócios com o Exchange 2013.
  
Repositório unificado de contatos fornece uma experiência consistente de contato em produtos do Microsoft Office e permite que os usuários armazenem todas as informações de contato no Exchange 2013, mas permite que as informações estejam disponíveis globalmente entre Skype para negócios, Exchange, o Outlook e o Outlook Web Access.
  
## <a name="requirements-for-unified-contact-store"></a>Requisitos do repositório unificado de contatos

Para implementar o armazenamento unificado de contatos no Skype para Business Server 2015:
  
- Você deve estar executando o Skype para Business Server 2015 e Exchange 2013.
    
- Os usuários devem usar Skype for Business para migrar seus contatos do Skype para Business Server 2015 para o Exchange 2013.
    
- Caixas de correio do usuário devem ser migradas para o Exchange 2013.
    
- Você deve ter a autenticação de servidor-para-servidor configurada entre Skype para Business Server 2015 e Exchange 2013.
    
    > [!NOTE]
    > Para obter requisitos detalhados sobre como configurar a autenticação entre Skype para Business Server 2015 e Exchange 2013, consulte [gerenciar autenticação de servidor-para-servidor (OAuth) e aplicativos de parceiros no Skype para Business Server 2015](../../manage/authentication/server-to-server-and-partner-applications.md) nas operações documentação.
