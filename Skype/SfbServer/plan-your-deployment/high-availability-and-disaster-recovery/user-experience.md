<span data-ttu-id="60b5b-109">Quaisquer chamadas, reuniões ou conversas de um usuário serão interrompidas quando o pool falhar, sendo necessário restabelecer essas sessões depois do failover para continuar.</span><span class="sxs-lookup"><span data-stu-id="60b5b-109">Any calls, meetings, or conversations a user has when the pool fails are disrupted, and the user must re-establish those sessions after failover to continue.</span></span>

<span data-ttu-id="60b5b-p103">Observe que o banco de dados do Location Information Server não é replicado no pool de backup. ## <a name="user-experience-during-failover"></a><span data-ttu-id="60b5b-115">Experiência do usuário durante o failover</span><span class="sxs-lookup"><span data-stu-id="60b5b-115">User experience during failover</span></span>

<span data-ttu-id="60b5b-p104">Quando um usuário está em um pool que falha, o usuário é desconectado. Qualquer sessão de ponto a ponto na qual o usuário estava participando é terminada, bem como conferências organizadas por esse usuário. o usuário não pode se conectar até o temporizador de resiliência do registrador expirar ou o administrador iniciar procedimentos de failover, o que ocorrer primeiro. Quando o usuário entrar novamente, isso ocorrerá no pool de backup. Se entrarem antes da conclusão do failover, estarão no modo Resiliência até a conclusão do failover. Somente então o usuário poderá estabelecer novas sessões ou estabelecer novamente sessões anteriores.</span><span class="sxs-lookup"><span data-stu-id="60b5b-p104">When a user is in a pool that fails, the user is logged out. Any peer-to-peer session the user was participating in is ended, as are conferences organized by that user. The user cannot log back in until either the registrar resiliency timer expires or the administrator initiates failover procedures, whichever comes first. When the user logs back in, they will log in to the backup pool. If they log in before the failover has completed, they will be in Resiliency mode until failover is complete. Only then can a user establish new sessions or re-establish previous sessions.</span></span>
