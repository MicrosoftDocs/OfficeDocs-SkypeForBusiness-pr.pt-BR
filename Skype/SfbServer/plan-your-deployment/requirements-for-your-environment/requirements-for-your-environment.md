# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Requisitos para seu ambiente do Skype for Business 2015
 
**Resumo:** Tópicos de tópicos para requisitos de sistemas e requisitos de rede ou infraestrutura para o Skype for Business Server 2015. Se você precisar instalar ou configurar algo como pré-requisito para instalar e implantar o Skype for Business Server 2015, ele estará aqui.
  
Esta é a página inicial para artigos sobre requisitos de uma implantação do Skype for Business Server 2015. Isso inclui requisitos de servidor e mais coisas gerais para preparar, incluindo sua estrutura de rede e domínio e certificados.
  
## <a name="whats-in-this-topic"></a>O que há neste tópico

Os dois artigos nos requisitos para a seção de seu ambiente explicam o que você precisa ter e/ou configurar antes de poder continuar e implantar ou atualizar com êxito o Skype for Business Server 2015 no ambiente de servidor da sua organização.
  
### <a name="server-requirements"></a>Requisitos de servidor

Os requisitos de servidor para o [Skype for Business Server 2015](server-requirements.md) são as coisas que você precisa preparar nos servidores individuais que pretende usar em sua implantação.
  
### <a name="environmental-requirements"></a>Requisitos de ambiente

Os requisitos ambientais do [Skype for Business Server 2015](environmental-requirements.md) são uma gama maior de coisas que precisam estar prontas em sua infraestrutura antes de implantar o Skype for Business Server 2015.
