# <a name="associate-branch-office-appliance-with-front-end"></a>Associar Aparelho de Filial com Front End
 
Para associar o aparelho de filial persistente ou servidor de filial persistente a um pool de front-ends ou um servidor Standard Edition, na página **selecionar o pool de front-ends** , selecione o pool de front-ends ou um servidor Standard Edition na lista suspensa **pool de front-ends** . Clique em **Avançar** para continuar.
