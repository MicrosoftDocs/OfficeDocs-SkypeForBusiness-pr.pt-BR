# <a name="associate-branch-office-appliance-with-front-end"></a>Associar Aparelho de Filial com Front End
 
Para associar o aparelho de ramificação sobreviventes ou o servidor de ramificação sobreviventes a um pool de front-end ou um servidor Standard Edition, na lista **selecionar o pool de front-** end, selecione o pool de front-end ou um servidor Standard Edition na lista suspensa **pool de front-ends** . Clique em **Avançar** para continuar.
