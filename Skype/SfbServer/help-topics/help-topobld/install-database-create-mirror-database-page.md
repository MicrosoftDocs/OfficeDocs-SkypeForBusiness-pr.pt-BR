# <a name="install-database-create-mirror-database-page"></a>Instalar Banco de Dados Criar página de Banco de Dados Espelho
 
Esta é uma documentação preliminar e está sujeita a alterações. Tópicos em branco são incluídos como espaços reservados.
  
Se você optou por não instalar os bancos de dados  espelho do SQL Server durante o processo de publicação, instale-os usando Instalar Banco de Dados com um clique com o botão direito do mouse em qualquer nó no Construtor de Topologias. Selecione o repositório de espelho para instalação. Clique em **Configurações** para definir quaisquer configurações avançadas.
