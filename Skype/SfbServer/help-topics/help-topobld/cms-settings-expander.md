# <a name="cms-settings-expander"></a>Expansor de Configurações CMS
 
O Servidor de Gerenciamento Central pode ser alterado e um pool de Front-Ends definido para outro pool de Front-Ends definido. Para alterar o local do Servidor de Gerenciamento Central, selecione o pool de Front-Ends na lista suspensa em **Servidor Front-End no qual instalar o Servidor de Gerenciamento Central**. Um Servidor de Front-End pode ser um pool de Front-Ends Enterprise Edition ou um Servidor Front-End Standard Edition.
  
> [!IMPORTANT]
> Se você tiver definido, publicado e implantado o repositório de Gerenciamento Central para a infraestrutura, não será possível alterar o local do repositório de Gerenciamento Central sem realocá-lo para outro Front-End por um processo externo. 
  
Para obter detalhes sobre como mover o repositório do servidor de gerenciamento central, consulte [move-CsManagementServer](https://docs.microsoft.com/powershell/module/skype/move-csmanagementserver?view=skype-ps) na referência do cmdlet do Windows PowerShell.
