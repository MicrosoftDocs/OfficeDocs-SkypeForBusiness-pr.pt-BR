# <a name="install-mirror-database-option-page"></a>Instalar Página de Opção de Banco de Dados de Espelhamento
 
Define as **Configurações de Banco de Dados** definindo o seguinte:
  
- Digite o **caminho para o compartilhamento de** arquivos para definir o local para os arquivos de SQL Server de backup para o banco de dados que está sendo espelhado.
    
    > [!NOTE]
    > A instância SQL Server principal (instância nomeada ou instância padrão) deve ter permissões de gravação para o compartilhamento de arquivos definido aqui. A instância SQL Server espelho (instância nomeada ou instância padrão) deve ter permissões de leitura para o mesmo compartilhamento de arquivo. 
  
  **OK** aceita e confirma as alterações na caixa de diálogo.
  
  **Cancelar** descarta as alterações e fecha a caixa de diálogo.
  
  **Ajuda** exibe essa tela de ajuda.
