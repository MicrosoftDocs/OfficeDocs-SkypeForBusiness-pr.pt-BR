# <a name="install-mirror-database-option-page"></a>Instalar Página de Opção de Banco de Dados de Espelhamento
 
Define as **Configurações de Banco de Dados** definindo o seguinte:
  
- Digite o **caminho para o compartilhamento de arquivo** para definir o local dos arquivos de backup do SQL Server para o banco de dados que está sendo espelhado.
    
    > [!NOTE]
    > A instância principal do SQL Server (instância nomeada ou padrão) deve ter permissões de gravação no compartilhamento de arquivos que você definir aqui. A instância do SQL Server de espelho (instância nomeada ou instância padrão) deve ter permissões de leitura para o mesmo compartilhamento de arquivo. 
  
  **OK** aceita e confirma as alterações na caixa de diálogo.
  
  **Cancelar** descarta as alterações e fecha a caixa de diálogo.
  
  **Ajuda** exibe essa tela de ajuda.
