# <a name="simple-url-settings-expander"></a>Expansor de Configurações de URL Simples

Para especificar novo, remover existente ou alterar o ativo as URLs simples, faça o seguinte:

## <a name="simple-urls"></a>URLs simples 

Para modificar ou definir as URLs simples, primeiro decida quais das três URLs simples que você deseja editar ou alterar. É possível escolher entre URL de acesso telefônico, URL de reunião e URL de acesso administrativo.

Para modificar a URL de acesso telefônico ou a URL de reunião, selecione a URL que deseja alterar. Clique em  **Editar URL**. Edite a URL e clique em **OK** para salvar a URL. Clique em **Cancelar** para descartar quaisquer alterações.

Para adicionar uma nova URL, clique em **Adicionar**. Na caixa de diálogo **Adicionar URL simples** , ESPECIFIQUE a URL e, em seguida, clique em **OK** para salvar a URL. Selecione **Tornar esta a URL ativa para o domínio selecionado** caso necessite definir a nova URL como URL ativa. Clique em **Cancelar** para descartar quaisquer alterações.

Para fazer uma URL diferente da URL ativa (conforme observado pela marca de seleção verde ao lado da URL), selecione a URL e clique em **Ativar ativa**.

> [!NOTE]
> Só pode haver uma URL ativa para cada domínio de protocolo de início de sessão (SIP).

Para remover uma URL, selecione a URL e clique em **remover**.

> [!CAUTION]
> Não deixe de ler as informações na página de diálogo Configurações de URLs simples. A remoção de uma URL de reunião pode renderizar reuniões existentes agendadas por usuários que não estão mais acessíveis. Considere deixar a URL existente depois de ativar a URL da nova reunião. Quando tiver certeza de que os usuários não estão mais usando a URL de reunião antiga, você pode removê-lo com segurança.

Para editar ou alterar a URL de acesso Administrativo, edite a entrada.

Após concluir todas as alterações na página Editar propriedades, clique em **OK** para salvar as alterações. Clique em **Cancelar** para descartar as alterações.
