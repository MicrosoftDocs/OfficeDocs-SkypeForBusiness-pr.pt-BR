# <a name="simple-url-settings-expander"></a>Expansor de Configurações de URL Simples

Para especificar novas, remover as existentes ou alterar a ativa as URLs simples, faça o seguinte:

## <a name="simple-urls"></a>URLs simples 

Para modificar ou definir os URLs simples, primeiro decida qual dos três URLs simples que você deseja editar ou alterar. É possível escolher entre URL de acesso telefônico, URL de reunião e URL de acesso administrativo.

Para modificar a URL de acesso telefônico ou a URL de reunião, selecione a URL que deseja alterar. Clique em  **Editar URL**. Editar a URL e clique em **Okey** para salvar a URL. Clique em **Cancelar** para descartar quaisquer alterações.

Para adicionar uma nova URL, clique em **Adicionar**. Na caixa de diálogo **Adicionar URL simples** , especifique a URL e clique em **Okey** para salvar a URL. Selecione **Tornar esta a URL ativa para o domínio selecionado** caso necessite definir a nova URL como URL ativa. Clique em **Cancelar** para descartar quaisquer alterações.

Para tornar uma URL diferente a URL ativa (conforme indicado pela marca de seleção verde ao lado da URL), selecione a URL e clique em **Tornar ativa**.

> [!NOTE]
> Só pode haver uma URL ativa para cada domínio de protocolo de iniciação de sessão (SIP).

Para remover uma URL, selecione a URL e clique em **Remover**.

> [!CAUTION]
> Certifique-se de ler as informações na página de diálogo de configurações de URLs simple. Remover uma URL de reunião pode renderizar existentes reuniões agendadas por usuários que não são mais acessíveis. Considere deixando a URL existente depois de fazer a nova URL de reunião ativo. Quando você tiver certeza de que os usuários não estiver mais usando a URL antigo da reunião, você poderá removê-lo com segurança.

Para editar ou alterar a URL de acesso Administrativo, edite a entrada.

Depois de terminar com todas as alterações na página Editar propriedades, clique em **Okey** para salvar as alterações. Clique em **Cancelar** para descartar as alterações.
