# <a name="legacy-merge-add-expanded-edge"></a>Borda Expandida de Adição de Mesclagem de Legado
 
Skype para Business Server não oferece suporte a mesclagem de um Office Communications Server 2007 R2 herdado configuração expandida do servidor de borda. O Servidor de Borda expandido deve primeiro ser convertido em uma configuração de Servidor de Borda consolidada antes de mesclar as informações do Servidor de Borda.
