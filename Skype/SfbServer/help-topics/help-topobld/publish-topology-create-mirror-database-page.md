# <a name="publish-topology-create-mirror-database-page"></a>Publicar Topologia Criar Página de Banco de Dados Espelho
 
Esta é uma documentação preliminar e está sujeita a alterações. Tópicos em branco são incluídos como espaços reservados.
  
O Construtor de Topologias analisará todas as definições de configuração no documento de topologia definido. Se você tiver configurado bancos de dados espelhados do SQL Server, será apresentada uma tela que exibe o armazenamento de espelho, o armazenamento principal, o caminho de compartilhamento de arquivos e os caminhos do banco de dados. Marque uma caixa de seleção associada a um repositório de espelho do banco de dados a fim de criar o banco de dados espelhado e o caminho do compartilhamento de arquivo. Des marque uma caixa de seleção para um espelho de banco de dados se não quiser criar e implantar o banco de dados no SQL Server definido.
  
 Se você optou por não instalar um ou todos esses bancos de dados no momento, limpe os bancos de dados que não serão instalados. Posteriormente, você escolhe usar  Instalar Banco de Dados clicando com o botão direito do mouse na maioria dos itens de nó no Construtor de Topologias.
