# <a name="publish-topology-create-mirror-database-page"></a>Criar Topologia Publicar Página de Banco de Dados de Espelho
 
Esta é uma documentação preliminar e está sujeita a alterações. Tópicos em branco são incluídos como espaços reservados.
  
O construtor de topologias analisa todas as definições de configuração em seu documento de topologia definido. Se você tiver configurado bancos de dados espelhados do SQL Server, será apresentada uma tela que exibe o armazenamento espelho, o repositório principal, o caminho do compartilhamento de arquivos e os caminhos do banco de dados. Marque uma caixa de seleção associada a um repositório de espelho do banco de dados a fim de criar o banco de dados espelhado e o caminho do compartilhamento de arquivo. Você desmarcará uma caixa de seleção para um espelho de banco de dados se não quiser criar e implantar o banco de dados no SQL Server definido.
  
 Se você optou por não instalar um ou todos esses bancos de dados no momento, limpe os bancos de dados que não serão instalados. Mais tarde, você escolhe usar **instalar banco de dados** clicando com o botão direito do mouse na maioria dos itens de nó no construtor de topologias.
