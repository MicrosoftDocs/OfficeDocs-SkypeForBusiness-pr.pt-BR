# <a name="add-director-features"></a>Adicionar Recursos de Diretor

Se você quiser que o pool de Diretores inclua o Monitoring Server, selecione **Monitoramento (CDR e Métricas de QoE)**.

Para obter mais detalhes sobre como adicionar um pool de Diretores, consulte [Scenarios for the Director](/previous-versions/office/lync-server-2013/lync-server-2013-scenarios-for-the-director) and [Setting Up the Director](/previous-versions/office/lync-server-2013/lync-server-2013-setting-up-the-director).
