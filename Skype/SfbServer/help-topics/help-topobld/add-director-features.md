# <a name="add-director-features"></a>Adicionar Recursos de Diretor

Se você quiser que o pool de diretor inclua o Monitoring Server, selecione **monitoração (métricas de QoE e QoE)**.

Para obter mais detalhes sobre como adicionar um pool de diretor, consulte [cenários do diretor](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx) e [Configurando o diretor](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx).
