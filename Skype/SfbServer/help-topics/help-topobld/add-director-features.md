# <a name="add-director-features"></a>Adicionar Recursos de Diretor

Se quiser que o pool de Diretores inclua o Monitoring Server, selecione **Monitoring (CDR e Métricas de QoE).**

Para obter mais detalhes sobre como adicionar um pool de Diretores, consulte [Cenários](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx) para o Diretor [e Configurando o Diretor.](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)
