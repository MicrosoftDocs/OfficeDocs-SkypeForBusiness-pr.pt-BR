# <a name="add-trusted-application-machine"></a>Adicionar Máquina de Aplicativo Confiável
 
Especifique o FQDN (nome de domínio totalmente qualificado) de cada computador que você deseja adicionar como Servidor de Aplicativos Confiável neste pool. Depois de adicionar um computador à lista, você poderá atualizar o FQDN do computador ou removê-lo do pool em qualquer ocasião antes de publicar a topologia. Depois da publicação da topologia, a alteração do FQDN exigirá a exclusão do servidor do Construtor de Topologia e a adição de um novo servidor ao pool com o novo FQDN.
