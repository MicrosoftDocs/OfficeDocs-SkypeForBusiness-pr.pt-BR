# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 Criar Novo Expansor
 
Você pode criar novas funções de servidor no nó do Lync Server 2013 do construtor de topologias clicando com o botão direito do mouse no **Lync server 2013**, selecione **novo**e, em seguida, selecione a função do servidor a ser criada.
