# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 Criar novo expansor
 
Você pode criar novas funções de servidor no nó Lync Server 2013 do Construtor de Topologia clicando com o botão direito do mouse no **Lync Server 2013**, selecione **Novo** e selecione a função de servidor a ser criado.
