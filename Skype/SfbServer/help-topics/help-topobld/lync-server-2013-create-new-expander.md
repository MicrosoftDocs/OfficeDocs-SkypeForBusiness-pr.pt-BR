# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 Criar novo expansor
 
Você pode criar novas funções de servidor no nó do construtor de topologias do Lync Server 2013 clicando com o botão direito do mouse no **Lync server 2013**, selecionando **novo**e selecionando a função de servidor a ser criada.
