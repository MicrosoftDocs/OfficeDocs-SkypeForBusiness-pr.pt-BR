# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 Criar Novo Expansor
 
Você pode criar novas funções de servidor no nó do construtor de topologias do Lync Server 2010 clicando com o botão direito do mouse no **Lync server 2010**, selecionando **novo**e selecionando a função de servidor a ser criada.
