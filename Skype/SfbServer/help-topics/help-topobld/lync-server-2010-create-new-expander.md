# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 Criar Novo Expansor
 
Você pode criar novas funções de servidor no nó Lync Server 2010 do Construtor de Topologia clicando com o botão direito do mouse no **Lync Server 2010**, selecione **Novo** e selecione a função de servidor a ser criado.
