# <a name="add-archiving-server"></a>Adicionar Servidor de Arquivamento
 
Para implantar um servidor de arquivamento em seu design de topologia, você precisa especificar o FQDN (nome de domínio totalmente qualificado) do servidor no qual deseja implantar o arquivamento. Antes de publicar uma topologia que inclui o arquivamento e a instalação do Skype for Business Server, você deve ter concluído todos os pré-requisitos para a implantação do arquivamento.
