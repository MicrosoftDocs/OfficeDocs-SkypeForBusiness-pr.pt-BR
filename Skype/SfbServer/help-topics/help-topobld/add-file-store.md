# <a name="add-file-store"></a>Adicionar Repositóriod e Arquivos
 
Para definir um novo repositório de arquivos, especifique o seguinte:
  
- O nome de domínio totalmente qualificado (FQDN) do servidor que hospedará o novo repositório de arquivos. O FQDN do servidor deve existir no sistema de nome de domínio (DNS) antes de publicar a topologia.
    
- O nome do compartilhamento de arquivo no servidor host. O compartilhamento de arquivo deve existir antes de publicar a topologia.
