# <a name="director-general-settings-expander-for-lync-server-2010"></a>Expansor de Configurações Gerais de Diretor para Lync Server 2010
 
Para configurar as propriedades **gerais** de diretor ou um pool de diretores:
  
 **FQDN**: digite o nome de domínio totalmente qualificado do Diretor ou um pool de diretores
  
 **Associações**
  
 **Repositório de arquivo**: selecione na lista o repositório de arquivos para o Diretor ou um pool de diretores
  
Clique em **novo** para definir um novo repositório de arquivo
  
Marque a caixa de seleção **Associar Monitoring Server** e selecione na lista um Monitoring Server
  
Clique em **novo** para definir um novo Monitoring Server
  
 **OK** Aceita e confirma as alterações na caixa de diálogo.
  
 **Cancelar** Descarta as alterações e fecha a caixa de diálogo.
  
 **Ajuda** Exibe essa tela de ajuda.
