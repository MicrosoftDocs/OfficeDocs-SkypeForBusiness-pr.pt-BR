# <a name="add-default-sip-domain"></a>Adicionar Domínio SIP Padrão
 
Especifique o domínio do protocolo SIP principal que receberá suporte da sua implantação. O domínio SIP principal pode ser o mesmo que ou diferente de seu domínio de serviços de domínio do Active Directory. Por exemplo, o domínio do Active Directory pode ser fabrikam.net, e você pode especificar o domínio do protocolo SIP principal como contoso.net.
