# <a name="legacy-merge-summary-page"></a>Página de Resumo de Mesclagem de Legado


Migração do Microsoft Office Communications Server 2007 R2 para o Lync Server 2013 usa recursos do construtor de topologias para mesclar as topologias existentes. A tela Resumo especifica todas as ações e seleções feitas durante a configuração da fusão do Office Communications Server 2007 R2 na configuração do construtor de topologias do Lync Server 2013.

Revise o resumo e determine se suas seleções foram definidas corretamente. Prossiga com o mesclador se estiver satisfeito com as seleções refletidas no resumo.
