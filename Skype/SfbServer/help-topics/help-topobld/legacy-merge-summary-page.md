# <a name="legacy-merge-summary-page"></a>Página de Resumo de Mesclagem de Legado

A migração do Microsoft Office Communications Server 2007 R2 para o Lync Server 2013 usa recursos do construtor de topologias para mesclar as topologias existentes. A tela Resumo especifica todas as ações e seleções feitas durante a configuração da fusão do Office Communications Server 2007 R2 na configuração do construtor de topologias 2013 do Lync Server.

Revise o resumo e determine se as suas seleções foram definidas corretamente. Prossiga com a fusão se estiver satisfeito com as seleções refletidas no resumo.
