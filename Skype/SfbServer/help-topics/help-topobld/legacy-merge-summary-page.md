# <a name="legacy-merge-summary-page"></a>Página de Resumo de Mesclagem de Legado

Migração do Microsoft Office Communications Server 2007 R2 para o Lync Server 2013 usa os recursos do construtor de topologia para mesclar as topologias existentes. A tela Resumo Especifica que todas as ações e as seleções feitas durante a configuração da fusão do Office Communications Server 2007 R2 para a configuração de construtor de topologia do Lync Server 2013.

Revise o resumo e determine se suas seleções foram definidas corretamente. Continue com a fusão se você estiver satisfeito com as seleções refletidas no resumo.
