# <a name="legacy-merge-summary-page"></a>Página de Resumo de Mesclagem de Legado

A migração do Microsoft Office Communications Server 2007 R2 para o Lync Server 2013 usa recursos do Construtor de Topologias para mesclar as topologias existentes. A tela de resumo especifica todas as ações e seleções que você fez durante a configuração da fusão do Office Communications Server 2007 R2 na configuração do construtor de topologias do Lync Server 2013.

Revise o resumo e determine se suas seleções foram definidas corretamente. Prossiga com o mesclador se estiver satisfeito com as seleções refletidas no resumo.
