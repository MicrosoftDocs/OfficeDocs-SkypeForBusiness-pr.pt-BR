# <a name="external-application-scope-node"></a>Nó de Escopo de Aplicativo Externo
 
Este nó mostra o FQDN (nome de domínio totalmente qualificado) do pool, a configuração de replicação e o pool de próximo salto desse pool de aplicativos confiáveis. Você pode executar as seguintes ações:
  
- **Novo Servidor**. Inicia o assistente **Definir Novo Servidor** que você usa para adicionar outro servidor a este pool de aplicativos confiáveis.
    
- **Editar Propriedades**. Abre a página **Editar Propriedades** desse servidor, na qual é possível definir o FQDN e a porta do pool interno e externo, habilitar e desabilitar a federação e a conversão do endereço de rede (NAT) e especificar o pool de próximo salto.
    
- **Excluir**. Exclui o pool da topologia.
    
Além dessas opções, você pode realizar as seguintes tarefas gerais:
  
- **Topologia**. É possível usar essa opção para definir uma nova topologia, abrir uma topologia existente, baixar a topologia atual, salvar uma cópia da topologia, publicar a topologia, instalar um banco de dados ou remover a implantação.
    
- **Exibir**. Você pode usar essa opção para personalizar o modo de exibição no Construtor de Topologias selecionando os itens que os modos de exibição da árvore de console deve mostrar ou ocultar.
    
- **Ajuda**. Você pode usar essa opção para acessar a Ajuda do nó ativo.
