# <a name="publish-topology-select-cms-page"></a>Publicar Página CMS de Seleção de Topologia

Você publica a topologia que você configurou usando o construtor de topologias. Você será solicitado a selecionar a partir de uma lista que o servidor front-end ou o pool de front-ends assumirá a função de manter o repositório de gerenciamento central. Somente um servidor front-end ou um pool de front-ends pode reter essa função a qualquer momento.

Para saber mais sobre o que é o repositório de gerenciamento central, confira [mover o servidor de gerenciamento central do Lync Server 2010 para o Lync server 2013](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx).
