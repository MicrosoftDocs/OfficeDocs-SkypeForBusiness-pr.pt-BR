# <a name="add-director-association-for-lync-server-2010"></a>Adicionar associação de diretor para Lync Server 2010
 
Defina se o Diretor ou o Diretor do pool vão ter um Monitoring Server associado a ela. Marque a caixa de seleção para associar um Monitoring Server com o Diretor ou diretor pool.
  
> [!TIP]
> Se você ainda não tiver criado um banco de dados do Monitoring Server (e uma instância opcional), você receberá a oportunidade para criá-lo mais tarde, se você selecionar essa caixa de seleção para associar um Monitoring Server com o Diretor ou diretor pool.
