# <a name="add-director-association-for-lync-server-2010"></a>Adicionar Associação de Diretor para Lync Server 2010
 
Você define se o Diretor ou pool de Diretores terá um Monitoring Server associado a ele. Marque a caixa de seleção para associar um Monitoring Server ao Diretor ou pool de Diretores.
  
> [!TIP]
> Se você ainda não tiver criado um banco de dados do Monitoring Server (e instância opcional), terá a oportunidade de criar o banco de dados posteriormente se marcar essa caixa de seleção para associar um Monitoring Server ao Diretor ou pool de Diretores.
