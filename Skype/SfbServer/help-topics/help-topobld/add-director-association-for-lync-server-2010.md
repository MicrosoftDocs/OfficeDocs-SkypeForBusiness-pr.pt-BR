# <a name="add-director-association-for-lync-server-2010"></a>Adicionar Associação de Diretores para o Lync Server 2010
 
Você define se o pool diretor ou diretor terá um Servidor de Monitoramento associado a ele. Marque a caixa de seleção para associar um Servidor de Monitoramento ao pool diretor ou diretor.
  
> [!TIP]
> Se você ainda não tiver criado um banco de dados do Monitoring Server (e instância opcional), terá a oportunidade de criar mais tarde se você selecionar essa caixa de seleção para associar um Servidor de Monitoramento ao pool diretor ou diretor.
