# <a name="archiving-server-scope-node"></a>Nó de Escopo de Servidor de Arquivamento
 
No construtor de topologias, você pode executar tarefas relacionadas ao arquivamento que está executando o servidor clicando com o botão direito do mouse no pool na árvore de console e selecionando **Editar propriedades**. Na página **Editar Propriedades** é possível alterar o nome de domínio totalmente qualificado (FQDN) do servidor que está executando o Arquivamento, o banco de dados SQL Server de arquivamento e o repositório de arquivos de arquivamento.
