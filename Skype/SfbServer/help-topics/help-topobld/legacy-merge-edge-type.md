# <a name="legacy-merge-edge-type"></a>Tipo de Borda de Mesclagem de Legado
 
Para um **Servidor de Borda Único** ou servidor de **Borda consolidada balanceada por carga**, clique em **Próximo** para especificar informações de configuração. Para uma **Borda Expandida**, a configuração de cada servidor deve ser alterada para uma configuração de Borda consolidada antes de poder ser mesclada com o Lync Server 2013.
