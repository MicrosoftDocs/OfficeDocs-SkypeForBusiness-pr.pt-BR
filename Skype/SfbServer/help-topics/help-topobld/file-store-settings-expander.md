# <a name="file-store-settings-expander"></a>Expansor de configurações de repositório de arquivos
 
Para editar as propriedades dos repositórios de arquivos definidos:
  
Para o nome do compartilhamento de arquivos, você pode definir o nome de um novo compartilhamento. Em seguida, você deve criar o compartilhamento no servidor que é nomeado na propriedade de nome (FQDN) do domínio totalmente qualificado do servidor de arquivos.
  
> [!NOTE]
> Você não pode alterar o FQDN do servidor que está especificado na propriedade FQDN do servidor de arquivos. Se você precisar colocar o compartilhamento de arquivo de repositório de arquivos em um servidor diferente, você pode criar uma nova entrada de repositório de arquivo.
