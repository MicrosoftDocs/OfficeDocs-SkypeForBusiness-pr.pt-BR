# <a name="legacy-merge-add-machine"></a>Máquina de Adição de Mesclagem de Legado
 
Para definir os computadores que compõem o pool de Servidor de Borda, insira o nome de domínio totalmente qualificado (FQDN) de pelo menos um computador que faz parte do pool de Borda consolidado e herdado, com carga balanceada.
