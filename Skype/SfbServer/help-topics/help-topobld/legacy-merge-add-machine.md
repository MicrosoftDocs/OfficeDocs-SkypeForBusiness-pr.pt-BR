# <a name="legacy-merge-add-machine"></a>Máquina de Adição de Mesclagem de Legado
 
Para definir os computadores que comem o pool do Servidor de Borda, insira o FQDN (nome de domínio totalmente qualificado) de pelo menos um computador que faz parte do pool de Borda com balanceação de carga herdada e consolidado.
