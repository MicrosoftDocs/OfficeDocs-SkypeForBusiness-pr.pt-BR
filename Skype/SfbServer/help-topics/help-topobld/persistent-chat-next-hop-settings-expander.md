# <a name="persistent-chat-next-hop-settings-expander"></a>Expansor de Configurações de Próximo Salto de Chat Persistente
 
Configure o pool de próximo **salto de** Chat Persistente selecionando um Servidor front-end ou pool de front-end na lista.
  
 **OK** aceita e confirma as alterações na caixa de diálogo.
  
 **Cancelar** descarta as alterações e fecha a caixa de diálogo.
  
 **Ajuda** exibe essa tela de ajuda.
