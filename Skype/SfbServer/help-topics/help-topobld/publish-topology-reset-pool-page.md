# <a name="publish-topology-reset-pool-page"></a>Publicar Página de Pool de Redefinição de Topologia
 
Esta é a documentação preliminar e está sujeita a alterações. Tópicos em branco são incluídos como espaços reservados.
  
Para adicionar novos computadores a um pool existente, você deve redefinir o pool para permitir que o Windows Fabric redefinir as conexões entre todos os membros do pool. Para fazer isso, selecione o pool ao que você adicionou servidores e clique em **Próximo**.
