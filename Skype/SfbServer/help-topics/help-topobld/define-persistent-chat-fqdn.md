# <a name="define-persistent-chat-fqdn"></a>Definir FQDN de Chat Persistente
 
Você cria um novo Servidor de Chat Persistente ou pool de Servidor de Chat Persistente usando o assistente **Definir Novo Pool de Chat Persistente.** Selecione um pool **de vários computadores** ou **um pool de computador único.** Se você selecionar um único pool de computadores e, posteriormente, precisar de um pool de vários computadores, será necessário remover o pool de computador único e definir um pool de vários computadores.
  
Você também deve definir um **FQDN do Pool** para o Servidor de Chat Persistente ou o pool do Servidor de Chat Persistente. O FQDN (nome de domínio totalmente qualificado do pool) para um único pool de computadores deve ser igual ao FQDN do computador que com o pool de servidores único. Para um pool de vários computadores, o FQDN deve ser o nome escolhido para representar esse pool de vários computadores e é definido em DNS por um registro A (e AAAA se IPv6 estiver sendo usado).
