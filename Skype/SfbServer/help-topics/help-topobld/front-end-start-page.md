# <a name="front-end-start-page"></a>Página inicial do Front-End

Use este assistente para definir um novo pool de Front-End para um site. Cada site deve ter pelo menos um pool Enterprise Edition ou servidor Standard Edition. Para obter detalhes sobre os requisitos para o uso do Assistente para criar um novo pool de Front-End, consulte [Define and Configure a Front End Pool](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) na documentação de implantação.
