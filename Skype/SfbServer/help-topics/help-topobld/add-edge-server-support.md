# <a name="add-edge-server-support"></a>Adicionar Suporte de Servidor de Borda

É possível adicionar suporte para acesso de usuário externo quando você implanta sua topologia inicial ou depois. Para obter detalhes sobre como adicionar Servidores de Borda a uma topologia existente, consulte [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) na documentação Implantação do Servidor de Borda.
