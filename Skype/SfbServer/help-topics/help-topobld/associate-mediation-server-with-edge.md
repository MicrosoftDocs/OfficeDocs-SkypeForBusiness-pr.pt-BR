# <a name="associate-mediation-server-with-edge"></a>Associar Servidor de Mediação com Borda
 
Para definir um Servidor de Mediação, especifique as configurações que associam o Servidor de Mediação a um Servidor Front-End e a um pool ou Servidor de Borda. Para associar o Servidor de Mediação ou o pool a um Servidor de Borda ou pool de Servidores de Borda, faça o seguinte:
  
1. Selecione o pool ou Servidor de Borda para associar com o Servidor de Mediação a partir da lista suspensa.
    
2. Clique em **Avançar** para continuar.
    
> [!NOTE]
> Caso você ainda não tenha definido um Servidor de Borda ou pool de Servidor de Borda em sua implantação, a lista suspensa não conterá entradas. Você pode clicar em **Próximo** para continuar e editar as definições para o Servidor de Mediação, após ter adicionado o pool ou Servidor de Borda à sua implantação.
  
Para obter detalhes adicionais, [consulte Deploy a Mediation Server in Topology Builder in Skype for Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
