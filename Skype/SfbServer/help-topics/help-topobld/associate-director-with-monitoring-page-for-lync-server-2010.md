# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Associar diretor com página de monitoramento para o Lync Server 2010
 
Para configurar um servidor de monitoramento e associá-lo com o Diretor ou diretor pool:
  
Em **Monitoring Server** selecione na lista um Monitoring Server.
  
Se não houver nenhum servidor de monitoramento e você deseja criar um Monitoring Server agora, clique em **novo**.
  
 **Voltar**: volta para a caixa de diálogo anterior
  
 **Concluir**: confirma as alterações e completa o Assistente
  
 **Cancelar**: descarta as alterações e cancela a configuração
  
 **Ajuda**: exibe informações, como este tópico de ajuda
