# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Associar Diretor com Página de Monitoramento para Lync Server 2010
 
Para configurar um servidor de monitoramento e associá-lo ao director ou ao pool do diretor:
  
Em **monitorando o servidor** , selecione na lista um servidor de monitoramento.
  
Se não houver um servidor de monitoramento e você quiser criar um servidor de monitoramento agora, clique em **novo**.
  
 **Voltar**: retorna para a caixa de diálogo anterior
  
 **Concluir**: confirma as alterações e conclui o assistente
  
 **Cancel**: descarta alterações e cancela a configuração
  
 **Ajuda**: exibe informações de ajuda, como este tópico
