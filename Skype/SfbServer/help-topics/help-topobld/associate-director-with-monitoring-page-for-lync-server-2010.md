# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Associar Diretor com Página de Monitoramento para Lync Server 2010
 
Para configurar um servidor de monitoramento e associá-lo ao diretor ou pool de diretores:
  
Em **Monitoring Server** select from the list a Monitoring Server.
  
Se não houver um servidor de monitoramento e você quiser criar um servidor de monitoramento agora, clique em **novo**.
  
 **Voltar**: volta para a caixa de diálogo anterior
  
 **Concluir**: confirma as alterações e completa o assistente
  
 **Cancelar**: descarta as alterações e cancela a configuração
  
 **Ajuda**: exibe informações de ajuda, como este tópico
