# <a name="add-mediation-server"></a>Adicionar Servidor de Mediação
 
Para definir o servidor de mediação, especifique um FQDN (nome de domínio totalmente qualificado) para o pool do servidor de mediação. Se você estiver criando um pool de servidores de mediação, especificará o nome do pool para os servidores de mediação que está implantando. Você irá definir os FQDNs dos servidores individuais em outra etapa. Se você estiver implantando um servidor de mediação único, o FQDN especificado será o FQDN do servidor de mediação único. Para obter detalhes, consulte [implantar um servidor de mediação no construtor de topologias no Skype for Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
