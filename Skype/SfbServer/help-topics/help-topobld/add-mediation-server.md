# <a name="add-mediation-server"></a>Adicionar Servidor de Mediação
 
Para definir o Servidor de Mediação, especifique um FQDN (nome de domínio totalmente qualificado) para o pool do Servidor de Mediação. Se estiver criando um pool de Servidores de Mediação, especifique o nome do pool para os Servidores de Mediação que estiver implantando. Defina os FQDNs dos servidores individuais em outra etapa. Se estiver implantando um único Servidor de Mediação, o FQDN que especificar será o FQDN do Servidor de Mediação único. Para obter detalhes, [consulte Deploy a Mediation Server in Topology Builder in Skype for Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
