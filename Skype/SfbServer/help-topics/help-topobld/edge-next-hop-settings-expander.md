# <a name="edge-next-hop-settings-expander"></a>Expansor de Configurações de Próximo Salto de Borda

Para definir ou modificar o **pool de próximo salto** que os servidores de borda usarão para se comunicar com a infraestrutura interna, selecione um diretor, um pool de diretores, um servidor front-end ou um pool de servidores front-end, na lista. Somente os diretores ou servidores front-end que foram configurados no construtor de topologias aparecem na lista.

Para detalhes sobre como definir e configurar as configurações para serviços de Borda, consulte  [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx).
