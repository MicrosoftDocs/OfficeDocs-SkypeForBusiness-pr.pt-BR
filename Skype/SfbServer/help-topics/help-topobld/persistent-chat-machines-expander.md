# <a name="persistent-chat-machines-expander"></a>Expansor de Máquinas de Chat Persistente
 
Você ativa ou desativa um servidor de chat persistente implantado ou um pool de servidores de chat persistentes usando os servidores ou pools listados nas **configurações de estado da máquina**. Selecione um servidor ou pool na lista e clique no botão **Tornar Ativo** para definir um servidor ou pool como ativo.
  
Você define um servidor de chat persistente implantado ou um pool de servidores de chat persistente para inativo, selecionando o servidor **** ou o pool na lista e clicando em disponibilizar inativo. Pelo menos um servidor precisa ser definido como ativo.
  
 **OK** Aceita e confirma as alterações na caixa de diálogo.
  
 **Cancelar** Descarta as alterações e fecha a caixa de diálogo.
  
 **Ajuda** Exibe essa tela de ajuda.
