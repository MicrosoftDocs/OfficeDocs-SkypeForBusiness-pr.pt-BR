# <a name="add-survivable-branch-appliance-pool"></a>Adicionar Pool de Aparelho de Escritório Persistente
 
Para **definir o FQDN do aparelho de filial persistente**, digite o FQDN (nome de domínio totalmente qualificado) (conforme definido no sistema de nomes de domínio (DNS)) do aparelho de filial persistente ou servidor de filial persistente na caixa de texto **FQDN** . Clique em **Avançar** para continuar.
  
> [!NOTE]
> A menos que seja declarado caso contrário, a definição de um aparelho de filial persistente ou de um servidor de filial persistente é o mesmo procedimento.
