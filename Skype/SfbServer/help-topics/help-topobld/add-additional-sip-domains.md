# <a name="add-additional-sip-domains"></a>Adicionar Domínios SIP Adicionais
 
Especifique o nome de domínio de cada domínio adicional para o qual você deseja dar suporte na sua implantação. Por exemplo, se você especificou contoso.net como o domínio de protocolo de início de sessão principal (SIP) e também deseja dar suporte a fabrikam.net para tráfego SIP na sua implantação, especifique fabrikam.net como um domínio adicional com suporte.
