# <a name="shared-components-scope-node"></a>Nó de escopo de componentes compartilhados
 
Você clica com o botão direito do mouse em **componentes compartilhados** na árvore de hierarquia do construtor de topologias para criar novos componentes compartilhados ou acessar as opções de nível de topologia.
  
![Nó de escopo de componentes compartilhados](../../media/Shared_Components_Scope_Node.jpg)
  
Selecione **novo** para criar um novo repositório de repositório, um novo gateway IP/PSTN, um novo servidor do Office Web Apps, uma nova loja do SQL Server ou um novo tronco.
  
> [!NOTE]
> Consulte os tópicos da ajuda para cada uma dessas opções. 
  
![Nó de escopo de componentes compartilhados](../../media/Shared_Components_Scope_NodeB.jpg)
  
Selecione **topologia** para clicar em opções para abrir um novo documento de topologia. Abra um documento de topologia existente, baixe a topologia atual do repositório de gerenciamento central, salve uma cópia do documento de topologia atual, publique o documento de topologia atual, instale o banco de dados para criar as configurações de banco de dados definidas no momento, Mescle o Office Communications Server 2007 R2 na implantação atual ou remova a implantação.
  
![Nó de escopo de componentes compartilhados C](../../media/Shared_Components_Scope_NodeC.jpg)
