# <a name="remove-deployment"></a>Remover Implantação
 
Clique **em Concluir** para criar uma topologia vazia como a primeira etapa no descomissionamento de toda a sua implantação.
  
> [!CAUTION]
> Não é possível desfazer essa etapa.
