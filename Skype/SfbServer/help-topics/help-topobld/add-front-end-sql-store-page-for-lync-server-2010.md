# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Adicionar Página de Repositório SQL de Front End para Lync Server 2010
 
Você define as propriedades das lojas do SQL Server para o novo pool de front-end que você está configurando.
  
> [!IMPORTANT]
> Se você estiver configurando um servidor Standard Edition, esta caixa de diálogo será exibida, mas você não poderá definir as configurações. Clique em **Avançar** para continuar.
  
Configurações que você pode configurar:
  
- **Use uma loja do SQL Server previamente definida**: selecione uma loja existente do SQL Server na lista do repositório do pool de front-ends.
    
- Selecione **definir uma nova loja do SQL Server** para criar uma nova loja do SQL Server.
    
    Você digita o nome de domínio totalmente qualificado do SQL Server no **FQDN do SQL Server**
    
    A **instância padrão** é selecionada inicialmente. Se você preferir definir uma instância, selecione **nomeada instância** e digite o nome da instância que você precisa criar.
    
  **Voltar**: retorna para a caixa de diálogo anterior
  
  **Próximo**: confirma as alterações e passa para a caixa de diálogo próxima etapa
  
  **Cancel**: descarta alterações e cancela a configuração
  
  **Ajuda**: exibe informações de ajuda, como este tópico
