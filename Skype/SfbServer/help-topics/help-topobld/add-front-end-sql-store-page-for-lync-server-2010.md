# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Adicioanr Página de Repositório SQL de Front End para Lync Server 2010
 
Você define as propriedades dos repositórios do SQL Server para o novo pool de front-ends que está configurando.
  
> [!IMPORTANT]
> Se você estiver configurando um servidor Standard Edition, esta caixa de diálogo será exibida, mas você não poderá definir as configurações. Clique em **Avançar** para continuar.
  
Configurações possíveis:
  
- **Use um repositório do SQL Server definido anteriormente**: selecione um repositório existente do SQL Server na lista para o repositório do pool de front-ends.
    
- Selecione **definir um novo repositório do SQL Server** para criar um novo repositório do SQL Server.
    
    Você digita o nome de domínio totalmente qualificado do SQL Server no **FQDN do SQL Server**
    
    A **Instância padrão** é selecionada inicialmente. Se você preferir definir uma instância, selecione **Instância nomeada** e digite o nome da instância que você precisa criar.
    
  **Voltar**: volta para a caixa de diálogo anterior
  
  **Avançar**: confirma as alterações e vai para a próxima caixa de diálogo de etapa
  
  **Cancelar**: descarta as alterações e cancela a configuração
  
  **Ajuda**: exibe informações de ajuda, como este tópico
