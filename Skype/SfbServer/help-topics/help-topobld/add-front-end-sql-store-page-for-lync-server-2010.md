# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Adicionar Página de Repositório SQL de Front End para Lync Server 2010
 
Defina as propriedades dos repositórios do SQL Server para o novo pool de Front-End que você está configurando.
  
> [!IMPORTANT]
> Se você estiver configurando um servidor Standard Edition, esta caixa de diálogo será exibida, mas não será capaz de definir as configurações. Clique em **Avançar** para continuar.
  
Configurações que você pode configurar:
  
- **Use um SQL Server store definido anteriormente**: selecione um repositório do SQL Server existente na lista para o repositório do pool de Front-End.
    
- Selecione o **repositório de definir um novo SQL Server** para criar um novo repositório do SQL Server.
    
    Você digitar o nome de domínio totalmente qualificado do SQL Server no **FQDN do SQL Server**
    
    A **instância padrão** está selecionada inicialmente. Se você preferir definir uma instância, selecione a **instância nomeada** e digite o nome da instância do que você precisa criar.
    
  **Voltar**: volta para a caixa de diálogo anterior
  
  **Avançar**: confirma as alterações e vai para a próxima caixa de diálogo de etapa
  
  **Cancelar**: descarta as alterações e cancela a configuração
  
  **Ajuda**: exibe informações, como este tópico de ajuda
