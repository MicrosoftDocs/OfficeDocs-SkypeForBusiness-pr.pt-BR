# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Adicionar Página de Front-end do Repositório do SQL para Lync Server 2010
 
Você define as propriedades dos SQL Server para o novo pool de Front-End que você está configurando.
  
> [!IMPORTANT]
> Se você estiver configurando um servidor Edição Standard, essa caixa de diálogo será exibida, mas você não poderá definir configurações. Clique em **Avançar** para continuar.
  
Configurações possíveis:
  
- **Use um armazenamento de SQL Server definido anteriormente:** selecione um SQL Server existente na lista para o armazenamento de pool de Front-End.
    
- Selecione **Definir um novo SQL Server para** criar um novo SQL Server store.
    
    Você digita o nome de domínio totalmente qualificado do SQL Server em **SQL Server FQDN**
    
    A **Instância padrão** é selecionada inicialmente. Se você preferir definir uma instância, selecione **Instância nomeada** e digite o nome da instância que você precisa criar.
    
  **Voltar**: volta para a caixa de diálogo anterior
  
  **Avançar**: confirma as alterações e vai para a próxima caixa de diálogo de etapa
  
  **Cancelar**: descarta as alterações e cancela a configuração
  
  **Ajuda**: exibe informações de ajuda, como este tópico
