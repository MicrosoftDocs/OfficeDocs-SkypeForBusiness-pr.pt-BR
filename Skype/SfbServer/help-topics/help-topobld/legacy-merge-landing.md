# <a name="legacy-merge-landing"></a>Patamar de Mesclagem de Legado
 
A página **especificar a configuração de borda** permite que você configure as informações do servidor de borda herdado. Clique em **Adicionar** para especificar a versão do servidor de borda herdada da implantação do Office Communications Server. Se você não tiver nenhum servidor de borda em seu ambiente herdado, clique em **Avançar** para continuar.
