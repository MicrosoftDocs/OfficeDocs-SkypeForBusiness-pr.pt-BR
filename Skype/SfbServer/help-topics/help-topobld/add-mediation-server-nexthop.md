# <a name="add-mediation-server-nexthop"></a>Adicionar NextHop de Servidor de Mediação
 
Para definir o próximo salto para um servidor de mediação que você está definindo, selecione na lista suspensa **pool de próximo salto** do servidor Enterprise Edition ou servidor Standard Edition que atuará como o próximo salto para o servidor de mediação. Clique em **Avançar** para continuar.
