# <a name="add-mediation-server-nexthop"></a>Adicionar NextHop de Servidor de Mediação
 
Para definir o próximo salto para um servidor de mediação que você está definindo, selecione na lista suspensa **próximo pool de saltos** , o servidor Enterprise Edition ou o servidor Standard Edition que atuará como o próximo salto para o servidor de mediação. Clique em **Avançar** para continuar.
