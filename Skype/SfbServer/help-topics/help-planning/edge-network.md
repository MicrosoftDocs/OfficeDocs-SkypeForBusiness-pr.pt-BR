# <a name="edge-network-planning-tool"></a>Edge Network (Planning Tool)
 
A maioria dos componentes de Borda é implantada em uma rede de perímetro. Os seguintes componentes criam a topologia de Borda da rede de perímetro. Exceto onde observado, os componentes fazem parte dos [cenários do servidor de borda no Skype for Business Server 2015](../../plan-your-deployment/edge-server-deployments/scenarios.md) e na rede de perímetro. Os componentes de Borda incluem o seguinte:
  
- Servidor(es) de borda
    
- Proxy reverso
    
- Firewall
    
- Diretores (opcional e logicamente localizado na rede interna)
    
- Balanceamento de carga para Topologias de Borda Dimensionadas (balanceamento de carga DNS ou um balanceador de carga de hardware)
