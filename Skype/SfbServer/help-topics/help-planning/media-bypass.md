# <a name="skype-for-business-server-media-bypass-planning-tool"></a>Skype for Business Server ferramenta de planejamento de Bypass de Mídia
 
Desvio de mídia refere-se à remoção do Servidor de Mediação do caminho da mídia sempre que possível para chamadas cuja sinalização percorre o Servidor de Mediação. Para obter mais detalhes, consulte [Plan for media bypass in Skype for Business 2015](../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
