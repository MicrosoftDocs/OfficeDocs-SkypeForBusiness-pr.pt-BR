# <a name="media-bypass-planning-tool"></a>Media Bypass (Planning Tool)
 
O bypass de mídia se refere a remover o servidor de mediação do caminho de mídia sempre que possível para chamadas cujo sinal percorre o servidor de mediação. Para obter mais detalhes, consulte [plano de bypass de mídia no Skype for Business 2015](../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
