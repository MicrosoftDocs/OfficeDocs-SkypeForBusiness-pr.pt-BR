# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
O recurso de mobilidade no Skype for Business Server 2015 é compatível com a funcionalidade do Skype for Business em dispositivos móveis.
  
A funcionalidade de mobilidade é a mesma do Lync Server 2013. Para obter mais detalhes sobre como preparar seu ambiente para mobilidade, consulte [planejando o Mobility no Lync Server 2013](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx).
