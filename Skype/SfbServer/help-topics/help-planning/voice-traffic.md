# <a name="voice-traffic-planning-tool"></a>Voice Traffic (Planning Tool)
 
A ferramenta de planejamento usa informações sobre o tráfego do usuário para determinar o número de portas a serem usadas na implantação. Esse número em vez disso determina o número de servidores de mediação e gateways que serão necessários. Para obter detalhes, consulte [estimando o uso de voz e o tráfego do Skype for Business Server 2015](../../plan-your-deployment/capacity/estimating-voice-traffic.md).
