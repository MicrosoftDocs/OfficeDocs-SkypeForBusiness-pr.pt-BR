# <a name="pin-policy"></a>Política de PIN

O que você deseja fazer? Clique no link da tarefa que você deseja executar.

 **Trabalhar com a política de PIN para telefones IP** Essa diretiva (PIN) número de identificação pessoal suporta autenticação para usuários que se conectam ao Skype para Business Server usando um telefone IP.
- [Criar uma nova política de PIN no Skype for Business Server 2015](../../manage/authentication/create-a-new-pin-policy.md)

- [Modificar uma política de PIN existente no Skype for Business Server 2015](../../manage/authentication/modify-an-existing-pin-policy.md)

- [Excluir uma política de PIN no Skype for Business Server 2015](../../manage/authentication/delete-a-pin-policy.md)

  **Trabalhar com política PIN para conferência discada** Essa política PIN permite que os usuários que têm credenciais do Active Directory Domain Services em sua organização para ingressar em conferências discadas como usuários autenticados, inserindo um PIN.
- [Create or Modify Dial-in Conferencing PIN Settings for a Site or Group of Users](https://technet.microsoft.com/library/c29bab5c-2b93-48e0-ae0b-29564daaff9a.aspx)

- [Modify the Default Dial-in Conferencing PIN Settings](https://technet.microsoft.com/library/2d110e94-ad29-4755-b17f-d8c2da9b78a4.aspx)

- [Delete Dial-in Conferencing PIN Settings for a Site or Group of Users](https://technet.microsoft.com/library/15a9faee-d024-4c0e-b2a0-fe7e7dc00589.aspx)

Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o Skype para o painel de controle do Business Server, consulte [Gerenciar Skype para Business Server 2015](../../manage/manage.md).
