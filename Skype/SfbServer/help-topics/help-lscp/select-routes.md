# <a name="select-routes"></a>Selecionar Rotas
 
Use a caixa de diálogo **Selecionar rotas** para localizar rotas de voz existentes que você pode usar.
  
> [!NOTE]
> Para recuperar todas as rotas de voz disponíveis, não digite nada na caixa Pesquisar e clique no ícone de pesquisa. 
  
Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o Skype para o painel de controle do Business Server, consulte [Gerenciar Skype para Business Server 2015](../../manage/manage.md).
