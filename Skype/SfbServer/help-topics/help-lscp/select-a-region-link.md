# <a name="select-a-region-link"></a>Selecionar um Link de Região
 
Use a **caixa de diálogo Link** de Região para encontrar links de região disponíveis em seu ambiente. Para usar um link de região existente, clique em um link de região na lista e clique em **OK**.
  
> [!NOTE]
> Clique no ícone de busca sem inserir nada na caixa de busca para obter todos links de região disponíveis. 
  
Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o Painel de Controle Skype for Business Server, consulte [Manage Skype for Business Server 2015](../../manage/manage.md).
