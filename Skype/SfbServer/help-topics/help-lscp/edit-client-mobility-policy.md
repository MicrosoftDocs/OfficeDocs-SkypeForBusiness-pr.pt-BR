# <a name="edit-client-mobility-policy"></a>Editar Política de Mobilidade do Cliente

Na caixa de diálogo **Editar Política de Mobilidade**, edite os detalhes de uma política de mobilidade.

## <a name="ui-reference"></a>Referência de UI

As listas a seguir descrevem os menus, comando, campos e propriedades na página.


- **Commit** Clique para salvar as alterações nas configurações de política.

- **Cancelar** Clique para parar de editar a política e descartar quaisquer alterações feitas nas configurações de política.

- **Nome** O nome da política de mobilidade.

- **Descrição** Uma descrição da política de mobilidade.

Para obter detalhes sobre as políticas de mobilidade, consulte [Mobility Policies](https://technet.microsoft.com/library/8caa5525-e16a-4e38-b3cd-acc0ae9ea375.aspx).
