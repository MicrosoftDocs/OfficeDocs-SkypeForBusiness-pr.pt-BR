# <a name="normalization-rule-type-a-regular-expression"></a>Regra de Normalização: Digite uma Expressão Regular
 
Digite uma expressão regular do .NET Framework para especificar o padrão correspondente para os números de telefone de destino aos quais a rota é aplicada. 
  
Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o painel de controle do Skype for Business Server, consulte [gerenciar o Skype for Business server 2015](../../manage/manage.md).
