# <a name="pin-set-successfully"></a>PIN definido com sucesso
 
Para usar o email para enviar o PIN para o usuário, clique em **abrir meu aplicativo de email para enviar o novo PIN ao usuário**. Para copiar o PIN para a mensagem de email, selecione **Mostrar PIN**. 
  
Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o Skype para o painel de controle do Business Server, consulte [Gerenciar Skype para Business Server 2015](../../manage/manage.md).
