# <a name="select-time-out-queue"></a>Selecionar Fila de Tempo Limite
 
Use a caixa de diálogo **Selecionar uma fila de tempo limite** para selecionar uma fila onde as chamadas são encaminhadas quando o tempo limite é ultrapassado.
  
Na caixa Pesquisar, digite todo ou parte do nome da fila que deseja usar. Para retornar todas as filas disponíveis, deixe a caixa de pesquisa vazia e clique no ícone de pesquisa.
  
Na lista de filas retornadas pela pesquisa, selecione a fila que você deseja usar para chamadas de tempo limite.
