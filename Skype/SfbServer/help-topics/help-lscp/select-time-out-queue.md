# <a name="select-time-out-queue"></a>Selecionar Fila de Tempo Limite
 
Use a caixa de diálogo **Selecionar uma fila de tempo limite** para selecionar uma fila na qual as chamadas são encaminhadas quando o tempo limite é ultrapassado.
  
Na caixa de pesquisa, digite todo ou parte do nome da fila que você deseja usar. Para retornar todas as filas, deixe a caixa de pesquisa vazia e clique no ícone de pesquisa.
  
Na lista de filas retornadas pela pesquisa, selecione a fila que você deseja usar para as chamadas com tempo limite ultrapassado.
