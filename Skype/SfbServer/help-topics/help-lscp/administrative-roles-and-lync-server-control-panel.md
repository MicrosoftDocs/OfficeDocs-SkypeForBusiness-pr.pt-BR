# <a name="administrative-roles-and-lync-server-control-panel"></a>Funções Administrativas e Painel de Controle Lync Server

Para permitir que você delegar tarefas administrativas enquanto mantém altos padrões de segurança, Skype for Business Server dá suporte ao controle de acesso baseado em função (RBAC). Com o RBAC, você concede privilégio administrativo atribuindo usuários a funções administrativas predefinidos. Skype for Business Server inclui um conjunto rico de funções administrativas e também permite que você crie novas funções e especifique uma lista personalizada de cmdlets para cada nova função. Você pode também adicionar scripts de cmdlets às tarefas permitidas para ambas as funções personalizadas e predefinidas do RBAC. Para obter detalhes, incluindo as funções administrativas predefinida disponíveis no Skype for Business Server, consulte Controle de Acesso [Baseado em Função](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-role-based-access-control).
