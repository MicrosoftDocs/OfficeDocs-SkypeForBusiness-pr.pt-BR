Com o RBAC, você concede privilégio administrativo atribuindo usuários a funções administrativas predefinidos. O Skype for Business Server inclui um conjunto rico de funções administrativas e também permite que você crie novas funções e especifique uma lista personalizada de cmdlets para cada nova função. Você pode também adicionar scripts de cmdlets às tarefas permitidas para ambas as funções personalizadas e predefinidas do RBAC. Para obter detalhes, incluindo as funções administrativas predefinida disponíveis no Skype for Business Server, consulte Role-Based Access Control.
