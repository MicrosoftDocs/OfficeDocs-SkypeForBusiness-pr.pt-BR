# <a name="set-pin"></a>Definir PIN

Para gerar o PIN do usuário, selecione **Gerar automaticamente um PIN válido.**

Selecione **Inserir um PIN específico manualmente** para criar seu próprio PIN, então insira um PIN na caixa de texto que atenda às exigências de PIN especificadas em suas definições de política de PIN.

Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o Painel de Controle Skype for Business Server, consulte [Manage Skype for Business Server 2015](../../manage/manage.md).
