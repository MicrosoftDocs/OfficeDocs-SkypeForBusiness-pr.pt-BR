# <a name="select-exchange-um-auto-attendant"></a>Selecionar um Atendedor Automático do UM do Exchange
 
Use a caixa de diálogo **Selecionar UM atendedor automático do Exchange** para selecionar um UM atendedor automático do Exchange que lidará com as chamadas recebidas para um intervalo de números não atribuídos.
  
Na caixa Pesquisar, digite todo ou parte do nome do Exchange UM atendedor automático que você deseja usar. Para retornar todos os disponíveis Exchange atendedores automáticos, deixe a caixa de pesquisa vazia e clique no ícone de pesquisa.
  
Na lista de Exchange atendedores automáticos retornados pela pesquisa, selecione o atendedor automático que você deseja usar para lidar com chamadas de entrada para o intervalo de números não atribuídos.
