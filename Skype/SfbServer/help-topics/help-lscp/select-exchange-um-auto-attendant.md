# <a name="select-exchange-um-auto-attendant"></a>Selecionar um Atendedor Automático do UM do Exchange
 
Use a caixa de diálogo **selecionar atendedor automático do Exchange um** para selecionar um atendedor automático do Exchange um que manipulará as chamadas de entrada para uma faixa de números não atribuídos.
  
Na caixa de pesquisa, digite todo ou parte do nome do atendedor automático do Exchange UM que você deseja usar. Para retornar todos os atendedores automáticos do Exchange UM disponíveis, deixe a caixa de pesquisa vazia e clique no ícone de pesquisa.
  
Na lista de atendedores automáticos de UM do Exchange retornados pela pesquisa, selecione o atendedor automático que você deseja usar para manipular as chamadas de entrada para o intervalo de números não atribuídos.
