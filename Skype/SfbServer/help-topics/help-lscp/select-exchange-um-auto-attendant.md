# <a name="select-exchange-um-auto-attendant"></a>Selecionar Atendimento Automático do UM do Exchange
 
Use a caixa de diálogo **Selecionar Atendedor Automático de UM do Exchange** para selecionar um Atendedor Automático de UM do Exchange que lidará com as chamadas de entrada para um intervalo de números não atribuídos.
  
Na caixa de pesquisa, digite todo ou parte do nome do Atendedor Automático de UM do Exchange que você deseja usar. Para retornar todos os Atendedores Automáticos de UM do Exchange disponíveis, deixe a caixa de pesquisa vazia e clique no ícone de pesquisa.
  
Na lista de Atendedores Automáticos de UM do Exchange retornada pela pesquisa, selecione o Atendedor Automático que você deseja usar para lidar com as chamadas de entrada para o intervalo de números não atribuídos.
