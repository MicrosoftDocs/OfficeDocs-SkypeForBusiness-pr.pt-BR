# <a name="hybrid-manageability-wizard"></a>Assistente de Capacidade de gerenciamento Híbrida

Você pode usar o assistente de Capacidade de Gerenciamento Híbrida para executar algumas das tarefas de configuração necessárias para configurar uma implantação híbrida com o Skype for Business Online. Para obter mais informações sobre como configurar sua implantação para a implantação híbrida, consulte Configurar seu ambiente com o [assistente de Gerenciabilidade Híbrida.](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)
