# <a name="hybrid-manageability-wizard"></a>Assistente de capacidade de gerenciamento de implantação híbrida

Você pode usar o assistente de gerenciamento híbrido para executar algumas das tarefas de configuração necessárias para configurar uma implantação híbrida com o Skype for Business online. Para obter mais informações sobre como configurar sua implantação para híbrido, consulte [configurar seu ambiente com o assistente de gerenciamento híbrido](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
