# <a name="select-regions"></a>Selecionar Regiões
 
Use a caixa de diálogo **Selecionar Regiões** para localizar as regiões disponíveis no seu ambiente. Para usar uma região existente, clique nela na lista e depois em **OK**.
  
> [!NOTE]
> Clique no ícone de busca na caixa de busca sem ter inserido nada para obter todas as regiões disponíveis.
