# <a name="select-regions"></a>Selecionar Regiões
 
Use a caixa de diálogo Selecionar **Regiões** para encontrar regiões disponíveis em seu ambiente. Para usar uma região existente, clique nela na lista e clique em **OK**.
  
> [!NOTE]
> Clique no ícone de busca na caixa de busca sem ter inserido nada para obter todas as regiões disponíveis.
