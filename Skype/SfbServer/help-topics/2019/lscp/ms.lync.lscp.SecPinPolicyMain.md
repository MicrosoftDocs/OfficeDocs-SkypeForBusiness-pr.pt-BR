# <a name="pin-policy"></a>Política de PIN
 
O que você deseja fazer? Clique no link da tarefa que você deseja executar.
  
 **Trabalhar com a política de PIN para telefones IP** Essa diretiva (PIN) número de identificação pessoal suporta autenticação para usuários que se conectam ao Skype para Business Server usando um telefone IP.
- [Criar uma nova política PIN no Skype para Business Server 2015](../../../manage/authentication/create-a-new-pin-policy.md)
    
- [Modificar uma política PIN existente no Skype para Business Server 2015](../../../manage/authentication/modify-an-existing-pin-policy.md)
    
- [Excluir uma política PIN no Skype para Business Server 2015](../../../manage/authentication/delete-a-pin-policy.md)
    
 **Trabalhar com política PIN para conferência discada** Essa política PIN permite que os usuários que têm credenciais do Active Directory Domain Services em sua organização para ingressar em conferências discadas como usuários autenticados, inserindo um PIN.
- [Criar ou modificar as configurações de PIN de conferência discada para um Site ou grupo de usuários](http://technet.microsoft.com/library/c29bab5c-2b93-48e0-ae0b-29564daaff9a.aspx)
    
- [Modifique as configurações de PIN de conferência discada padrão](http://technet.microsoft.com/library/2d110e94-ad29-4755-b17f-d8c2da9b78a4.aspx)
    
- [Excluir configurações de PIN de conferência discada para um Site ou grupo de usuários](http://technet.microsoft.com/library/15a9faee-d024-4c0e-b2a0-fe7e7dc00589.aspx)
    
Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o Skype para o painel de controle do Business Server, consulte [Gerenciar Skype para Business Server 2015](../../../manage/manage.md).
