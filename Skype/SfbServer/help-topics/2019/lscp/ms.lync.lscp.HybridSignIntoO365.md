# <a name="hybrid-wizard-sign-in-to-microsoft-365-or-office-365"></a>Assistente híbrido: entre no Microsoft 365 ou office 365

Use o assistente de Gerenciamento Híbrido do Skype for Business Server para ajudá-lo a configurar seu ambiente para híbrido com o Skype for Business Online.

## <a name="sign-in-to-microsoft-365-or-office-365"></a>Entrar no Microsoft 365 ou Office 365

Insira o nome de usuário e a senha para sua organização do Microsoft 365 ou Office 365 para fazer logon e conectar sua implantação local com sua organização do Microsoft 365 ou Office 365. Você precisará usar a conta de administrador que foi criada para o domínio onmicrosoft quando você se inscreveu para o Microsoft 365 ou o Office 365, por exemplo, admin@contoso.onmicrosoft.com.

Para obter mais informações sobre como configurar seu ambiente para híbrido, consulte [Overview of hybrid deployments](/previous-versions/office/lync-server-2013/lync-server-2013-overview-of-hybrid-deployments).
