# <a name="select-exchange-um-auto-attendant"></a>Selecione o atendedor automático do Exchange UM

> [!NOTE]
> UM do Exchange permanece disponível na Skype para Business Server 2019 ao integrar Skype para negócios 2019 com Exchange 2013 ou 2016 do Exchange. Devido às alterações no suporte no Exchange 2019, a integração de UM do Exchange está sendo desprovisionamento emphasised em favor de recursos de caixa postal de nuvem e atendedor automático de nuvem.  Consulte o [serviço de caixa postal de nuvem planejar](../../../../SfBServer2019/hybrid/plan-cloud-voicemail.md) [serviço caixa postal de nuvem planejar] e [Planejar Skype para Business Server e migração do Exchange Server](../../../../SfBServer2019/hybrid/plan-um-migration.md) (… /.. /.. / SfBServer2019/hybrid/plan-um-migration.md) para obter mais informações.
 
Use a caixa de diálogo **Selecionar UM atendedor automático do Exchange** para selecionar um UM atendedor automático do Exchange que lidará com as chamadas recebidas para um intervalo de números não atribuídos.
  
Na caixa Pesquisar, digite todo ou parte do nome do Exchange UM atendedor automático que você deseja usar. Para retornar todos os disponíveis Exchange atendedores automáticos, deixe a caixa de pesquisa vazia e clique no ícone de pesquisa.
  
Na lista de Exchange atendedores automáticos retornados pela pesquisa, selecione o atendedor automático que você deseja usar para lidar com chamadas de entrada para o intervalo de números não atribuídos.
