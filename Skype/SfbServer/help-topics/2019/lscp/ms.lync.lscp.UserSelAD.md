# <a name="select-from-active-directory"></a>Selecionar no Active Directory
 
Use a caixa **de diálogo Selecionar do Active Directory** para localizar os usuários que você deseja habilitar para o Skype for Business Server. Na caixa de pesquisa, digite o endereço SIP ou o nome para exibição do usuário que você deseja localizar. Para recuperar todos os usuários disponíveis, não digite nada na caixa de pesquisa.
