# <a name="select-contacts-from-active-directory"></a>Selecionar Contatos no Active Directory
 
Use a caixa **de diálogo Selecionar do Active Directory** para localizar os contatos que você deseja habilitar no Skype for Business. Na caixa de pesquisa, digite o endereço SIP ou o nome para exibição do contato que deseja localizar. Para recuperar todos os contatos disponíveis, deixe a caixa de pesquisa vazia.
