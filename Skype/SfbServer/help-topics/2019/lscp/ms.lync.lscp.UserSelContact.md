# <a name="select-contacts-from-active-directory"></a>Selecionar Contatos no Active Directory
 
Use a caixa de diálogo **Selecionar no Active Directory** para localizar contatos que você deseja habilitar para Skype para negócios. Na caixa de pesquisa, digite o endereço SIP ou o nome para exibição do contato que deseja localizar. Para recuperar todos os contatos disponíveis, deixe a caixa de pesquisa vazia.
