# <a name="select-an-announcement-service"></a>Selecionar um serviço de comunicado
 
Use a caixa de diálogo **Selecionar um serviço de comunicado** para selecionar um serviço de anúncio que lidará com as chamadas recebidas para um intervalo de números não atribuídos.
  
Na caixa Pesquisar, digite todo ou parte do nome do serviço anúncio que você deseja usar. Para retornar todos os serviços de anúncio disponíveis, deixe a caixa de pesquisa vazia e clique no ícone de pesquisa.
  
Na lista de serviços de anúncio retornada pela pesquisa, selecione o serviço que você deseja usar para lidar com chamadas de entrada para o intervalo de números não atribuídos.
