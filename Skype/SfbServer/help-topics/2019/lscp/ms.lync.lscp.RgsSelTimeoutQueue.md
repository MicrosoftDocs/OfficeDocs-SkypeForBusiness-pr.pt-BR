# <a name="select-time-out-queue"></a>Selecionar Fila de Tempo Limite
 
Use a caixa de diálogo **selecionar uma fila de tempo limite** para selecionar uma fila onde as chamadas são encaminhadas quando elas se esgotam.
  
Na caixa de pesquisa, digite todo ou parte do nome da fila que você deseja usar. Para retornar todas as filas disponíveis, deixe a caixa de pesquisa vazia e clique no ícone de pesquisa.
  
Na lista de filas retornadas pela pesquisa, selecione a fila que você deseja usar para chamadas com tempo limite.
