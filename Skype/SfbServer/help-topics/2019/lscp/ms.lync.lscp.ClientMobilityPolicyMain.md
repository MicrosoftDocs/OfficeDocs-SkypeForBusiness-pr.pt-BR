# <a name="clients-mobility-policy"></a>Clientes: Política de mobilidade
 
A página de **Política de mobilidade** de **clientes** exibe as políticas de mobilidade disponíveis em sua implantação.
  
## <a name="tasks-you-can-perform"></a>Tarefas que podem ser executadas

Você pode executar as seguintes tarefas na página **Política de mobilidade** :
  
- Exibir políticas de mobilidade existente
    
- Criar uma nova política de mobilidade
    
- Editar uma política de mobilidade
    
## <a name="ui-reference"></a>Referência de UI

As listas a seguir descrevem os menus, comando, campos e propriedades na página.
  


- **Campo de pesquisa** Digite uma cadeia de caracteres na caixa de texto para filtrar as políticas exibidas.
    
- **Novo** Clique em **novo**e selecione **Site** ou **usuário** para criar uma nova política de mobilidade.
    
- **Editar** Selecione uma política e clique em **Editar** para editar as propriedades da política selecionada.
    
- **Atualizar** Atualiza a lista de políticas exibidas. Quando você clicar em **Atualizar**, a cadeia de caracteres de filtro de pesquisa está desmarcada e todas as diretivas existentes são exibidas.
    
Para obter detalhes sobre como configurar políticas de mobilidade, consulte [Configuring Mobility Policy](http://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx).
