# <a name="prefix-edit"></a>Edição de prefixo
 
Para definir os prefixos ou iniciando dígitos, adicione os prefixos de número que essa rota deve lidar, incluindo o + se necessário. Por exemplo, você pode corresponder todos os números, começando com +123.
  
Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o Skype para o painel de controle do Business Server, consulte [Gerenciar Skype para Business Server 2015](../../../manage/manage.md).
