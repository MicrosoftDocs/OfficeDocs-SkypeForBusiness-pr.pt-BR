# <a name="select-routes"></a>Selecionar Rotas
 
Use a caixa de diálogo **selecionar rotas** para localizar as rotas de voz existentes que você pode usar.
  
> [!NOTE]
> Para recuperar todas as rotas de voz disponíveis, não digite nada na caixa de pesquisa e, em seguida, clique no ícone de pesquisa.
