# <a name="select-trunk"></a>Selecionar Tronco
 
Use a caixa de diálogo **selecionar tronco** para localizar um tronco existente que você pode usar.
  
> [!NOTE]
> Para recuperar todos os troncos disponíveis, não digite nada na caixa de pesquisa e, em seguida, clique no ícone de pesquisa.
