# <a name="select-a-region-link"></a>Selecionar um Link de Região
 
Use a caixa de diálogo **Link de Região** para localizar links de região disponíveis no seu ambiente.  Para usar um link de região existente, clique em um link de região na lista e depois em **OK**.
  
> [!NOTE]
> Clique no ícone de busca sem inserir nada na caixa de busca para obter todos os links de região disponíveis. 
  
Para obter detalhes sobre os diferentes procedimentos que você pode executar usando o Skype para o painel de controle do Business Server, consulte [Gerenciar Skype para Business Server 2015](../../../manage/manage.md).
