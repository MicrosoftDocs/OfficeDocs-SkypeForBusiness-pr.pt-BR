# <a name="assign-properties"></a>Atribuir Propriedades
 
Use a caixa de diálogo **atribuir propriedades** para especificar uma descrição e uma ID de site de rede para várias sub-redes ao mesmo tempo. Em **Descrição**, digite a descrição que você deseja usar para as sub-redes que você selecionou. Em **ID de site de rede**, selecione a identificação de site de rede que você deseja atribuir às subredes que você selecionou. Para atribuir essas propriedades às sub-redes selecionadas, clique em **OK**.
