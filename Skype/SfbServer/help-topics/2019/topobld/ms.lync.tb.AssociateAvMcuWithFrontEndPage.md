# <a name="associate-av-mcu-with-front-end"></a>Associar AV MCU com Front End
 
Todos os pools de Front End do Enterprise Edition com a conferência habilitado devem ter um A / o serviço de conferência V colocado, ou deve usar uma stand-alone A / pool de conferência V. Qualquer existente Enterprise Edition pools Front-End que possuírem um A / conferência V colocado de serviço podem ser selecionada para usar este nova stand-alone A / pool de conferência de V em vez disso.
