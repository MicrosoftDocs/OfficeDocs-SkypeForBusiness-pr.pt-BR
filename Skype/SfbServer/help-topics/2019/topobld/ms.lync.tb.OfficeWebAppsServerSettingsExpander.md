<span data-ttu-id="65f41-105">Você editar as propriedades de Office Web Apps Server configurado.</span><span class="sxs-lookup"><span data-stu-id="65f41-105">You edit the properties of the configured Office Web Apps Server.</span></span> <span data-ttu-id="65f41-106">Estas propriedades estão disponíveis para edição:</span><span class="sxs-lookup"><span data-stu-id="65f41-106">The following properties are available to edit:</span></span>

 <span data-ttu-id="65f41-107">**FQDN do servidor do Office Web Apps**: essa propriedade define o nome de domínio totalmente qualificado do servidor Office Web Apps e deve corresponder a um sistema de nomes de domínio (DNS) host A ou AAAA (se estiver sendo usado o IPv6) registro.</span><span class="sxs-lookup"><span data-stu-id="65f41-107">**Office Web Apps Server FQDN**: This property defines the fully qualified domain name of the Office Web Apps Server and should match a domain name system (DNS) host A or AAAA (if IPv6 is being used) record.</span></span>
