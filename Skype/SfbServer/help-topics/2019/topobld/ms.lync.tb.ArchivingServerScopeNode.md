# <a name="archiving-server-scope-node"></a>Nó de Escopo de Servidor de Arquivamento
 
No construtor de topologia, você pode executar tarefas relacionadas ao servidor que está executando o arquivamento clicando com o botão direito do mouse no pool na árvore do console e selecionando **Editar propriedades**. Na página **Editar Propriedades** é possível alterar o nome de domínio totalmente qualificado (FQDN) do servidor que está executando o Arquivamento, o banco de dados SQL Server de arquivamento e o repositório de arquivos de arquivamento.
