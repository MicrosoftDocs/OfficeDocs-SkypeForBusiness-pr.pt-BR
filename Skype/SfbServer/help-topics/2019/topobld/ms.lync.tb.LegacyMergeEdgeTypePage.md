# <a name="legacy-merge-edge-type"></a>Tipo de Borda de Mesclagem de Legado
 
Para um servidor de **borda único** ou um servidor **de borda consolidada para carga balanceada** , clique em **Avançar** para especificar informações de configuração. Para uma **borda expandida**, a configuração de cada servidor deve ser alterada para uma configuração de borda consolidada antes de poder ser mesclada.
