# <a name="legacy-merge-edge-type"></a>Tipo de Borda de Mesclagem de Legado
 
Para um servidor **Único servidor de borda** ou **com balanceamento de carga de borda consolidada** , clique em **próximo** para especificar informações de configuração. Para uma **Borda expandida**, cada configuração de servidor deve ser alterada para uma configuração de borda consolidada, antes que ele possa ser mesclado.
