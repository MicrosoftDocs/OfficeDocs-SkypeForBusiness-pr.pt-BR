# <a name="legacy-merge-edge-type"></a>Tipo de borda de mesclagem de legado
 
Para um servidor **Único servidor de borda** ou **com balanceamento de carga de borda consolidada** , clique em **próximo** para especificar informações de configuração. Para uma **Borda expandida**, cada configuração de servidor deve ser alterada para uma configuração de borda consolidada, antes que ele possa ser mesclado com o Lync Server 2013.
