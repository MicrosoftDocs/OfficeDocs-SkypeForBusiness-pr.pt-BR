# <a name="legacy-merge-landing"></a>Patamar de Mesclagem de Legado
 
A página **Especificar configuração de borda** permite configurar as informações do servidor de borda herdadas. Clique em **Adicionar** para especificar a versão herdada do servidor de borda da sua implantação do Office Communications Server. Se você não tiver nenhum servidor de borda no seu ambiente herdado, clique em **Avançar** para continuar.
