# <a name="add-monitoring-server"></a>Adicionar Servidor de Monitoramento

Para incorporar um Monitoring Server ao seu design de topologia, você deve especificar o FQDN (nome de domínio totalmente qualificado) do servidor no qual deseja implantar o Monitoring Server. Antes de publicar uma topologia que inclui o Monitoring Server e instalar o Skype for Business Server no servidor, você deve concluir todos os pré-requisitos para implantar o monitoramento. Para obter detalhes sobre os pré-requisitos, consulte  [Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) na documentação de Implantação.
