# <a name="add-mediation-server-nexthop"></a>Adicionar NextHop de Servidor de Mediação
 
Para definir o próximo salto para um Servidor de  Mediação que você está definindo, selecione na lista de espera de pool de próximo salto o servidor Edição Enterprise ou o servidor Edição Standard que atuará como o próximo salto para o Servidor de Mediação. Clique em **Avançar** para continuar.
