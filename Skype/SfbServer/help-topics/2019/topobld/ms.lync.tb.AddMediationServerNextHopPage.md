# <a name="add-mediation-server-nexthop"></a>Adicionar NextHop de Servidor de Mediação
 
Para definir o próximo salto para um Servidor de Mediação que você está definindo, selecione na lista drop-down do **pool** de próximo salto o servidor Enterprise Edition ou o servidor Standard Edition que atuará como o próximo salto para o Servidor de Mediação. Clique em **Avançar** para continuar.
