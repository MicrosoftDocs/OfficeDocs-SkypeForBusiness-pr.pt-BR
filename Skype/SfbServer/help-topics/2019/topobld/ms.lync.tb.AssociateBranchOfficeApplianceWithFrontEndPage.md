# <a name="associate-branch-office-appliance-with-front-end"></a>Associar Aparelho de Filial com Front-end
 
Para associar o Aparelho de Filial Survivível ou Servidor de Filial Survivível a um pool de Front-End ou um servidor Standard Edition, na página Selecionar o pool de **Front-End,** selecione o pool de Front-End ou um servidor Standard Edition na lista de lista drop-down do pool de **Front-End.** Clique em **Avançar** para continuar.
