# <a name="add-default-sip-domain"></a>Adicionar Domínio SIP Padrão
 
Especifique o domínio SIP (protocolo de início de sessão) primário para o qual você deseja que a implantação dê suporte. O domínio SIP principal pode ser igual ou diferente do domínio dos serviços de domínio Active Directory. Por exemplo, seu domínio do Active Directory pode ser fabrikam.net, e você pode especificar seu domínio SIP primário para ser contoso.net.
