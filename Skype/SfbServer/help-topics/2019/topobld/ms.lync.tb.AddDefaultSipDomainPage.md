# <a name="add-default-sip-domain"></a>Adicionar Domínio SIP Padrão
 
Especifique o domínio primário do protocolo de iniciação de sessão (SIP) que você deseja para dar suporte a sua implantação. O domínio SIP primário pode ser igual, ou diferente de, seu domínio do Active Directory Domain Services. Por exemplo, seu domínio do Active Directory poderia ser fabrikam.net e você pode especificar seu domínio SIP primário para ser contoso.net.
