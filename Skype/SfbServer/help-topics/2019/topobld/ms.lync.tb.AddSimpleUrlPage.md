# <a name="add-simple-url"></a>Adicionar URL Simples
 
Para adicionar outras URLs simples à definição da implantação, especifique a URL da nova URL simples. O formato da URL é _http://\<tipo\>de URL simples.\< \> nome do domínio_ ou _\<tipo\>de\>URL simples do/\<nome do domínio http://_, onde o tipo de URL simples é um dos seguintes:
  
- Atendimento
    
- Discar
    
> [!IMPORTANT]
> Você pode ter uma URL de discagem simples e uma única URL simples ativa a qualquer momento. Adicionar uma nova URL simples e fazer com que a URL ativa defina todas as solicitações futuras para usar a nova URL simples. Você deixa a URL simples não ativa para manipular conferências e outras solicitações que foram feitas quando a URL simples mais antiga estava ativa.
