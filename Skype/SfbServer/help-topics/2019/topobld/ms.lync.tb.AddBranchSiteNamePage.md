# <a name="add-branch-site-name"></a>Adicionar Nome de Site de Filial

Para definir um novo local de filial, especifique o seguinte:

Um nome para o novo site de filial. O nome que você definir deve ser exclusivo em sua implantação. Qualquer convenção de nomenclatura decidida por você pode ser usada. Recomendações comuns são por cidade, nome de local, nome de escritório de filial. O valor nome é uma entrada obrigatória.

O campo descrição, que é opcional, é um espaço onde você especifica informações descritivas adicionais que não serão parte do nome de infraestrutura, mas sim um local para inserir texto e detalhar seu site mais ainda.

Para obter detalhes, consulte [Sites](https://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx).
