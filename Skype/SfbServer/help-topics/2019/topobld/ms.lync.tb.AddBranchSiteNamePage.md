# <a name="add-branch-site-name"></a>Adicionar Nome de Site de Filial

Para definir um novo site de filial, especifique o seguinte:

Um nome para o novo site de filial. O nome que você deseja definir deve ser exclusivo em sua implantação. Qualquer convenção de nomenclatura decidida por você pode ser usada. Recomendações comuns são por cidade, nome de local, nome de escritório de filial. O valor nome é uma entrada exigida.

O campo descrição, que é opcional, oferece um local para você especificar informações descritivas adicionais que não serão parte do nome de infraestrutura, mas oferece um local para texto para descrever seu site mais a fundo.

Para obter detalhes, consulte [Sites](/previous-versions/office/lync-server-2013/lync-server-2013-sites).
