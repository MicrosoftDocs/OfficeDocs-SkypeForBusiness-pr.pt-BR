# <a name="add-edge-server-support"></a>Adicionar suporte ao servidor de borda
 
Você pode adicionar suporte para acesso de usuário externo ao implantar a topologia inicial, ou posteriormente. Para obter detalhes sobre como adicionar servidores de borda a uma topologia existente, consulte [Define Your Edge Topology](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) na documentação de implantação de servidor de borda.
