# <a name="front-end-start-page"></a>Página de Início do Front-end

Use este assistente para definir um novo pool de Front-Ends para um site. Cada site deve ter pelo menos um pool Enterprise Edition ou um servidor Standard Edition. Para obter detalhes sobre os requisitos de utilização do assistente para criar um novo pool de Front- Ends, consulte  [Define and Configure a Front End Pool](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) na documentação de Implantação.
