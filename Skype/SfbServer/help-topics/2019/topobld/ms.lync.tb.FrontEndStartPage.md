# <a name="front-end-start-page"></a>Página de Início do Front End

Use este assistente para definir um novo pool de front-ends para um site. Cada site deve ter pelo menos um pool de edição empresarial ou um servidor Standard Edition. Para obter detalhes sobre os requisitos para usar o assistente para criar um novo pool de front-end, consulte [definir e configurar um pool de front-ends](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) na documentação de implantação.
