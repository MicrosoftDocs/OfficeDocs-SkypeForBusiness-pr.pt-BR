# <a name="front-end-start-page"></a>Página de Início do Front-end

Use este assistente para definir um novo pool de Front-Ends para um site. Cada site deve ter pelo menos um pool Enterprise Edition ou um servidor Standard Edition. Para obter detalhes sobre os requisitos de utilização do assistente para criar um novo pool de Front- Ends, consulte  [Define and Configure a Front End Pool](/previous-versions/office/lync-server-2013/lync-server-2013-define-and-configure-a-front-end-pool-or-standard-edition-server) na documentação de Implantação.
