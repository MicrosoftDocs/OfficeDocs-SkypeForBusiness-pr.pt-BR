# <a name="define-the-pstn-gateway-fqdn"></a>Definir o Gateway PSTN do FQDN

Você especifica o nome de domínio totalmente qualificado (FQDN) do gateway da rede de telefonia pública comutada (PSTN) que conectará seus componentes de voz empresarial a um PBX ou PSTN (Private Branch Exchange).

Para obter detalhes, consulte [componentes de conectividade PSTN no Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)
