# <a name="legacy-merge-add-expanded-edge"></a>Borda Expandida de Adição de Mesclagem de Legado
 
O Skype for Business Server não dá suporte à mesclação de uma configuração herdada do Servidor de Borda expandido do Office Communications Server 2007 R2. O Servidor de Borda expandido deve primeiro ser convertido em uma configuração de Servidor de Borda consolidada antes de mesclar as informações do Servidor de Borda.
