# <a name="add-archiving-server"></a>Adicionar Servidor de Arquivamento
 
Para implantar um Servidor de Arquivamento ao design da sua topologia, você precisa especificar o FQDN (nome de domínio totalmente qualificado) do servidor no qual deseja implantar o Arquivamento. Antes de publicar uma topologia que inclui o arquivamento e instalando Skype para Business Server, você deve concluir todos os pré-requisitos para a implantação de arquivamento.
