# <a name="test---skype-for-business-server-site-scope-node"></a>Nó de Escopo de Site do Servidor Lync
 
Este nó mostra um dos sites definidos na sua topologia. É possível realizar as seguintes ações: 
  
- **Novo.** Adiciona um novo pool ou servidor ao site.
    
- **Editar propriedades.** Edita propriedades do site, como nome e local, e especifica se o site utiliza controle de admissão de chamadas para gerenciamento de largura de banda e se um pool de Diretor ou Borda desse site é usado como rota de federação.
    
- **Excluir.** Remove o site da topologia.
    
Além dessas opções, você pode realizar as seguintes tarefas gerais:
  
- **Topologia.** É possível usar essa opção para definir uma nova topologia, abrir uma topologia existente, baixar a topologia atual, salvar uma cópia da topologia, publicar a topologia, instalar um banco de dados, mesclar os componentes ou remover a implantação.
    
- **Exibir.** Você pode usar essa opção para personalizar o modo de exibição no Construtor de Topologias selecionando os itens que deseja mostrar ou ocultar nos modos de exibição da árvore de console.
    
- **Ajuda.** Você pode usar essa opção para acessar a Ajuda do nó ativo.
