# <a name="test---skype-for-business-server-site-scope-node"></a>Teste - Site Nó de Escopo Raiz do Skype for Business Server
 
Este nó mostra um dos sites definidos na sua topologia. É possível realizar as seguintes ações: 
  
- **Novo.** Adiciona um novo pool ou servidor ao site.
    
- **Editar propriedades.** Edita propriedades do site tais como nome e local, especifica se o site utiliza controle de admissão de chamadas para gerenciamento de largura de banda e se um pool de Diretor ou Borda neste site é usado como rota de federação.
    
- **Excluir.** Remove este site da topologia.
    
Além destas opções você pode realizar as seguintes tarefas gerais:
  
- **Topologia.** Você pode usar essa opção para definir uma nova topologia, abrir uma topologia existente, baixar a topologia atual, salvar uma cópia da topologia, publicar a topologia, instalar um banco de dados, mesclar componentes ou remover a implantação.
    
- **Visualização.** Você pode usar esta opção para personalizar a visualização no Construtor de Topologia selecionando os itens que serão exibidos ou escondidos na visão de árvore de comando.
    
- **Ajuda.** Você pode usar esta opção para acessar a Ajuda para o nó ativo.
