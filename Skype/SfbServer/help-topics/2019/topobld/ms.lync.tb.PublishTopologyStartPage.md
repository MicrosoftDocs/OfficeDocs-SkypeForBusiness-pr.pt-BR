# <a name="publish-topology"></a>Publicar Topologia
 
A página **Publicar a topologia** é exibida quando você inicia o processo de publicação. Leia e confirme que as tarefas listadas estão concluídas antes de prosseguir. Quando estiver pronto para prosseguir, clique em **Avançar**. Se for preciso fechar o processo de publicação, clique em **Cancelar**.
