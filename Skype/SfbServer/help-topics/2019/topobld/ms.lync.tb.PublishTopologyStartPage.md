# <a name="publish-topology"></a>Publicar Topologia
 
A página **publicar a topologia** é exibida quando você inicia o processo de publicação. Leia e confirme se as tarefas listadas estão completas antes de prosseguir. Quando estiver pronto para continuar, clique em **Avançar**. Se você precisar sair do processo de publicação, clique em **Cancelar**.
