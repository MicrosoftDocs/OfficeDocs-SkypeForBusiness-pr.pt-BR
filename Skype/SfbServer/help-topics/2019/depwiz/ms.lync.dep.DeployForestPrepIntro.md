# <a name="prepare-current-forrest-intro"></a>Preparar Floresta Atual (Introdução)
 
Para começar a tarefa **Preparar Floresta**, clique em **Avançar**. Executar a tarefa **preparar floresta** cria configurações globais e cria grupos universais para uso pelo Skype for Business Server.
