# <a name="prepare-current-forrest-intro"></a>Preparar Floresta Atual (Introdução)
 
Para começar a tarefa **Preparar Floresta**, clique em **Avançar**. A execução **da tarefa Preparar Floresta** cria configurações globais e cria grupos universais para uso por Skype for Business Server.
