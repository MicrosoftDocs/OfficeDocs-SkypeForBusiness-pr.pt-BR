# <a name="prepare-current-forrest-intro"></a>Preparar Floresta Atual (Introdução)
 
Para começar a tarefa **Preparar Floresta**, clique em **Avançar**. Executando a tarefa de **Preparar floresta** cria configurações globais e grupos universais para uso por Skype para Business Server.
