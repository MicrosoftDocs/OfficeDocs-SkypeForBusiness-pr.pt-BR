# <a name="prepare-current-forrest-intro"></a>Preparar Floresta Atual (Introdução)
 
Para começar a tarefa **Preparar Floresta**, clique em **Avançar**. A execução **da tarefa Preparar** Floresta cria configurações globais e grupos universais para uso pelo Skype for Business Server.
