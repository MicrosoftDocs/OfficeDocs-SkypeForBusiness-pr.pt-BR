# <a name="startservices-intro"></a>Iniciar Serviços (Introdução)
 
Para iniciar serviços no servidor de função do Skype for Business Server que você está implantando, revise a tela de introdução. Observe que os comandos emitidos para iniciar o Skype for Business Server não aguardam o início do serviço. Esse é o melhor esforço para iniciar os serviços, e você deve monitorar o status real dos serviços usando o Console de Gerenciamento Microsoft (MMC) dos Serviços do Windows. Clique em **Avançar** para iniciar a tarefa **Iniciar Serviços**.
