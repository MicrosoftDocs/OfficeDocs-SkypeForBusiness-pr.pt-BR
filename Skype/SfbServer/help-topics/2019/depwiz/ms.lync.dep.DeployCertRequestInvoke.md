# <a name="certificate-request-invoke"></a>Solicitação de Certificado (Invocar)
 
A página **comandos em execução** exibe um painel de status da tarefa como os comandos são executados para a geração da solicitação de certificado. Após a conclusão das tarefas, a lista suspensa abaixo do painel Resumo contém uma lista de arquivos de log. Use a lista suspensa para selecionar um arquivo de log para exibir e, em seguida, clique em **Exibir log**. O Internet Explorer será iniciado e exibirá o arquivo de log. Em caso de falha de solicitação, você deve revisar os arquivos de log para determinar a causa da falha. Você pode executar novamente a solicitação de certificado para resolver o problema, corrigindo o problema indicado no arquivo de log. Continue embora as páginas de solicitação de certificado e solicitem o certificado novamente.
