# <a name="install-or-update-skype-for-business-server-system"></a>Instalar ou Atualizar o Sistema Skype for Business Server

Para começar a instalação ou atualizar uma implantação existente do Skype for Business Server, clique em **instalar ou atualizar o Skype for Business Server**. As etapas nesta seção instalarão os componentes específicos de uma determinada função de servidor, com base no documento do construtor de topologias. Antes de iniciar esta série de etapas, você deve definir a implantação no construtor de topologias. Você deve estar conectado ao computador que está sendo implantado como administrador local e como membro do grupo Domain admins no domínio onde o servidor está localizado. O computador deve estar associado ao domínio antes de concluir esta etapa.

Se você estiver atualizando um servidor atualmente implantado, será necessário fazer as alterações que você pretende fazer no construtor de topologias.
