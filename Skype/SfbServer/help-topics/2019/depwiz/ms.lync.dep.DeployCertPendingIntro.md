# <a name="pending-certificate-intro"></a>Certificado Pendente (Introdução)
 
Para processar uma solicitação de certificado online pendente, ou seja, uma solicitação que foi emitida para sua autoridade de certificação online, mas está definida administrativamente para exigir um processo de aprovação manual, clique em **Avançar**.
