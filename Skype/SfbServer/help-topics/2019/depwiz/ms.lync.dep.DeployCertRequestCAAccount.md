# <a name="certificate-request-certificate-authority-account"></a>Solicitação de Certificado (Conta da Autoridade de Certificação)
 
Para enviar uma solicitação, sua autoridade de certificação (CA) pode exigir credenciais diferentes das usadas para fazer o logon atual. Para permitir uma solicitação de certificado como um usuário diferente, marque a caixa de seleção Especificar credenciais alternativas para a autoridade de certificação e digite o nome de usuário ou nome de usuário do domínio de um usuário que pode solicitar o \   certificado. Na caixa de texto **Senha**, digite a senha do usuário especificado. O nome de usuário e senha são enviados como parte do processo de solicitação do certificado, mas não na solicitação de certificado real, para uma CA online.
