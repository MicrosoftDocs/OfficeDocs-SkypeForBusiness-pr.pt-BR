# <a name="sql-server-reporting-services-credentials"></a>SQL Server Reporting Services (Credenciais)
 
Para especificar as credenciais para o SQL Server Reporting Services (SSRS) para acessar o banco de dados de monitoramento, faça o seguinte:
  
Na **Página especificar de credencial**, especifique as credenciais de usuário que você deseja usar para acessar o banco de dados de monitoramento (QoEMetrics e LcsCDR). 
  
> [!NOTE]
> Para obter detalhes, consulte "especificando credenciais e Conexão informações para relatório de fontes de dados (SSRS)" em [https://go.microsoft.com/fwlink/p/?LinkId=263431](https://go.microsoft.com/fwlink/p/?LinkId=263431). 
  
Para obter detalhes sobre como implantar os relatórios do Reporting Services para o servidor executando o monitoramento, consulte [Instalar o Monitoring Reports no Skype para Business Server](../../../deploy/deploy-monitoring/install-monitoring-reports.md).
