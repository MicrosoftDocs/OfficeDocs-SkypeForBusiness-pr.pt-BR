# <a name="sql-server-reporting-services-credentials"></a>SQL Server Reporting Services (Credenciais)
 
Para especificar as credenciais do SQL Server Reporting Services (SSRS) para acessar o banco de dados de monitoramento, faça o seguinte:
  
Na **página especificar credencial**, especifique as credenciais do usuário que você deseja usar para acessar o banco de dados de monitoramento (QoEMetrics e LcsCDR). 
  
> [!NOTE]
> Para obter detalhes, consulte "especificando as informações de credenciais e conexão para fontes de dados de relatório [https://go.microsoft.com/fwlink/p/?LinkId=263431](https://go.microsoft.com/fwlink/p/?LinkId=263431)(SSRS)" em. 
  
Para obter detalhes sobre a implantação dos relatórios do Reporting Services para o servidor que está executando o monitoramento, consulte [instalar relatórios de monitoramento no Skype for Business Server](../../../deploy/deploy-monitoring/install-monitoring-reports.md).
