# <a name="start-services"></a>Iniciar serviços
 
Para iniciar os serviços na função de servidor implantado, clique em **Executar** no **etapa 4: iniciar serviços**. Você deve estar conectado como membro do grupo Administradores local ou um equivalente. 
  
> [!NOTE]
> Se **Iniciar serviços** tiver sido executado, você pode executar novamente o comando clicando em **Executar novamente**.
