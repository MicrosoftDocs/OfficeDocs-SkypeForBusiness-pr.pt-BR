# <a name="certificate-assignment-invoke"></a>Atribuição de Certificado (Invocar)
 
A página **comandos em execução** exibe as ações executadas pelas tarefas para atribuir o certificado ao servidor. Quando as tarefas estiverem concluídas, você poderá exibir o log que detalha o processo de atribuição de certificado. Para exibir o arquivo de log, clique em **Exibir Log**.
