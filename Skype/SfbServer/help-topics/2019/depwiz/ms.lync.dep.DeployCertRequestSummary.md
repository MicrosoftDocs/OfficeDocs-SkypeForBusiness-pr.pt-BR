# <a name="certificate-request-summary"></a>Solicitação de Certificado (Resumo)
 
A página **Resumo da Solicitação de Certificado** exibe um resumo detalhado de todas as opções e configurações para o certificado. O resumo representa o que a solicitação de certificado conterá, e subsequentemente, como o certificado será emitido. Revise com atenção o resumo para se certificar  de que todas as outras entradas e opções selecionadas estão corretas. Se suas seleções estiverem corretas, clique em **Avançar** para continuar com a geração da solicitação e prepará-la para envio para a Autoridade de certificação (CA).
