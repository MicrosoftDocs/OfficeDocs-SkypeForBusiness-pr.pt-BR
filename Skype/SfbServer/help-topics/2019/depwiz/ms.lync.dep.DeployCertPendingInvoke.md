# <a name="pending-certificate-invoke"></a>Certificado pendente (invocar)
 
Página **Executando comandos** exibe o status dos comandos que estiver recuperando o certificado pendente. Quando a tarefa for concluída, o arquivo de log de **Recuperar certificados pendente** está disponível. Clique em **Exibir Log** para revisar o processamento e o status de conclusão da tarefa. Clique em **Avançar** para continuar.
  
> [!IMPORTANT]
> Se o certificado ainda não foi aprovado pelo administrador de autoridade de certificação, o arquivo de log de solicitação pendente recuperar indicará se houver solicitações pendentes. Se o administrador da autoridade de certificação negou a solicitação, o arquivo de log indicará uma falha e informará o erro, indicando que a solicitação foi negada e o nome de usuário da pessoa que negou a solicitação.
