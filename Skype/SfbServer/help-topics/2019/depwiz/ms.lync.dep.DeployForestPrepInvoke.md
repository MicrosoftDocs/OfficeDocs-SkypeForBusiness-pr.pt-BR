Under the summary pane, you should see **Task status: Completed**. In the drop-down list, the log file **Prepare Forest** should be visible. If it is not, select the log file and then click **View Log** to review the detailed actions and outcome of the tasks. Click **Finish** to complete the preparation of the forest.</span></span>
