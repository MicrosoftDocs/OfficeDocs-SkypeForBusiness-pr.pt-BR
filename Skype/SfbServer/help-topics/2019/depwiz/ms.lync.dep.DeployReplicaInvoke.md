# <a name="install-local-configuration-store-invoke"></a>Instalar Repositório de Configuração Local (Invocar)
 
O **painel Executar Comandos** exibe o status das tarefas para instalar a réplica somente leitura local do armazenamento de Gerenciamento Central. Enquanto as tarefas estão sendo executadas, o painel de tarefas é atualizado à medida que os comandos são executados. Você deve examinar o status das tarefas e observar qualquer erro ou aviso exibido. Quando as tarefas são concluídas, o status da mensagem **Tarefa:** Concluído é exibido e a listada para os arquivos de log exibe os nomes dos arquivos de log. Para exibir os arquivos de log que foram criados, escolha um arquivo de log para exibir e clique **em Exibir Log**. Revise todos os arquivos de log para ver se há êxito, erros e mensagens de aviso.
  
Depois de revisar os arquivos de log, clique em **Concluir** para concluir a instalação do armazenamento de configuração local.
