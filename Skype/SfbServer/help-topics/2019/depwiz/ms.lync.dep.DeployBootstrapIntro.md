# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Configurar ou Remover Componentes do Skype for Business Server (Introdução)
 
Você está preparando para instalar e ativar ou desativar e desinstalar Skype para componentes de servidor de negócios. Os processos que serão executados são definidos pela configuração atual da definição publicada no repositório de Gerenciamento Central. Para concluir com êxito as tarefas, você deve estar logado como membro do grupo Administradores local e, no mínimo, você deve ser capaz de ler o Active Directory Domain Services.
  
Clique em **Avançar** para continuar na tela **Executando Comandos**.
