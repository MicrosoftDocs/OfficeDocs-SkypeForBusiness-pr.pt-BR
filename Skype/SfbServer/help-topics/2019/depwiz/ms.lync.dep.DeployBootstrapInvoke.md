# <a name="setup-or-remove-skype-for-business-server-components-invoke"></a>Instalar ou remover Skype para componentes de servidor de negócios (invocar)
 
Os **Comandos executar** exibe um resumo de execução das tarefas que são executadas para implantar a função de servidor. Após a conclusão das tarefas — implantar e ativar ou desativar e desinstalar um resumo é exibido. Além disso, na lista suspensa, você pode selecionar qualquer um dos arquivos de log criados quando os comandos são executados. Use a lista suspensa para selecionar o arquivo de log que você deseja exibir e clique em **Exibir Log**.
  
Depois de terminar de revisar os arquivos de log e de resumo, clique em **Concluir**.
