# <a name="remove-certificate-invoke"></a>Remover Certificado (Invocar)
 
Página **Executando comandos** fornece detalhes para remover o certificado. Revise as informações resumidas contra a remoção de certificado. Certifique-se de que o arquivo de log de remover o certificado está visível na lista suspensa e, em seguida, clique em **Exibir Log** para revisar os detalhes da remoção certificado. Quando você terminar de revisar o resumo e o arquivo de log, clique em **Concluir**.
