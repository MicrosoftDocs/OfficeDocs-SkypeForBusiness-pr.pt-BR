# <a name="certificate-request-certificate-signing-request"></a>Solicitação de certificado (solicitação de assinatura de certificado)
 
Para criar uma solicitação offline, crie primeiro um certificado (CSR) de solicitação de assinatura. Digite o caminho e nome do arquivo CSR na caixa de texto **Nome do arquivo** . Você pode clicar em **Procurar** para escolher um local para criar o CSR. Clique em **Avançar** para continuar.
  
Clique em **Voltar** para voltar à tela anterior. Clique em **Cancelar** para sair sem fazer alterações no sistema.
  
> [!IMPORTANT]
> O arquivo criado nesta etapa é enviado para a autoridade de certificação pública (CA), a sua autoridade de certificação interna offline, ou enviado ao seu administrador da autoridade de certificação para processamento. Observe o local do arquivo e o nome para que você possa recuperá-la e enviar o CSR à autoridade de certificação. O local padrão é o diretório de documentos da pessoa que está executando o Assistente de certificado.
