# <a name="sql-server-reporting-services-intro"></a>SQL Server Reporting Services (Introdução)
 
Cada pool de Front-End e o aparelho de filial persistente podem ter apenas um que Monitoring Server associado a ela. Quando monitoring estiver habilitado para o site, o Monitoring Server fornece detalhes das chamadas (CDR) de gravação e coleta de dados de qualidade da experiência (QoE) e relatórios.
  
Todos os pools em um site e os pools de vários sites centrais podem usar o mesmo Monitoring Server, se o uso não exceder a capacidade do Monitoring Server. Para obter detalhes sobre como projetar uma topologia para dar suporte ao monitoramento, consulte [associar um repositório de monitoramento com um pool de Front-End no Skype para Business Server](../../../deploy/deploy-monitoring/associate-a-monitoring-store.md) na documentação de implantação.
