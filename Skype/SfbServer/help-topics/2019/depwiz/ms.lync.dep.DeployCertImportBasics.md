# <a name="import-certificate-intro"></a>Importar Certificado (Introdução)
 
Para importar um certificado, você deve fornecer um caminho para o arquivo de certificado. Na caixa de texto **Selecionar arquivo do certificado** , você pode digitar o caminho completo e o nome do arquivo ou clicar no botão **procurar** e navegar até o local do caminho e o nome do arquivo (geralmente, um arquivo. p7b,. pfx ou. cer).
  
Se o certificado contiver uma chave privada, selecione o arquivo de certificado caixa de seleção **contém a chave privada do certificado**. Quando essa caixa de seleção estiver marcada, a entrada de texto da **senha** estará habilitada. Se você tiver um certificado com uma chave privada associada a ele, uma senha geralmente será colocada na chave privada quando o certificado for criado. Você insere a senha da chave privada para permitir que o certificado e a chave privada sejam importados para o repositório de certificados. Quando você tiver fornecido as informações para o caminho do arquivo de certificado e, opcionalmente, a senha da chave particular, se necessário, clique em **Avançar**.
  
> [!IMPORTANT]
> Se você não souber a senha da chave privada, a importação falhará.
