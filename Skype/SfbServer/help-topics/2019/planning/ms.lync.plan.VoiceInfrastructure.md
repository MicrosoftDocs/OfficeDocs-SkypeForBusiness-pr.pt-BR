# <a name="voice-infrastructure-planning-tool"></a>Infraestrutura de Voz (Ferramenta de Planejamento)
 
Skype for Business Server fornece conectividade PSTN confiável e escalonável usando troncos SIP, conexões SIP diretas a um gateway PSTN ou conexões SIP diretas para um PBX. Para obter detalhes, [consulte Plan for PSTN connectivity in Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md).
