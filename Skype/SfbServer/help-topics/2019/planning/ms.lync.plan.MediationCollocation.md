# <a name="mediation-collocation-planning-tool"></a>Colocação de mediação (ferramenta de planejamento)
 
Por padrão, o servidor de mediação é colocado com o servidor front-end. O servidor de mediação também pode ser implantado em um pool autônomo por motivos de desempenho, ou se você implantar o tronco SIP, caso em que o pool autônomo seja altamente recomendado. A colocação em funções do Skype for Business Server, exatamente como fazia no Lync Server 2013. Para obter informações detalhadas, consulte os seguintes tópicos:
  
- [Colocação de servidor suportado no Lync Server 2013](https://technet.microsoft.com/library/gg425885%28v=ocs.15%29.aspx)
    
- [Componente do servidor de mediação no Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
