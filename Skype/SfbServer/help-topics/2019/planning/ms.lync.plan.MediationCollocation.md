# <a name="mediation-collocation-planning-tool"></a>Localização de Mediação (Ferramenta de Planejamento)
 
O Servidor de Mediação é alocado com o Servidor Front-End por padrão. O Servidor de Mediação também pode ser implantado em um pool autônomo por motivos de desempenho ou se você implantar tronco SIP, nesse caso, o pool autônomo é altamente recomendado. A localização Skype for Business Server funções exatamente como no Lync Server 2013. Para obter informações detalhadas, consulte os seguintes tópicos:
  
- [Localização de servidor com suporte no Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-supported-server-collocation)
    
- [Componente do Servidor de Mediação Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
