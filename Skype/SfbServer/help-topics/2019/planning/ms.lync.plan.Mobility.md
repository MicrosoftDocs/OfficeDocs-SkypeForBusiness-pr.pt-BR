# <a name="mobility-planning-tool"></a>Mobilidade (ferramenta de planejamento)
 
O recurso de mobilidade no Skype para Business Server suporta Skype para a funcionalidade de negócios em dispositivos móveis.
  
Funcionalidade de mobilidade é a mesma no Skype para Business Server, como no Lync Server 2013. Para obter mais detalhes sobre como preparar seu ambiente para mobilidade, consulte [Planning for mobility no Lync Server 2013](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx).
