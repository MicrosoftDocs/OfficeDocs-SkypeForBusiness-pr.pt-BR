# <a name="mobility-planning-tool"></a>Mobilidade (ferramenta de planejamento)
 
O recurso de mobilidade no Skype for Business Server oferece suporte à funcionalidade do Skype for Business em dispositivos móveis.
  
A funcionalidade de mobilidade é a mesma no Skype for Business Server, como no Lync Server 2013. Para obter mais detalhes sobre como preparar seu ambiente para mobilidade, consulte [Planning for Mobility in Lync Server 2013](https://technet.microsoft.com/library/hh689981%28v=ocs.15%29.aspx).
