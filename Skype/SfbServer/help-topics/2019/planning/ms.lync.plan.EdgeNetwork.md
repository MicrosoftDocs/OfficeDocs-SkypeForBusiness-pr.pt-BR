# <a name="edge-network-planning-tool-in-skype-for-business"></a>Edge Network (Planning Tool) in Skype for Business 
 
A maioria dos componentes de Borda é implantada em uma rede de perímetro. Os componentes a seguir comem a topologia de Borda da rede de perímetro. Exceto quando anotou, os componentes fazem parte dos [cenários](../../../plan-your-deployment/edge-server-deployments/scenarios.md) do Servidor de Borda no Skype for Business Server e estão na rede de perímetro. Os componentes de Borda incluem:
  
- Servidores de borda
    
- Proxy reverso
    
- Firewall
    
- Diretores (opcional e logicamente localizados na rede interna)
    
- Balanceamento de carga para Topologias de Borda Dimensionadas (balanceamento de carga DNS ou um balanceador de carga de hardware)
