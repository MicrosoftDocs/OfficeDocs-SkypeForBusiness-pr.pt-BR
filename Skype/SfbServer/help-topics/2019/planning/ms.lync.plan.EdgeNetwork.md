# <a name="edge-network-planning-tool-in-skype-for-business"></a>Rede de Borda (Ferramenta de Planejamento) em Skype for Business 
 
A maioria dos componentes de Borda é implantada em uma rede de perímetro. Os componentes a seguir comem a topologia de Borda da rede de perímetro. Exceto quando notados, os componentes fazem parte dos [cenários](../../../plan-your-deployment/edge-server-deployments/scenarios.md) do Servidor de Borda Skype for Business Server e estão na rede de perímetro. Os componentes de Borda incluem:
  
- Servidores de borda
    
- Proxy reverso
    
- Firewall
    
- Diretores (opcional e localizado logicamente na rede interna)
    
- Balanceamento de carga para Topologias de Borda Dimensionadas (balanceamento de carga DNS ou um balanceador de carga de hardware)
