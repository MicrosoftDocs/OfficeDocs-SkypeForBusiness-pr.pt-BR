# <a name="voice-traffic-planning-tool"></a>Tráfego de Voz (Ferramenta de Planejamento)
 
A ferramenta de planejamento usa informações sobre o tráfego do usuário para determinar o número de portas a ser usada na implantação. Esse número, por sua vez, determina o número de Servidores de Mediação e gateways que serão necessários. Para obter detalhes, [consulte Estimando o](../../../plan-your-deployment/capacity/estimating-voice-traffic.md)uso de voz e o tráfego para Skype for Business Server .
