# <a name="voice-traffic-planning-tool"></a>Tráfego de voz (ferramenta de planejamento)
 
A ferramenta de planejamento usa as informações sobre o tráfego de usuário para determinar o número de portas para usar na implantação. Por sua vez, esse número determina o número de servidores de mediação e gateways que serão necessários. Para obter detalhes, consulte [Estimating uso de voz e o tráfego para Skype para Business Server](../../../plan-your-deployment/capacity/estimating-voice-traffic.md).
