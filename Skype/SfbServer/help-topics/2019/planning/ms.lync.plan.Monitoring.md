# <a name="skype-for-business-server-monitoring-planning-tool"></a>Skype for Business Server Ferramenta de planejamento de monitoramento

No Skype for Business Server, o monitoramento é usado para coletar informações de uso e dados de QoE (Qualidade da Experiência) sobre as sessões de comunicação em que seus usuários estão envolvidos.

Para obter mais detalhes sobre o planejamento do Monitoring Server, consulte [Planning for Monitoring](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-monitoring).
