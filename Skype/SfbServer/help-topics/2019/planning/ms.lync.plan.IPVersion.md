# <a name="ip-version-planning-tool"></a>Versão IP (Ferramenta de Planejamento)
 
Você tem três opções ao configurar endereços IP no Skype for Business Server. Você pode configurar o Skype for Business Server para dar suporte somente a IP versão 4 (IPv4), somente IP versão 6 (IPv6) ou uma combinação de ambos (conhecido como pilha dupla).
  
Para obter mais detalhes sobre como escolher sua infraestrutura IP, consulte [Plano para IPv6 no Skype for Business.](../../../plan-your-deployment/network-requirements/ipv6.md)
