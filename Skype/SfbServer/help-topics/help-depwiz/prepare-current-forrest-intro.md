# <a name="prepare-current-forrest-intro"></a>Preparar Floresta Atual (Introdução)
 
Para começar a tarefa **Preparar Floresta**, clique em **Avançar**. Executar a **tarefa Preparar Floresta** cria configurações globais e cria grupos universais para uso Skype for Business Server 2015.
