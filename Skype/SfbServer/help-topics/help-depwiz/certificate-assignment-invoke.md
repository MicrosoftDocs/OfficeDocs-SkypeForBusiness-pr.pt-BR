# <a name="certificate-assignment-invoke"></a>Atribuição de Certificado (Invocar)
 
A página **Comandos em Execução** exibe as ações realizadas pelas tarefas para atribuir o certificado ao servidor. Quando as tarefas são concluídas, é possível exibir o log com os detalhes do processo de atribuição do certificado. Para exibir o arquivo de log, clique em **Exibir Log**.
