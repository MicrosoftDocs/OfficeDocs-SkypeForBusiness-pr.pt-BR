# <a name="certificate-assignment-invoke"></a>Atribuição de Certificado (Invocar)
 
Página **Executando comandos** exibe as ações realizadas pelas tarefas para atribuir o certificado ao servidor. Quando as tarefas forem concluídas, você pode exibir o arquivo de log que detalha o processo de atribuição de certificado. Para exibir o arquivo de log, clique em **Exibir Log**.
