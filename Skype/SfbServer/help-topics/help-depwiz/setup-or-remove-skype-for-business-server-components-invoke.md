# <a name="setup-or-remove-skype-for-business-server-components-invoke"></a>Configurar ou Remover Componentes do Skype for Business Server (Invocar)
 
Os **comandos executar** exibem um resumo em execução das tarefas que são executadas para implantar a função do servidor. Após a conclusão das tarefas, implante e ative ou desative e desinstale-um resumo será exibido. Além disso, na lista suspensa, você pode selecionar qualquer um dos arquivos de log que são criados quando os comandos são executados. Use a lista suspensa para selecionar o arquivo de log que você deseja exibir e clique em **Exibir log**.
  
Quando terminar de revisar os arquivos de resumo e log, clique em **concluir**.
