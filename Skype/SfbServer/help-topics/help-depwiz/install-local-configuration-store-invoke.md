# <a name="install-local-configuration-store-invoke"></a>Instalar Repositório de Configuração Local (Invocar)
 
O painel **executar comandos** exibe o status das tarefas para instalar a réplica somente leitura local do repositório de gerenciamento central. Enquanto as tarefas estiverem em execução, o painel de tarefas será atualizado à medida que os comandos forem executados. Você deve revisar o status da tarefa e observar os erros ou avisos exibidos. Quando as tarefas estiverem concluídas, o **status da tarefa da mensagem: concluído** será exibido e a lista suspensa para os arquivos de log exibirá os nomes dos arquivos de log. Para exibir os arquivos de log que foram criados, escolha um arquivo de log para exibir e clique em **Exibir log**. Revise todos os arquivos de log para obter sucesso, erros e mensagens de aviso.
  
Depois de revisar os arquivos de log, clique em **concluir** para concluir a instalação do repositório de configuração local.
