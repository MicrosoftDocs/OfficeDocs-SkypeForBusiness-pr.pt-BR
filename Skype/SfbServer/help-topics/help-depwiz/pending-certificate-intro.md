# <a name="pending-certificate-intro"></a>Certificado Pendente (Introdução)
 
Para processar uma solicitação de certificado online pendente, isto é, uma solicitação que tenha sido emitida para sua autoridade de certificação online, mas esteja definida de forma administrativa para exigir um processo de aprovação manual, clique em **Avançar**.
