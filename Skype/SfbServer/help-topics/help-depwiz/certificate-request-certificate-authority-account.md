# <a name="certificate-request-certificate-authority-account"></a>Solicitação de Certificado (Conta da Autoridade de Certificação)
 
Para enviar uma solicitação, sua autoridade de certificação (CA) pode exigir credenciais diferentes das do usuário em que você está conectado. Para permitir uma solicitação de certificado como um usuário diferente, marque a caixa de seleção **especificar credenciais alternativas para a autoridade de certificação**e digite o nome de usuário ou o nome de usuário do _domínio_\ _para um_ usuário que pode solicitar o certificado. Na caixa de texto **senha** , digite a senha do usuário que você especificou. O nome de usuário e a senha são enviados como parte do processo de solicitação de certificado, mas não na solicitação de certificado real, a uma CA online.
