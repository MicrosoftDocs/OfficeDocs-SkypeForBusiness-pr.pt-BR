# <a name="certificate-request-organization-information"></a>Solicitação de certificado (informações da organização)
 
O painel de **Informações da organização** permite que você especifique o nome da sua organização e uma unidade organizacional (UO) (normalmente a divisão ou departamento) que solicita o certificado. As informações que você digita na **organização** e as caixas de texto de **Unidade organizacional** aparece no certificado e são usadas para identificar quem solicitou o certificado, quando o certificado é exibido por outras pessoas.
  
> [!NOTE]
> O nome da unidade Organizacional em um certificado não é necessariamente o mesmo que uma unidade Organizacional no Active Directory Domain Services. Dependendo de como você definiu do Active Directory, o nome que você usa pode ou não ser os mesmos.
