# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Configurar ou Remover Componentes do Skype for Business Server (Introdução)
 
Você está se preparando para instalar e ativar ou desativar e desinstalar componentes do Skype for Business Server 2015. Os processos que serão executados são definidos pela configuração atual da definição publicada no repositório de Gerenciamento Central. Para concluir as tarefas com êxito, você deve estar conectado como membro do grupo Administradores local e, no mínimo, você deve ser capaz de ler os serviços de domínio do Active Directory.
  
Clique em **Avançar** para continuar na tela **Executando Comandos**.
