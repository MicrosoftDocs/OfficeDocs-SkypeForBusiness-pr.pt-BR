# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Configurar ou Remover Componentes do Skype for Business Server (Introdução)
 
Você está se preparando para instalar e ativar ou desativar e desinstalar Skype for Business Server componentes 2015. Os processos que serão executados são definidos pela configuração atual da definição publicada no repositório de Gerenciamento Central. Para concluir com êxito as tarefas, você deve estar conectado como membro do grupo local Administradores e, no mínimo, você deve ser capaz de ler os Serviços de Domínio do Active Directory.
  
Clique em **Próximo** para continuar à tela **Executando Comandos** tela.
