# <a name="install-or-update-skype-for-business-server-system"></a>Instalar ou Atualizar o Sistema Skype for Business Server

Para iniciar a instalação ou atualizar uma implantação existente do Skype for Business Server 2015, clique em Instalar ou **Atualizar Skype for Business Server**. As etapas nesta seção instalarão componentes específicos de uma determinada função de servidor, com base no documento construtor de topologias. Antes de começar essa série de etapas, você deve definir a implantação no Construtor de Topologias. Você deve estar conectado ao computador que está implantando como administrador local e como membro do grupo Administradores de Domínio no domínio onde o servidor está localizado. O computador deve ser ingressado no domínio antes de concluir esta etapa.

Se você estiver atualizando um servidor implantado no momento, deverá fazer as alterações que pretende fazer no Construtor de Topologias.
