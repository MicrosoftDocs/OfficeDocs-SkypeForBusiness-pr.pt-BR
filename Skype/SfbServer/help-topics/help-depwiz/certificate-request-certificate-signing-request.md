# <a name="certificate-request-certificate-signing-request"></a>Solicitação de Certificado (Solicitação de Assinatura de Certificado)
 
Para criar uma solicitação offline, primeiro você deve criar uma solicitação de assinatura de certificado (CSR). Digite o caminho e o nome de arquivo do CSR na caixa de texto **nome do arquivo** . Você pode clicar em **procurar** para escolher um local para criar o CSR. Clique em **Avançar** para continuar.
  
Clique em **voltar** para voltar para a tela anterior. Clique em **Cancelar** para sair sem fazer alterações no sistema.
  
> [!IMPORTANT]
> O arquivo criado nesta etapa é enviado à sua CA (autoridade de certificação) pública para a sua CA interna offline ou enviado ao administrador da autoridade de certificação para processamento. Observe o local e o nome do arquivo para que você possa recuperá-lo e enviar o representante do SAC para a CA. O local padrão é o diretório de documentos da pessoa que está executando o assistente de certificado.
