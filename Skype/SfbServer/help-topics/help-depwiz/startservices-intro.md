# <a name="startservices-intro"></a>Iniciar Serviços (Introdução)
 
Para iniciar serviços no servidor de funções do Skype for Business Server 2015 que você está implantando, examine a tela de introdução. Observe que os comandos emitidos para iniciar o Skype for Business Server não esperam que o serviço seja iniciado. Essa é uma iniciativa para iniciar os serviços, e você deve monitorar o status real dos serviços usando o Console de Gerenciamento Microsoft (MMC) dos Serviços do Windows. Clique em **Avançar** para começar a tarefa **Iniciar Serviços**.
