# <a name="startservices-intro"></a>Iniciar Serviços (Introdução)
 
Para iniciar serviços no servidor de função Skype for Business Server 2015 que você está implantando, revise a tela de introdução. Observe que os comandos emitidos para iniciar o Skype for Business Server não aguardam o início do serviço. Este é um melhor esforço para iniciar os serviços e você deve monitorar o status real dos serviços usando o Console de Gerenciamento da Microsoft (MMC) de serviços Windows serviços. Clique em **Avançar** para iniciar a tarefa **Iniciar Serviços**.
