# <a name="startservices-intro"></a>Iniciar Serviços (Introdução)
 
Para iniciar os serviços no Skype para servidor de função Business Server 2015 que está sendo implantado, examine a tela de Introdução. Observe que os comandos emitidos para iniciar o Skype para Business Server não esperar para iniciar o serviço. Essa é uma iniciativa para iniciar os serviços, e você deve monitorar o status real dos serviços usando o Console de Gerenciamento Microsoft (MMC) dos Serviços do Windows. Clique em **Avançar** para começar a tarefa **Iniciar Serviços**.
