# <a name="skype-for-business-server-2015"></a>Skype for Business Server 2015

Skype para negócios é uma plataforma de comunicação e colaboração que reúne uma experiência de cliente tendo como Skype com controle, conformidade e segurança de nível empresarial. Skype para negócios oferece recursos incluindo presença, mensagens Instantâneas, voz e chamadas de vídeo e reuniões online.
