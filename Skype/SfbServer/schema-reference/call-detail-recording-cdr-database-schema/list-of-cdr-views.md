# <a name="list-of-cdr-views"></a>Lista de Exibições de CDR
 
Os modos de exibição fornecem uma maneira fácil de acessar informações sobre os cenários mais comuns usados para retornar dados do banco de dados CDR. É recomendável que você use modos de exibição para criar relatórios personalizados em vez de usar as tabelas de banco de dados CDR reais; Isso porque os modos de exibição de banco de dados são mais prováveis de manter a compatibilidade com versões anteriores.
  
|**Nome do modo de exibição**|**Descrição**|
|:-----|:-----|
|[Exibição ClientVersions](clientversions-0.md) <br/> |Retorna informações sobre o software/dispositivos cliente usados em uma sessão de comunicação.  <br/> |
|[Exibição ConferenceMessageCount](conferencemessagecount-0.md) <br/> |Retorna informações sobre o número de mensagens enviadas pelos usuários em uma conferência.  <br/> |
|[Modo de exibição conferências](conferences-0.md) <br/> |Retorna informações de conferência, incluindo a hora de início, a hora de término e o organizador de conferências.  <br/> |
|[Exibição ConferenceSessionDetails](conferencesessiondetails.md) <br/> |Retorna detalhes de sessão para todas as sessões de conferência, incluindo hora de início e término, IDs de usuário, códigos de resposta e IDs de diagnóstico.  <br/> |
|[Exibição ConferenceUris](conferenceuris-0.md) <br/> |Retorna informações sobre URIs de conferência usados em uma conferência  <br/> |
|[Exibição ErrorReport](errorreport-0.md) <br/> |Retorna informações sobre erros ocorridos durante uma sessão.  <br/> |
|[Modo de exibição de transferência de fileviews](filetransfers.md) <br/> |Retorna informações sobre as sessões de transferência de arquivos, incluindo o nome do arquivo e se a transferência foi aceita, rejeitada ou cancelada.  <br/> |
|[Exibição FocusJoinsAndLeaves](focusjoinsandleaves-0.md) <br/> |Retorna informações sobre atividades de ingressar em conferência e licença.  <br/> |
|[Exibição McuJoinsAndLeaves](mcujoinsandleaves-0.md) <br/> |Retorna informações combinadas sobre atividades de ingressar em conferência e licença (cada junção de conferência está emparelhada com a saída de conferência correspondente).  <br/> |
|[Exibição MCUs](mcus-0.md) <br/> |Retorna informações sobre servidores de conferência.  <br/> |
|[Modo de exibição de mídia](media-0.md) <br/> |Retorna informações sobre os tipos de mídia usados em sessões de comunicação ponto a ponto.  <br/> |
|[Exibição ProgressReport](progressreport-0.md) <br/> |Retorna informações sobre sessões concluídas.  <br/> |
|[Modo de exibição registro](registration-0.md) <br/> |Retorna informações sobre registros com o Skype for Business Server 2015.  <br/> |
|[Exibição SessionDetails](sessiondetails-0.md) <br/> |Retorna informações sobre sessões ponto a ponto, incluindo chamadas telefônicas VoIP-VoIP, sessões de mensagens instantâneas de duas partes ou outras sessões de comunicação ponto a ponto.  <br/> |
|[Exibição do usuário](user.md) <br/> |Retorna informações sobre os usuários que participaram de sessões de comunicação.  <br/> |
|[Exibição VoIPDetails](voipdetails.md) <br/> |Retorna informações sobre sessões ponto a ponto que envolvem pelo menos um usuário VoIP (voz sobre e/s).  <br/> |
