# <a name="servers-table"></a>Tabela de servidores
 
A tabela de servidores é uma tabela de suporte que armazena informações sobre os vários servidores. Cada registro na tabela representa um servidor.
  
|**Coluna**|**Tipo de dados**|**Chave/índice**|**Detalhes**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Primária  <br/> |Número exclusivo que identifica este servidor.  <br/> |
|**ServerFQDN** <br/> |nvarchar(256)  <br/> | <br/> |FQDN do servidor.  <br/> |
