# <a name="servers-table"></a>Tabela Servidores
 
A tabela Servidores é uma tabela de suporte que armazena informações sobre os vários servidores. Cada registro na tabela representa um servidor.
  
|**Coluna**|**Tipo de dados**|**Chave/Índice**|**Detalhes**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Primário  <br/> |Número exclusivo que identifica esse servidor.  <br/> |
|**ServerFQDN** <br/> |nvarchar(256)  <br/> | <br/> |FQDN do servidor.  <br/> |
