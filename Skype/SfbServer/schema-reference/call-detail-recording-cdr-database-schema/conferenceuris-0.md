# <a name="conferenceuris-view"></a>Exibir ConferenceUris
 
O modo de exibição ConfernceUris armazena informações sobre os URIs que tenham participado em sessões de conferência. Este modo de exibição foi introduzido no Microsoft Lync Server 2013.
  
|**Coluna**|**Tipo de dados**|**Detalhes**|
|:-----|:-----|:-----|
|ConferenceUriId  <br/> |int  <br/> |Número exclusivo que identifica o URI de conferência.  <br/> |
|ConferenceUri  <br/> |nvarchar(450)  <br/> |URI da conferência.  <br/> |
|ConferenceUriType  <br/> |nvarchar(256)  <br/> |Tipo de URI de conferência. Consulte a [tabela UriTypes](uritypes.md) para obter mais informações. <br/> |
