# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>Tabela HardwareVersions no Skype para Business Server 2015
 
A tabela HardwareVersions é uma tabela de suporte. Cada registro armazena informações sobre a versão de hardware de um dispositivo.
  
|**Coluna**|**Tipo de dados**|**Chave/índice**|**Detalhes**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |int  <br/> |Primária  <br/> |Número exclusivo identificando esta versão de hardware.  <br/> |
|**Versão** <br/> |nvarchar(256)  <br/> | <br/> |Versão de hardware.  <br/> |
