# <a name="mcus-view"></a>Exibir MCUs
 
O modo de exibição de Mcus armazena informações sobre os MCUs que tenham participado em sessões de conferência. Este modo de exibição foi introduzido no Microsoft Lync Server 2013.
  
|**Coluna**|**Tipo de dados**|**Detalhes**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |Número exclusivo que identifica o MCU.  <br/> |
|**McuUri** <br/> |nvarchar(450)  <br/> |URI do MCU.  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |Tipo de URI do MCU. Consulte a [tabela UriTypes](uritypes.md) para obter mais informações. <br/> |
