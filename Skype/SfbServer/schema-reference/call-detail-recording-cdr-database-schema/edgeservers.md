# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>Tabela EdgeServers no Skype for Business Server 2015
 
A tabela EdgeServers é uma tabela de suporte. Cada registro armazena informações sobre um Servidor de Borda envolvido em chamadas que possuem registros no banco de dados.
  
|**Coluna**|**Tipo de dados**|**Chave/Índice**|**Detalhes**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |int  <br/> |Primário  <br/> |Número exclusivo que identifica este Servidor de Borda.  <br/> |
|**EdgeServer** <br/> |nvarchar(256)  <br/> | <br/> |Nome do Servidor de Borda.  <br/> |
