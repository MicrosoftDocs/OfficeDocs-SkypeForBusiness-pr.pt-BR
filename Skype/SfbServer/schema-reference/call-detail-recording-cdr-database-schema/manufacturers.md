# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Tabela de fabricantes no Skype para Business Server 2015
 
A tabela de fabricantes é uma tabela de suporte. Cada registro armazena informações sobre o fabricante de um dispositivo (telefone de mesa).
  
|**Coluna**|**Tipo de dados**|**Índice de chaves /**|**Detalhes**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Primária  <br/> |Número exclusivo que identifica esse fabricante.  <br/> |
|**Fabricante** <br/> |nvarchar(256)  <br/> | <br/> |Nome do fabricante.  <br/> |
