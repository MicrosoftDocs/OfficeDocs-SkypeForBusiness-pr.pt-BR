# <a name="networkconfigurationsettings-view"></a>Exibir NetworkConfigurationSettings
 
O modo de exibição NetworkConfigurationSettings armazena informações sobre a configuração de rede. Este modo de exibição foi introduzido no Microsoft Lync Server 2013.
  
|**Coluna**|**Tipo de dados**|**Detalhes**|
|:-----|:-----|:-----|
|SubnetIP  <br/> |int  <br/> |Endereço IP da subrede (apenas endereços IPv4 são suportados).  <br/> |
|UserSiteName  <br/> |nvarchar (128)  <br/> |Nome do site do usuário.  <br/> |
|RegionName  <br/> |nvarchar (128)  <br/> |Nome do país/região do usuário.  <br/> |
