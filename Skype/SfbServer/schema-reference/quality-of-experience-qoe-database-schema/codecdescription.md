# <a name="codecdescription-table"></a>Tabela CodecDescription
 
A tabela CodecDescription mapeia identificadores exclusivos de codec ao codec correspondente. Codecs são usados para codificar sinais digitais para transmissão e difusão e depois para decodificar esses sinais para reprodução. Esta tabela foi introduzida no Microsoft Lync Server 2013
  
|**Coluna**|**Tipo de dados**|**Chave/índice**|**Detalhes**|
|:-----|:-----|:-----|:-----|
|**Chave** <br/> |smallint  <br/> |Primária  <br/> |Identificador exclusivo atribuído ao codec.  <br/> |
|**CodecDescription** <br/> |varchar(256)  <br/> |Exclusivo  <br/> |Descrição exclusiva do codec correspondendo à chave.  <br/> |
