# <a name="macaddress-table"></a>Tabela MacAddress
 
A tabela MacAddress é uma tabela de suporte. Cada registro representa uma fonte.
  
|**Coluna**|**Tipo de dados**|**Índice de chaves /**|**Detalhes**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Primária  <br/> |Número exclusivo que identifica o endereço Mac.  <br/> |
|**MacAddress** <br/> |varchar(256)  <br/> |Exclusivo  <br/> |Cadeia de caracteres de endereço MAC.  <br/> |
