# <a name="devicedriver-table"></a>Tabela DeviceDriver
 
A tabela DeviceDriver é uma tabela de suporte. Cada registro representa um driver usado por um dispositivo de captura ou um dispositivo de renderização.
  
|**Coluna**|**Tipo de dados**|**Chave/índice**|**Detalhes**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |Primária  <br/> |Número exclusivo que identifica este registro de driver de dispositivo.  <br/> |
|**DeviceDriver** <br/> |varchar (256)  <br/> |exclusividade  <br/> |Nome do driver do dispositivo.  <br/> |
