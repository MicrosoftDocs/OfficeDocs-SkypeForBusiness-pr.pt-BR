|:-----|:-----|:-----|:-----|
|**SubnetIP** <br/> |int  <br/> |Primário, estrangeiro  <br/> |Representação de inteiro para a sub-rede.  <br/> |
|**NextUpdateTS** <br/> |datetime  <br/> ||Somente para uso interno.  <br/> |
