# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision contém o número de revisão usada com a tabela tblAdminLock para obter consistência entre vários clientes administradores.
  
**Colunas**

|**Coluna**|**Tipo**|**Descrição**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint, não nulo  <br/> |Número de revisão.  <br/> |
