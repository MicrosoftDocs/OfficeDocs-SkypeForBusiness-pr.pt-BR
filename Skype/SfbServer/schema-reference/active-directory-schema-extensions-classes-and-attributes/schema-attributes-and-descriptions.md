# <a name="schema-attributes-and-descriptions-in-skype-for-business-server"></a>Atributos e descrições de esquema no Skype for Business Server
 
Esta seção descreve todos os atributos de esquema usados pelo Skype for Business Server. Para as classes associadas aos atributos, consulte Atributos de esquema [por classe no Skype for Business Server.](schema-attributes-by-class.md)
  
Atributos que são pares vinculados são especificados como links de encaminhamento ou de retorno. Um atributo que se refere a outro objeto é um link de encaminhamento; o atributo do outro objeto que se refere ao primeiro objeto é um link de retorno. Os links de encaminhamento são atualizáveis, enquanto os links de retorno são somente leitura.
  
Alguns atributos têm um valor de máscara de bits. Para esses atributos, cada configuração é representada por um bit e o valor decimal exibido representa a posição do bit. As posições de bits começam com bit 0. Por exemplo, 1 (binário) é bit 0 definido e 10000 (binário) é bit 4 definido. Cada bit representa uma propriedade. Estes são alguns exemplos: 
  
- 10000 (binário) tem um valor decimal de 16 (ou seja, bit 4 está definido).
    
- 100000000 (binário) tem um valor decimal de 256 (ou seja, bit 8 está definido).
    
- 1100 (binário) tem um valor decimal de 12 (ou seja, os bits 2 e 3 são definidos; as propriedades representadas por ambos os bits são habilitadas).
    
- 1111000001 (binário) tem um valor decimal de 961 (ou seja, os bits 0, 6, 7, 8 e 9 estão definidos; as propriedades de cada um desses bits estão habilitadas).
