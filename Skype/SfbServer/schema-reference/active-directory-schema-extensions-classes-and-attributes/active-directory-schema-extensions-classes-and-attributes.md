# <a name="active-directory-schema-extensions-classes-and-attributes"></a>Extensões, classes e atributos do esquema do Active Directory
 
Esta seção de referência inclui as seguintes informações: 
  
- Extensões de esquema Active Directory que são novas ou alteradas para Skype para Business Server
    
    Esquema do Active Directory contém definições formais de cada classe de objeto que podem ser criados em uma floresta do Active Directory. O esquema também contém definições formais de todos os atributos que podem existir em um objeto do Active Directory. O catálogo global do Active Directory contém réplicas de todos os objetos da floresta, além de um subconjunto dos atributos de cada objeto. Esta seção descreve as classes e atributos que são novas ou alteradas no Skype para Business Server.
    
- Todas as classes usadas pelo Skype para Business Server, com uma descrição de cada
    
- Todas os atributos usados pelo Skype para Business Server, com uma descrição de cada
    
- Uma lista das classes usadas pelo Skype para Business Server, com os atributos que cada uma pode conter
    
- Configurações globais e objetos, além de grupos universais de serviço e administração criados durante a preparação da floresta
    
- Entradas de controle de acesso (ACEs) que são criadas nos domínio raiz e nos contêineres internos durante a preparação do domínio
    
- Alterações feitas em uma unidade organizacional (UO) do Active Directory pelo cmdlet Grant_CsSetupPermission.
    
- Alterações feitas em uma unidade Organizacional do Active Directory pelo cmdlet Grant_CsOUPermission.
