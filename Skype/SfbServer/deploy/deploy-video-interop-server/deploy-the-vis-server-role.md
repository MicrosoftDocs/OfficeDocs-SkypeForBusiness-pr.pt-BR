# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>Implantar a função de servidor VIS Skype for Business Server
 
**Resumo:** Implante a função VIS (Servidor de Interop de Vídeo) Skype for Business Server.
  
Para configurar o serviço VIS no servidor criado no Construtor de Topologias, inicie o assistente de implantação do Skype for Business Server, pressione Instalar ou Atualizar Skype for Business Server **Sistema** e siga estas etapas no assistente:
  
1.  Selecione **Instalar o Armazenamento de Configuração Local**.
    
2. Selecione **Configurar ou Remover Skype for Business Server Componentes**.
    
3. Selecione **Solicitar, Instalar ou Atribuir Certificados.**
    
4. Selecione **Iniciar serviços**.
    
O software para esse serviço agora está instalado e em execução. Você pode abrir a ferramenta mmc Serviços para ver se o serviço Skype for Business Server Servidor de **Interop** de Vídeo está sendo executado juntamente com outros serviços Skype for Business Server de vídeo. Em seguida, você deve configurar o servidor ou pool do VIS.
