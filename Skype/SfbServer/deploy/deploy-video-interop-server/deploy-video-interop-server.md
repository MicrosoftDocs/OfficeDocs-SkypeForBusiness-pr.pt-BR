# <a name="deploy-video-interop-server-in-skype-for-business-server"></a>Implantar o Servidor de Interop de Vídeo no Skype for Business Server
 
**Resumo:** Implantar a função de servidor VIS no Skype for Business Server.
  
O Skype for Business Server agora pode se integrar diretamente aos sistemas de teleconferência (VTCs) da Cisco, como o Cisco C60 ou o Cisco MX300. Isso exige a introdução de uma nova função de servidor chamada Servidor de Interop de Vídeo (VIS) e a configuração correta do VIS e do equipamento com o que ele interoperará. Um VTC se registra na infraestrutura da Cisco existente, como o Cisco Unified Communication Manager (CUCM), e um tronco SIP de vídeo é usado entre o CUCM e o pool do VIS.

Configurar a interoperabilidade entre um servidor ou pool vis e sistemas VTC requer a execução dos cinco procedimentos a seguir:

- [Implantar a função de servidor VIS no Skype for Business Server](deploy-the-vis-server-role.md)
    
- [Configurar o Servidor de Interop de Vídeo no Skype for Business Server](configure-the-vis.md)
    
- [Configurar o CUCM para interoperação com o Skype for Business Server](configure-cucm-for-interoperation.md)
