# <a name="deploy-video-interop-server-in-skype-for-business-server"></a>Implantar o servidor de interoperabilidade de vídeo no Skype for Business Server
 
**Resumo:** Implante a função de servidor VIS no Skype for Business Server.
  
O Skype for Business Server agora pode se integrar diretamente a Cisco teleconferência Systems (VTCs), como o Cisco C60 ou o Cisco MX300. Isso requer a introdução de uma nova função de servidor chamada servidor de interoperabilidade de vídeo (VIS) e a configuração correta do VIS e do equipamento com o qual ele interoperará. Um VTC registra com o Cisco Unified Communication Manager (CUCM) existente, e um tronco SIP de vídeo é usado entre o CUCM e o pool de VIS.
  
## <a name="in-this-section"></a>Nesta seção

A configuração da interoperabilidade entre um servidor ou pool VIS e sistemas VTC requer a realização dos cinco procedimentos a seguir: 
  
- [Criar um pool de VIS no Skype for Business Server](create-a-vis-pool.md)
    
- [Implantar a função de servidor VIS no Skype for Business Server](deploy-the-vis-server-role.md)
    
- [Configurar o servidor de interoperabilidade de vídeo no Skype for Business Server](configure-the-vis.md)
    
- [Configurar o CUCM para interoperação com o Skype for Business Server](configure-cucm-for-interoperation.md)
    
- [Configurar um VTC para interoperação com o Skype for Business Server](configure-a-vtc-for-interoperation.md)
