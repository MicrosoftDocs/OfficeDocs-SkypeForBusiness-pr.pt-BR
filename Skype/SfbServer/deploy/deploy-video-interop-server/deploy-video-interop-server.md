# <a name="deploy-video-interop-server-in-skype-for-business-server-2015"></a>Implantar Servidor de Interoperabilidade de Vídeo no Skype for Business Server 2015
 
**Resumo:** Implante a função de servidor VIS no Skype para Business Server 2015.
  
Skype para Business Server agora pode integrar diretamente com sistemas de teleconferência Cisco (VTCs) como o Cisco C60 ou Cisco MX300. Isso requer a introdução de uma nova função de servidor chamada o servidor de interoperabilidade de vídeo (VIS) e a configuração correta do VIS e o equipamento com que ele irá interoperar. Um VTC registra com o Cisco Unified Communication Manager (CUCM) existente, e um tronco SIP de vídeo é usado entre o CUCM e o pool de VIS.
  
## <a name="in-this-section"></a>Nesta seção

A configuração da interoperabilidade entre um servidor ou pool VIS e sistemas VTC requer a realização dos cinco procedimentos a seguir: 
  
- [Criar um pool VIS no Skype para Business Server 2015](create-a-vis-pool.md)
    
- [Implantar a função de servidor VIS no Skype para Business Server 2015](deploy-the-vis-server-role.md)
    
- [Configurar o servidor de interoperabilidade de vídeo no Skype para Business Server 2015](configure-the-vis.md)
    
- [Configurar CUCM para interoperação com Skype para Business Server 2015](configure-cucm-for-interoperation.md)
    
- [Configurar um VTC para interoperação com Skype para Business Server 2015](configure-a-vtc-for-interoperation.md)
