# <a name="deploy-call-management-features-in-skype-for-business-2015"></a>Implantar recursos de gerenciamento de chamada no Skype for Business 2015
 
Implantando os recursos de gerenciamento de chamada do Skype para Business Server Enterprise Voice.
  
Como as chamadas de entrada de controle de recursos de gerenciamento de chamada de Enterprise Voice são roteadas e atendidas. Skype para Business Server fornece recursos de gerenciamento da chamada a seguir: 
  
- **Estacionamento de chamada**: permite que os usuários de voz estacionem temporariamente uma chamada e a continuem do mesmo telefone ou outro telefone.
    
- **Atendimento de chamada de grupo**: permite que os usuários atender as chamadas feitas para outro usuário que é atribuído a um grupo de retirada discando o número de retirada de grupo de chamada.
    
- **Grupo de resposta**: roteia chamadas recebidas para grupos de agentes usando grupos de busca ou perguntas de IVR (resposta) de voz interativa e respostas.
    
- **Anúncio**: Reproduz uma mensagem para chamadas realizadas a um número não atribuído ou direciona a chamada para qualquer outro local ou ambos.
    
Esta seção descreve como configurar esses recursos de gerenciamento de chamada durante uma implantação do Enterprise Voice.
