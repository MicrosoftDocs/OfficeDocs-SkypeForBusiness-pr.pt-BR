# <a name="deploy-call-management-features-in-skype-for-business"></a>Implantar recursos de gerenciamento de chamadas no Skype for Business
 
Implantação de recursos de gerenciamento de chamadas no Skype for Business Server Enterprise Voice.
  
Os recursos de gerenciamento de chamadas do Enterprise Voice controlam como as chamadas de entrada são roteadas e respondidas. O Skype for Business Server fornece os seguintes recursos de gerenciamento de chamadas: 
  
- **Estacionamento de chamadas**: permite que os usuários de voz disquemm temporariamente uma chamada e o escolham no mesmo telefone ou em outro telefone.
    
- **Recebimento de chamadas em grupo**: permite que os usuários atendam chamadas feitas a outro usuário atribuído a um grupo de coleta discando o número do grupo de recebimento de chamadas.
    
- **Grupo de resposta**: roteia as chamadas recebidas para grupos de agentes usando grupos de busca ou perguntas e respostas de IVR (resposta interativa de voz).
    
- **Anúncio**: Reproduz uma mensagem para chamadas realizadas a um número não atribuído ou direciona a chamada para qualquer outro local ou ambos.
    
Esta seção descreve como configurar esses recursos de gerenciamento de chamadas durante uma implantação do Enterprise Voice.
