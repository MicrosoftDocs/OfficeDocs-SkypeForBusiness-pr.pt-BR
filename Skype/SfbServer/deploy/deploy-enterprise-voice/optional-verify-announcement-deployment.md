# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a>(Opcional) Verificar a implantação do comunicado na Skype para negócios
 
Verificando a implantação do aplicativo de anúncio no Skype para Business Server Enterprise Voice. Isso afeta como o sistema lida com as chamadas para números não atribuídos.
  
Após instalar e configurar o Comunicado, você precisará verificar a configuração para garantir que as chamadas a números não atribuídos funcionem conforme o esperado. No mínimo, verifique o seguinte:
  
- Ligue para um número válido para a organização, mas que seja um número não atribuído.
    
- Ligue para o número não atribuído e verifique se o comunicado correto foi reproduzido.
