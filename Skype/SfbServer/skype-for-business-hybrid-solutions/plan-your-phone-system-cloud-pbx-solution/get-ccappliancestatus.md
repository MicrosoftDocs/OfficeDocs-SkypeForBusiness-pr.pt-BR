# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
O cmdlet Get-CcApplianceStatus fornece informações de diagnóstico para o Skype for Business Cloud Connector Edition dispositivo.
  
Este cmdlet se aplica Skype for Business Cloud Connector Edition 2.1.
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>Descrição detalhada

O Get-CcApplianceStatus cmdlet, quando executado na máquina host do Cloud Connector, fornece informações de diagnóstico para o Skype for Business do Cloud Connector.
  
## <a name="parameters"></a>Parâmetros

Nenhum
  
## <a name="input-types"></a>Tipos de entrada

Nenhum. O Get-CcApplianceStatus cmdlet não aceita entrada canalada.
  
## <a name="return-types"></a>Tipos de Retorno

O comando retorna o status do dispositivo.
