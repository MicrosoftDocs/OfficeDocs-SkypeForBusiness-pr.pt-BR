# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
O cmdlet Get-CcApplianceStatus fornece informações de diagnóstico para o Skype para aparelho de conector de nuvem Business Edition.
  
Este cmdlet se aplica a Skype para negócios nuvem conector Edition 2.1.
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>Descrição detalhada

O cmdlet Get-CcApplianceStatus, quando executado na máquina host do conector de nuvem, fornece informações de diagnóstico para o Skype para aparelho de conector de nuvem de negócios.
  
## <a name="parameters"></a>Parâmetros

Nenhum
  
## <a name="input-types"></a>Tipos de entrada

Nenhuma. O cmdlet Get-CcApplianceStatus não aceita a entrada.
  
## <a name="return-types"></a>Tipos de retorno

O comando retorna o status do aparelho.
