# <a name="restore-cccredentials"></a>Restore-CcCredentials
 
O cmdlet Restore Cc-credenciais restaura todas as credenciais do Skype atual para a implantação do conector de nuvem Business Edition. 
  
Este cmdlet se aplica a Skype para negócios nuvem conector Edition 2.1.
  
```
Restore-CcCredentials 
```

## <a name="detailed-description"></a>Descrição detalhada

O cmdlet Restore-CcCredentials limpa todas as credenciais e solicita digitem novamente todas as credenciais usadas para o atual Skype para implantação do conector de nuvem de negócios.
  
## <a name="parameters"></a>Parâmetros

Nenhum
  
## <a name="input-types"></a>Tipos de entrada

Nenhum. O cmdlet Restore-CcCredentials não aceita a entrada.
  
## <a name="return-types"></a>Tipos de retorno

Nenhuma.
  
## <a name="example"></a>Exemplo

O exemplo a seguir restaura todas as credenciais da implantação do conector de nuvem atual:
  
```
    PS C:\>Restore-CcCredentials
```
