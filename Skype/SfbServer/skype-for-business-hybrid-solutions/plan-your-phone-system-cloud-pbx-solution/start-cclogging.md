# <a name="start-cclogging"></a>Start-CcLogging
 
O Start-CcLogging cmdlet gera o log de chamadas de entrada e saída para um Skype for Business Cloud Connector Edition de entrada. 
  
```powershell
Start-CcLogging
```

## <a name="parameters"></a>Parâmetros

Nenhum
  
## <a name="examples"></a>Exemplos
<a name="Examples"> </a>

### <a name="example-1"></a>Exemplo 1

O exemplo a seguir gera o log de chamadas de entrada e saída:
  
```powershell
Start-CcLogging
```

## <a name="detailed-description"></a>Descrição detalhada
<a name="DetailedDescription"> </a>

O Start-CcLogging cmdlet fornece uma maneira para os administradores iniciarem o registro em log de chamadas de entrada e saída em um dispositivo do Cloud Connector. Por padrão, o registro em log será automaticamente parado após quatro horas.
  
## <a name="input-types"></a>Tipos de Entrada
<a name="InputTypes"> </a>

Nenhum. O Start-CcLogging cmdlet não aceita entrada canalada.
  
## <a name="return-types"></a>Tipos de Retorno
<a name="ReturnTypes"> </a>

Nenhum
