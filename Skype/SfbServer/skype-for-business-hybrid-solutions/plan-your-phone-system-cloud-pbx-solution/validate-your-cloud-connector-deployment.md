# <a name="validate-your-cloud-connector-deployment"></a>Validar a implantação do Cloud Connector
 
Depois de concluir as etapas para implantar e configurar o Skype para o conector de nuvem Business Edition, verifique se que você pode fazer e receber chamadas.
  
## <a name="validate-the-deployment"></a>Validar a implantação

1. Faça logon no cliente Skype for Business com uma conta de usuário que você acabou de configurar.
    
2. Ligue para um número PSTN pelo cliente e confirme se a chamada será bem-sucedida.
    
3. Use seu telefone ou outro dispositivo associado com o número PSTN para ligar para seu Cliente Skype for Business e confirmar se a chamada será bem-sucedida.
    
    > [!NOTE]
    > Você deve confirmar se pode fazer ligações bem-sucedidas depois de incluir sites adicionais ou converter sites para habilitar o suporte para alta disponibilidade.
