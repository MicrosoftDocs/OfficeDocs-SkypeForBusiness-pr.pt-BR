# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
O recurso de mobilidade no Skype for Business Server 2015 é compatível com a funcionalidade do Skype for Business em dispositivos móveis.
  
A funcionalidade de mobilidade é a mesma do Lync Server 2013. Para obter mais detalhes sobre como preparar seu ambiente para mobilidade, consulte [planejar o Mobility para o Skype for Business Server](../../plan-your-deployment/mobility.md).
