# <a name="disaster-recovery-planning-tool"></a>Disaster Recovery (Planning Tool)
 
O Skype for Business Server 2015 inclui medidas de recuperação de desastres, permitindo que você emparelhe pools de front-end localizados em dois data centers.
  
Para obter detalhes sobre como preparar seu ambiente para a recuperação de desastres, consulte [planejar a alta disponibilidade e a recuperação de desastres no Skype for Business Server 2015](../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md).
