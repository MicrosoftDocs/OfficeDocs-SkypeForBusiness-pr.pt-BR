# <a name="ip-version-planning-tool"></a>IP Version (Planning Tool)
 
Você tem três opções ao configurar endereços IP no Skype para Business Server 2015. Você pode configurar Skype para negócios 2015 de servidor dar suporte a IP versão 4 (IPv4), somente IP versão 6 (IPv6), ou uma combinação de ambos (conhecida como uma pilha dupla).
  
Para obter mais detalhes sobre como escolher a sua infra-estrutura de IP, consulte [Planejar IPv6 no Skype para negócios](../../plan-your-deployment/network-requirements/ipv6.md).
