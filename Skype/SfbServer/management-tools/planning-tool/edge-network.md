# <a name="edge-network-planning-tool"></a>Ferramenta de planejamento de rede de borda
 
A maioria dos componentes de Borda é implantada em uma rede de perímetro. Os componentes listados aqui comem a topologia de Borda da rede de perímetro. Exceto quando notados, os componentes fazem parte dos [cenários do Servidor de Borda no Skype for Business Server 2015](../../plan-your-deployment/edge-server-deployments/scenarios.md) e estão na rede de perímetro. Os componentes de Borda incluem:
  
- Servidores de borda
    
- Proxy reverso
    
- Firewall
    
- Diretores (opcional e localizado logicamente na rede interna)
    
- Balanceamento de carga para Topologias de Borda Dimensionadas (balanceamento de carga DNS ou um balanceador de carga de hardware)
