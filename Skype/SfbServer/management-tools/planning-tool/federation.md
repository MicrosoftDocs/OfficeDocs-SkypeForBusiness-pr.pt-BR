# <a name="federation-planning-tool"></a>Federation (Planning Tool)
 
Os Servidores de Borda podem ser configurados para permitir que seus usuários internos e externos acessem contatos em organizações ou serviços parceiros.
  
 A federação funciona como no Microsoft Lync 2013. Para obter mais detalhes sobre como preparar seu ambiente para Federação, consulte os seguintes tópicos:
  
- [Planejamento para federação do Lync Server 2013 e Office Communications Server](https://technet.microsoft.com/library/jj205335%28v=ocs.15%29.aspx)
    
- [Planejamento para conectividade de mensagens instantâneas públicas no Lync Server 2013](https://technet.microsoft.com/library/jj205349%28v=ocs.15%29.aspx)
    
- [Planejamento para federação XMPP (extensible messaging and presence protocol) no Lync Server 2013](https://technet.microsoft.com/library/jj205107%28v=ocs.15%29.aspx)
