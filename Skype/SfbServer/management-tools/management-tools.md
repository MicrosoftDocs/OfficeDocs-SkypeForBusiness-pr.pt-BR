# <a name="skype-for-business-server-2015-management-tools"></a>Ferramentas de gerenciamento do Skype for Business Server 2015
 
**Resumo:** Saiba mais sobre as ferramentas de gerenciamento do serviço do Skype para Business Server 2015.
  
Skype Business Server 2015 software de comunicações (anteriormente Lync Server) oferece mensagens instantâneas (IM), presença, conferência e soluções de telefonia que oferecem suporte a requisitos de colaboração de nível empresarial. As ferramentas para gerenciar esses serviços são flexíveis e poderosas.
