# <a name="skype-for-business-server-2015-management-tools"></a>Ferramentas de gerenciamento do Skype for Business Server 2015
 
**Resumo:** Saiba mais sobre as ferramentas de gerenciamento de serviço no Skype for Business Server 2015.
  
O software de comunicações do Skype for Business Server 2015 (anteriormente Lync Server), oferece soluções de mensagens instantâneas (IM), presença, conferência e telefonia que dão suporte a requisitos de colaboração no nível empresarial. As ferramentas para gerenciar esses serviços são flexíveis e poderosas.
