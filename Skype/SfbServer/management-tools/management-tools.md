# <a name="skype-for-business-server-2015-management-tools"></a>Ferramentas de Gerenciamento do Skype for Business Server 2015
 
**Resumo:** Saiba mais sobre as ferramentas de gerenciamento de serviços no Skype for Business Server 2015.
  
O software de comunicações do Skype for Business Server 2015 (antigo Lync Server), oferece soluções de mensagens instantâneas, presença, conferência e telefonia que oferecem suporte a requisitos de colaboração de nível empresarial. As ferramentas para gerenciar esses serviços são flexíveis e poderosas.
