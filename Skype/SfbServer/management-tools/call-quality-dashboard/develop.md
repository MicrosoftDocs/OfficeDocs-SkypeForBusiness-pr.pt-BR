# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desenvolver o Painel de Qualidade da Chamada para o Skype for Business Server
 
**Resumo:** Saiba como desenvolver soluções para o Painel de Qualidade da Chamada. O Painel de Qualidade da Chamada é uma ferramenta do Skype for Business Server.
  
Este tópico fornece documentação de desenvolvimento para o Painel de Qualidade de Chamadas (CQD) do Skype for Business Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desenvolver o Painel de Qualidade da Chamada para o Skype for Business Server

O CQD pode ser estendido e integrado ao seu ambiente de relatórios usando desenvolvimento personalizado. 
  
Dois conjuntos de APIs da Web estão disponíveis, um para acessar dados no cubo (por meio de APIs de Dados) e outro para acessar os metadados do relatório (por meio da API de Repositório). 
  
- [API de Dados para Painel de Qualidade de Chamadas (CQD) no Skype for Business Server](data-api.md)
    
- [REPOSITÓRIO API para Painel de Qualidade de Chamadas (CQD) no Skype for Business Server](repository-api.md)
    
- [Amostras de Desenvolvimento para o CQD](cqd-development-samples.md)
