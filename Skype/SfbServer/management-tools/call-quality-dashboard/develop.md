# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desenvolver o painel de controle de qualidade de chamada para Skype para Business Server
 
**Resumo:** Aprenda a desenvolver soluções para o painel de controle de qualidade de chamada. Painel de controle de qualidade de chamada é uma ferramenta para Skype para Business Server.
  
Este tópico fornece documentação de desenvolvimento para o painel de qualidade de chamada (CQD) para Skype para Business Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desenvolver o painel de controle de qualidade de chamada para Skype para Business Server

CQD pode ser estendido e integrado ao seu ambiente de relatórios usando o desenvolvimento personalizado. 
  
Dois conjuntos de APIs de web estão disponíveis, um para acessar os dados no cubo (por meio de APIs de dados) e outro para acessar os metadados de relatório (por meio da API do repositório). 
  
- [API de dados para o painel de qualidade de chamada (CQD) no Skype para Business Server](data-api.md)
    
- [API do repositório para o painel de qualidade de chamada (CQD) no Skype para Business Server](repository-api.md)
    
- [Amostras de Desenvolvimento para o CQD](cqd-development-samples.md)
