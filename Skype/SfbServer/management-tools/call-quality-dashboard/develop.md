# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desenvolver Painel de Qualidade de Chamada para Skype for Business Server
 
**Resumo:** Saiba como desenvolver soluções para o Painel de Qualidade de Chamada. O Painel de Qualidade de Chamada é uma ferramenta para Skype for Business Server.
  
Este tópico fornece documentação de desenvolvimento para o Painel de Qualidade de Chamada (CQD) para Skype for Business Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desenvolver Painel de Qualidade de Chamada para Skype for Business Server

O CQD pode ser estendido e integrado ao seu ambiente de relatórios usando o desenvolvimento personalizado. 
  
Dois conjuntos de APIs da Web estão disponíveis, um para acessar dados no cubo (por meio de APIs de Dados) e um para acessar os metadados do relatório (por meio da API do Repositório). 
  
- [API de dados para CQD (Painel de Qualidade de Chamada) no Skype for Business Server](data-api.md)
    
- [API de repositório para CQD (Painel de Qualidade de Chamada) no Skype for Business Server](repository-api.md)
    
- [Amostras de Desenvolvimento para o CQD](cqd-development-samples.md)
