# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desenvolva o painel de qualidade de chamada para o Skype for Business Server
 
**Resumo:** Saiba como desenvolver soluções para o painel de qualidade da chamada. O painel de qualidade de chamada é uma ferramenta para o Skype for Business Server.
  
Este tópico fornece documentação de desenvolvimento para o painel de qualidade de chamada (CQD) para o Skype for Business Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desenvolva o painel de qualidade de chamada para o Skype for Business Server

O CQD pode ser estendido e integrado ao seu ambiente de geração de relatórios usando desenvolvimento personalizado. 
  
Dois conjuntos de APIs da Web estão disponíveis, um para acessar dados no cubo (por meio de APIs de dados) e outro para acessar os metadados do relatório (por meio da API do repositório). 
  
- [API de dados para o painel de qualidade de chamada (CQD) no Skype for Business Server](data-api.md)
    
- [API do repositório para o painel de qualidade da chamada (CQD) no Skype for Business Server](repository-api.md)
    
- [Amostras de Desenvolvimento para o CQD](cqd-development-samples.md)
