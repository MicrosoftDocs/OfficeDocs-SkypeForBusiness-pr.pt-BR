# <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Desenvolver o painel de controle de qualidade de chamada para Skype para Business Server 2015
 
**Resumo:** Aprenda a desenvolver soluções para o painel de controle de qualidade de chamada. Painel de controle de qualidade de chamada é uma ferramenta para Skype para Business Server 2015.
  
Este tópico fornece documentação de desenvolvimento para o painel de qualidade de chamada (CQD) para Skype para Business Server 2015.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Desenvolver o painel de controle de qualidade de chamada para Skype para Business Server 2015

CQD pode ser estendido e integrado ao seu ambiente de relatórios usando o desenvolvimento personalizado. 
  
Dois conjuntos de APIs de web estão disponíveis, um para acessar os dados no cubo (por meio de APIs de dados) e outro para acessar os metadados de relatório (por meio da API do repositório). 
  
- [Dados API para painel de controle de qualidade de chamada (CQD) no Skype para Business Server 2015](data-api.md)
    
- [API do repositório para o painel de controle de qualidade de chamada (CQD) no Skype para Business Server 2015](repository-api.md)
    
- [Amostras de desenvolvimento CQD](cqd-development-samples.md)
