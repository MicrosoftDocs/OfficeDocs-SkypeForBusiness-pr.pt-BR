# <a name="call-quality-dashboard-for-skype-for-business-server-2015"></a>Painel de qualidade de chamada do Skype para Business Server 2015
 
**Resumo:** Saiba mais sobre o painel de qualidade de chamada, que é uma ferramenta de relatório do Skype para Business Server 2015.
 
O Skype para Business Server 2015 chamada qualidade Dashboard (CQD) é uma camada de relatório na parte superior de qualidade da experiência do banco de dados no Monitoring Server Skype para Business Server 2015 e Lync Server 2013. CQD usa o Microsoft SQL Server Analysis Services para fornecer uso agregado e filtragem, bem como informações de qualidade e dinamização no conjunto de dados de chamadas.
  
Os tópicos a seguir estão incluídos nesta seção e devem ser revisados antes de implantar o painel de controle de qualidade de chamada.
