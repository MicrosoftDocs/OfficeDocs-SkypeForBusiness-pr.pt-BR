# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Painel de qualidade de chamada do Skype para Business Server
 
**Resumo:** Saiba mais sobre o painel de qualidade de chamada, que é uma ferramenta de relatório do Skype para Business Server.
 
O Skype para Business Server chamada qualidade Dashboard (CQD) é uma camada de relatório na parte superior de qualidade da experiência do banco de dados no Monitoring Server Skype para Business Server. CQD usa o Microsoft SQL Server Analysis Services para fornecer uso agregado e filtragem, bem como informações de qualidade e dinamização no conjunto de dados de chamadas.
  
Os tópicos a seguir estão incluídos nesta seção e devem ser revisados antes de implantar o painel de controle de qualidade de chamada.

- [Implantar o painel de controle de qualidade de chamada para Skype para Business Server](deploy-0.md)
