# <a name="get-new-user-phone-numbers-request-forms"></a>Obtenha novos formulários de solicitação números de usuário de telefone

Às vezes (dependendo do seu país/região) não será possível obter seus números de novos usando o **Skype para centro de administração de negócios**ou, você precisará de números de telefone específico ou códigos de área. Nesse caso, você precisará fazer o download de um formulário e enviá-la de volta para nós. Nosso atendimento irá colocá-lo a partir daí e permitem que você saiba sobre o andamento ou seu pedido.
  
    > [!CAUTION]
    > In **Belgium, Ireland, Germany, and the Netherlands**, you can't get phone new phone numbers using the **Skype for Business admin center**; you have to download and fill out the form send it to us. 
  
É importante saber que, quando você estiver realizando as tarefas de gerenciamento de número de telefone, não existem [diferentes tipos de números de telefone usados para chamar planos](different-kinds-of-phone-numbers-used-for-calling-plans.md) que são usados durante a conexão de números de telefone Skype para Business Online. Se você estiver procurando por outras tarefas de gerenciamento de número de telefone, consulte [gerenciar números de telefone para sua organização](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md).

## <a name="what-do-you-need-to-do"></a>O que é preciso fazer?

- **Etapa 1** : Use os links de download abaixo para baixar o formulário correto.
- **Etapa 2** - preencha o formulário corretamente.
- **Etapa 3** - envie o formulário. Para números nos EUA, [enviá-la para nós](mailto:ptn@microsoft.com). Para números na União Europeia (UE), [enviá-la para nós](mailto:ptneu@microsoft.com).
