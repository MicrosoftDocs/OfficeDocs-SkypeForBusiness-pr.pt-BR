# <a name="phone-number-management-for-belgium"></a>Gerenciamento de número de telefone para Bélgica

Use esta tabela para encontrar informações sobre a obtenção e gerenciando números de telefone na Bélgica para Skype para negócios Online. 
  
Para obter mais informações, consulte [gerenciar números de telefone para sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Baixe os formulários corretos e enviá-las para nós**|
|:-----|:-----|
|Obtenha números de telefone do novo usuário.  <br/> |Na Bélgica, você não pode obter números de telefone para sua organização usando o Skype para centro de administração de negócios, você deve enviar uma nova solicitação de número de telefone. Baixe uma dessas formas: <br/> [Nova solicitação de número de telefone para Bélgica (números geográficas) (v.3.0)(de.be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(de.be).pdf) <br/> [Nova solicitação de número de telefone para Bélgica (números geográficas) (v.3.0)(fr.be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(fr.be).pdf) <br/> [Nova solicitação de número de telefone para Bélgica (números geográficas) (v.3.0)(nl.be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(nl.be).pdf) <br/>  Preencha-out e [enviá-lo para nós](mailto:ptneu@microsoft.com).  <br/> |
|Você precisa obter os números de telefone transferidos para nós.  <br/> | Baixe uma dessas formas: <br/> [Letra de autorização (LOA) para a Bélgica (números geográficas) (v.1.0)(nl.be)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(nl.be).pdf) <br/> [Letra de autorização (LOA) para a Bélgica (números geográficas) (v.1.0)(fr.be)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(fr.be).pdf) <br/> [Letra de autorização (LOA) para a Bélgica (números geográficas) (v.1.0)(de.be)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(de.be).pdf) <br/>  Preencha-out e [enviá-lo para nós](mailto:ptneu@microsoft.com). <br/> |
|Você precisará publicar/cancelar a publicação de números de telefone no diretório nacional de telefone.  <br/> |[Envie](mailto:ptneu@microsoft.com) a lista de números de telefone que você deseja publicar/cancelamento de publicação.      <br/> |
|Você precisa alterar o número de um usuário para um número de serviço.  <br/> |[Envie](mailto:ptneu@microsoft.com) a lista de números de usuário que você deseja converter em números de serviço. <br/> |
