# <a name="phone-number-management-for-the-us"></a>Gerenciamento de número de telefone para dos EUA

Use esta tabela para encontrar informações sobre a obtenção e gerenciando números de telefone nos Estados Unidos (e Porto Rico) para Skype para negócios Online. 
  
Para obter mais informações, consulte [gerenciar números de telefone para sua organização](manage-phone-numbers-for-your-organization.md).
  
|**O que você deseja fazer?**|**Siga estas etapas e baixe os formulários corretos**|
|:-----|:-----|
|Você precisa obter os números de usuário que não estejam listados no Skype para centro de administração de negócios.  <br/> |Baixe o [nova solicitação de número de telefone para EUA (números de usuário) (v.3.0(en.us)](../../downloads/new-phone-number-request-for-the-u.s.-(user-numbers)-(v.3.0)-(en.us).pdf) de formulário, preencha-out e [enviá-lo para nós](mailto:gcstnmsd@microsoft.com).  <br/> |
|Você precisa obter os números de telefone transferidos para nós.  <br/> | Baixe uma dessas formas: <br/> [Letra de autorização (LOA) para EUA (números de usuário e de serviço) (v.3.1)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(user-and-service-numbers)-(v.3.1)-(en.us).pdf) <br/> [Letra de autorização (LOA) para EUA (números para ligações gratuitas) (v.3.1)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(toll-free-numbers)-(v.3.1)-(en.us).pdf) <br/><br/>**Observação:** Se você estiver tentando fazer mais do que 999 números de telefone do assinante ou do usuário, você precisará preencher o check-out uma letra de autorização (LOA) e enviá-la para nós. Para obter mais informações, consulte [transferir os números de telefone para o Office 365](../../what-are-calling-plans-in-office-365/transfer-phone-numbers-to-office-365.md) .           Preencha-out e [enviá-lo para nós](mailto:ptn@microsoft.com).  <br/> |
|Você precisa alterar o número de um usuário para um número de serviço.  <br/> |[Envie](mailto:ptn@microsoft.com) seu PIN e a lista de números de usuário que você deseja converter em números de serviço. Para configurar um PIN, vá [aqui](../../what-are-calling-plans-in-office-365/set-your-pin-for-transferring-numbers-to-a-new-service-provider.md).  <br/> |
|Você precisa transferir os números de uma organização do Office 365 para outro.  <br/> |[Envie](mailto:ptn@microsoft.com) a lista de números de telefone do usuário (assinante) que deseja transferir, juntamente com o PIN da organização fonte onde você deseja transferir os números de. Para configurar um PIN, vá [aqui](../../what-are-calling-plans-in-office-365/set-your-pin-for-transferring-numbers-to-a-new-service-provider.md).  <br/> |
