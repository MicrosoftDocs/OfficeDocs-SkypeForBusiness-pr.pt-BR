# <a name="download-a-letter-of-authorization-loa"></a>Baixar uma carta de autorização (LOA) 

A **Letra de autorização (LOA)** nos dá permissão para solicitar em seu nome, transferindo o número existente (s) de um provedor de serviço diferentes em caso de uma contestação. Use um LOA se você tiver os números de serviço de pontes de conferência de áudio, atendedores automáticos ou outros números de serviço, gratuitos números de telefone, ou ter mais do que 999 números de telefone do usuário (assinante) que você precisa transferir para Skype para negócios (ou se você não deseja usar o **Assistente portar de número Local**). Você deve ser [manualmente enviar uma solicitação de serviço personalizado](manually-submit-a-custom-service-request.md) com este LOA para obter os números de telefone transferidos do seu provedor de serviço atual.
  
Se você tiver os números de serviço de pontes de conferência de áudio, atendedores automáticos ou outros números de serviço, gratuitos números de telefone, ou ter mais do que 999 números de telefone do usuário que você precisa transferir para Skype para a empresa, você precisa enviar manualmente uma ordem de porta.
  
Se você não tiver certeza dos tipos de números de telefone que você irá pedindo nossa transferir, consulte [diferentes tipos de números de telefone usados para chamar planos](different-kinds-of-phone-numbers-used-for-calling-plans.md).
  
> [!IMPORTANT]
> [!IMPORTANTE] Ao fazer a portabilidade manual de números de telefone usando uma LOA, **selecione o tipo correto de número de telefone**. É necessário **enviar pedidos de portabilidade separados** para cada tipo de número de telefone que você queira transferir.
