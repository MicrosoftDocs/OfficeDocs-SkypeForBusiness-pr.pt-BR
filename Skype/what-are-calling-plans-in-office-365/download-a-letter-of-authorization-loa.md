# Baixar uma carta de autorização (LOA)

> [!IMPORTANT]
> Este artigo foi traduzido por um sistema de tradução automática, leia o [aviso de isenção de responsabilidade](c0ab5bc9-44f1-46dd-b401-828e4f10b7ac.md#MT_Footer). Para sua referência, veja a versão em inglês deste artigo [aqui](https://support.office.com/en-us/article/c0ab5bc9-44f1-46dd-b401-828e4f10b7ac). 
  
A  * **Letra de autorização (LOA)*** nos dá permissão para solicitar, em seu nome, transferindo os números existentes de um provedor de serviços diferente no caso de uma controvérsia. Usar um LOA se você tiver números de serviço para pontes de conferência de áudio, atendedores automáticos ou outros números de serviço, chamada gratuitos números de telefone ou tiver mais do que 999 números de telefone do usuário (assinante) que você precisa transferir para Skype for Business (ou se você não quiser usar o **Assistente de portabilidade de número local** ). Você deve[Enviar uma solicitação de atendimento ao cliente manualmente](manually-submit-a-custom-service-request.md) com essa LOA para obter números de telefone transferidos do seu provedor de serviço atual.
  
Se você tiver números de serviço para pontes de conferência de áudio, atendedores automáticos ou outros números de serviço, chamada gratuitos números de telefone ou tiver mais do que 999 números de telefone do usuário que você precisa transferir para Skype for Business, você precisa enviar manualmente uma ordem de porta.
  
Se você não tiver certeza dos tipos de números de telefone que você vai estar se perguntando conosco transferir, consulte [Tipos diferentes de números de telefone usados para planos de chamada](different-kinds-of-phone-numbers-used-for-calling-plans.md).
  
> [!IMPORTANT]
> Ao fazer a portabilidade manual de números de telefone usando uma LOA, **selecione o tipo correto de número de telefone**. É necessário **enviar pedidos de portabilidade separados** para cada tipo de número de telefone que você queira transferir.
