# Conexões SIP diretas no Lync Server 2013

 

_**Tópico modificado em:** 2012-08-13_

Você pode usar *conexões SIP diretas* para conectar o Lync Server a qualquer um dos itens abaixo:

  - Um IP-PBX (para detalhes, consulte [Opções de implantação de SIP Direto no Lync Server 2013](lync-server-2013-direct-sip-deployment-options.md)).

  - Um gateway PSTN gateway (para detalhes, consulte [Opções de implantação do gateway PSTN no Lync Server 2013](lync-server-2013-pstn-gateway-deployment-options.md)).

Para implantar uma conexão SIP direta, você segue, basicamente, as mesmas etapas de implantação de um tronco SIP. Em cada caso, você implanta a conexão usando a interface externa de um Servidor de Mediação. A única diferença é que você conecta os troncos SIP a uma entidade externa, como um gateway ITSP, e conecta as conexões SIP diretas a uma entidade interna dentro da sua rede local, como um IP-PBX ou um gateway de uma Rede Telefônica Pública Comutada (PSTN).
