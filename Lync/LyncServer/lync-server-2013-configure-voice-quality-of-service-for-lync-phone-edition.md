# Configurar a qualidade de voz do serviço para o Lync Phone Edition

 

_**Tópico modificado em:** 2012-09-29_

É possível configurar os requisitos de Qualidade do Serviço (QoS) de voz para dispositivos do Lync Phone Edition em um pool configurando o nível de QoS para telefones IP que se conectam ao Lync Server 2013.

## Para configurar a Qualidade do Serviço de voz para o Lync Phone Edition

1.  Abra uma janela do navegador e insira a URL do Administrador para abrir o Painel de Controle do Lync Server. Para obter detalhes sobre os diferentes métodos que você pode usar para iniciar o Painel de Controle do Lync Server, consulte [Abrir ferramentas administrativas do Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

2.  Na barra de navegação esquerda, clique em **Cliente** e em **Configuração de Dispositivo**.

3.  Na página **Configuração de Dispositivo**, na lista de configurações de dispositivo, clique duas vezes na configuração da qual deseja você alterar as configurações de QoS.

4.  Em **Editar Configuração do Dispositivo**, em **Qualidade do serviço de voz**, especifique o nível de QoS. O nível padrão é **40**.
