# <a name="performing-an-abc-front-end-pool-failover-in-lync-server-2013"></a>Executando um failover de pool de front-ends ABC no Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<span> </span>

_**Última modificação do tópico:** 2013-02-21_

Os dois tópicos desta seção descrevem o procedimento para executar um failover de pool ABC no Lync Server 2013, onde há pools de front-ends do Lync Server A e B e o pool A se torna irrecuperável. Usando esse procedimento, você cria um novo pool de front-ends C com um novo FQDN (nome de domínio totalmente qualificado). O pool C é construído a partir das informações do pool A que falhou. O procedimento também inclui emparelhamento de pools B e C.

  - [Pré-requisitos de backup para failover de pool ABC no Lync Server 2013](lync-server-2013-backup-prerequisites-for-abc-pool-failover.md)

  - [Procedimento de failover ABC do pool de front-ends no Lync Server 2013](lync-server-2013-front-end-pool-abc-failover-procedure.md)
