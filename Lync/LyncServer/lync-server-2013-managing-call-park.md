# <a name="managing-call-park-in-lync-server-2013"></a>Gerenciando o estacionamento de chamadas no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-09-10_

O aplicativo de estacionamento de chamada permite que um usuário do Enterprise Voice Coloque uma chamada em espera de um telefone e, em seguida, recupere a chamada posteriormente de qualquer telefone. Quando o usuário armazena uma chamada, o Lync Server transfere a chamada para um número temporário, chamado de *órbita*, onde a chamada é mantida até que alguém a recupere ou expire.

Os tópicos desta seção fornecem procedimentos passo a passo para tarefas que você pode executar para personalizar e manter o aplicativo de estacionamento de chamada em sua implantação.
