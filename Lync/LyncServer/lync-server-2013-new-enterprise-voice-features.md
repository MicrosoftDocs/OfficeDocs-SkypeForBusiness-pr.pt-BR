# Novos recursos do Enterprise Voice no Lync Server 2013

 

_**Tópico modificado em:** 2013-05-01_

O Lync Server 2013 introduz vários novos recursos de gerenciamento de chamada e roteamento que melhoram o Enterprise Voice.

O Lync Server 2013 suporta vários troncos entre o Servidor de Mediação e gateways. Um *tronco* é uma associação lógica entre um número de porta e o Servidor de Mediação com um número de porta e gateway. Isto significa que um Servidor de Mediação pode ter vários troncos para diferentes gateways e um gateway pode ter vários troncos para Servidores de Mediação diferentes. O roteamento entre troncos torna possível o Lync Server 2013 interconectar um IP-PBX a um gateway PSTN ou interconectar vários sistemas IP-PBX. O Lync Server 2013 serve como a cola (isto é, a interconexão) entre diferentes sistemas de telefonia.

O Microsoft Lync Server 2013 faz melhorias nas áreas de encaminhamento de chamadas, ligação simultânea e apresentação da ID do chamador. Estes recursos enriquecem a experiência de chamada do Enterprise Voice.
