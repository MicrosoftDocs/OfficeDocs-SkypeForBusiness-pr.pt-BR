# <a name="shared-line-appearance-sla-cmdlets"></a>Cmdlets de aparência de linha compartilhada (SLA)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2016-04-12_

Os cmdlets relacionados à aparência de uma única linha (SLA) permitem que você configure um número de telefone compartilhado com vários representantes que podem pegar essa linha.

<div>

## <a name="sla-cmdlets"></a>Cmdlets de SLA

Use os cmdlets a seguir para o SLA.

**Configurando SLA**

  - <span></span>  
    [Get-CsSlaConfiguration](https://technet.microsoft.com/library/Mt703200(v=OCS.15))

  - <span></span>  
    [Set-CsSlaConfiguration](https://technet.microsoft.com/library/Mt703202(v=OCS.15))

  - <span></span>  
    [Remove-CsSlaConfiguration](https://technet.microsoft.com/library/Mt703201(v=OCS.15))

**Configurando representantes de SLA**

  - <span></span>  
    [Add-CsSlaDelegates](https://technet.microsoft.com/library/Mt703199(v=OCS.15))

  - <span></span>  
    [Remove-CsSlaDelegates](https://technet.microsoft.com/library/Mt703203(v=OCS.15))
