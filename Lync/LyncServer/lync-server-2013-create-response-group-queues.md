# Criar filas do Grupo de Resposta no Lync Server 2013

 

_**Tópico modificado em:** 2012-01-18_

As filas retêm os chamadores até que um agente atenda à chamada. Quando o Aplicativo Grupo de Resposta pesquisa um agente disponível, pesquisa grupos de agentes na ordem em que você os lista. Você pode selecionar os grupos de agentes que são atribuídos à fila e especificar o comportamento dela, como a limitação do número de chamadas que a fila pode reter e o período de tempo que uma chamada aguarda até que um agente a atenda.
