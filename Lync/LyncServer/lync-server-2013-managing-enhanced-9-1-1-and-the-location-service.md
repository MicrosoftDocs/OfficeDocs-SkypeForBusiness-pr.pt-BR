# <a name="managing-enhanced-9-1-1-and-the-location-service-in-lync-server-2013"></a>Gerenciando 9-1-1 Avançado e serviço de Local no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2012-11-01_

O Lync Server 2013 dá suporte à chamada Enhanced 9-1-1 (E9-1-1) dos clientes do Lync e dos dispositivos Lync Phone Edition. Quando você configura o Lync Server 2013 para E9-1-1, as chamadas de emergência do Lync 2013 ou do Lync Phone Edition incluem informações de local de resposta de emergência (ERL) do banco de dados do serviço de informações de localização. Use os procedimentos desta seção para gerenciar a política de localização.

<div>


> [!NOTE]  
> Para obter detalhes sobre a implantação de recursos avançados do Enterprise Voice, como E9-1-1 e o local Information Service, consulte <A href="lync-server-2013-deploying-advanced-enterprise-voice-features.md">implantando recursos avançados do Enterprise Voice no Lync Server 2013</A>.
