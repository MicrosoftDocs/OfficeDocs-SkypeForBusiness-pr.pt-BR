# <a name="configure-advanced-e9-1-1-features-in-lync-server-2013"></a>Configurar recursos avançados do E9-1-1 no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-06-06_

O Lync Server 2013 inclui os seguintes recursos que você pode usar para personalizar a implantação do E9-1-1:

  - Uma interface de serviço web para conectar o servidor de informações de local a um aplicativo do SNMP.

  - Uma interface de serviço web para a conexão com um banco de dados de fonte de localização secundária.
