# <a name="configure-advanced-e9-1-1-features-in-lync-server-2013"></a>Configurar recursos do E9-1-1 avançado no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2012-06-06_

O Lync Server 2013 inclui os seguintes recursos que você pode usar para personalizar a implantação do E9-1-1:

  - Uma interface de serviço Web para conectar o servidor de informações de localização a um aplicativo SNMP.

  - Uma interface de serviço Web para se conectar a um banco de dados de origem de local secundário.
