# Implantando mobilidade no Lync Server 2013

 

_**Tópico modificado em:** 2012-09-08_

Ao implantar o recurso de mobilidade do Lync Server 2013, os usuários móveis podem usar os dispositivos móveis suportados para a funcionalidade do Lync como mensagem instantânea (IM), presença e contatos.

Para obter detalhes sobre os requisitos de implantação do recurso de mobilidade, consulte [Planejamento para mobilidade no Lync Server 2013](lync-server-2013-planning-for-mobility.md).

Esta seção o orienta pelas etapas de implantação e verificação dos recursos de mobilidade e descoberta automática disponíveis.
