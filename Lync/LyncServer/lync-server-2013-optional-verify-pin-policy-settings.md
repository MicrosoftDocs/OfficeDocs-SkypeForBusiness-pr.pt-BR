# <a name="optional-verify-pin-policy-settings-in-lync-server-2013"></a>Opcion Verificar as configurações de política de PIN no Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-06-20_

Lync Server 2013 os usuários que têm credenciais de serviços de domínio Active Directory podem inserir um PIN (número de identificação pessoal) para ingressar em conferências discadas como usuários autenticados. Uma política de PIN define as regras de funcionamento de PINs de conferência discada.

Ao implantara conferência discada, verifique se a política de PIN global padrão atende aos seus requisitos. Se for necessário fazer alterações, você poderá modificar a política global padrão ou criar uma nova política de PIN. É possível criar políticas de PIN que se aplicam a um site específico, um usuário específico ou um grupo específico de usuários.
