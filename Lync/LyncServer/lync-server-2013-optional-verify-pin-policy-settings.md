# (Opcional) Verificar configurações de política de PIN no Lync Server 2013

 

_**Tópico modificado em:** 2012-06-20_

Os usuários do Lync Server 2013 que possuem credenciais dos Serviços de Domínio Active Directory podem inserir um PIN (número de identificação pessoal) para entrar em conferências como usuários autenticados. Uma política de PIN define as regras de funcionamento de PINs de conferência discada.

Ao implantara conferência discada, verifique se a política de PIN global padrão atende aos seus requisitos. Se for necessário fazer alterações, você poderá modificar a política global padrão ou criar uma nova política de PIN. É possível criar políticas de PIN que se aplicam a um site específico, um usuário específico ou um grupo específico de usuários.
