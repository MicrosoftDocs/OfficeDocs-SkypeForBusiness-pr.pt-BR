# Gerenciar parceiros XMPP federados no Lync Server 2013

 

_**Tópico modificado em:** 2012-10-19_

Esta documentação é preliminar e está sujeita a alterações. Os tópicos em branco são incluídos como espaços reservados.

Para gerenciar suporte a usuários de domínios federados XMPP, você precisa fazer o seguinte:

  - Configurar uma ou mais políticas de acesso externo para suportar usuários de domínios federados XMPP.

  - Configurar a política de Configuração de Borda de Acesso para suportar federação.

  - Criar definições de Parceiros Federados XMPP.

  - Compreender as configurações de negociação disponíveis para federação XMPP.

Para executar essas tarefas, use os procedimentos nesta seção.
