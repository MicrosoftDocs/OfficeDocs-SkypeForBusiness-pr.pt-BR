# <a name="location-based-routing-for-conferencing-in-lync-server-2013"></a>Roteamento baseado em local para conferências no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2013-07-31_

O roteamento baseado em local torna possível restringir o roteamento de chamadas entre pontos de extremidade VoIP e pontos de extremidade PSTN com base no local das partes na chamada. Com a atualização cumulativa 2 do Lync Server 2013, as regras de roteamento baseadas em local podem ser impostas em reuniões do Lync (ou seja, conferências) para impedir o bypass da PSTN. O aplicativo monitora uma conferência ativa e impõe restrições de roteamento com base na localização com base na localização dos usuários participantes. O aplicativo de conferência de roteamento baseado em localização Adicionalmente permite a imposição de restrições de roteamento baseado em localização para transferências consultivas envolvendo pontos de extremidade PSTN.
