# <a name="deploying-devices-in-lync-server-2013"></a>Implantando dispositivos no Lync Server 2013

_**Última modificação do tópico:** 2013-09-10_

O Lync Server 2013 inclui o Lync Phone Edition, software que é executado em dispositivos qualificados e fornece recursos de telefonia tradicionais e avançados, segurança integrada, capacidade de gerenciamento e muito mais. O Lync Phone Edition funciona da mesma maneira que o Lync Server 2013, como no Lync Server 2010. Para obter detalhes sobre a implantação de dispositivos, consulte [Deploying Lync Phone Edition](https://go.microsoft.com/fwlink/p/?linkid=285880) na biblioteca do TechNet do Lync Server 2010.
