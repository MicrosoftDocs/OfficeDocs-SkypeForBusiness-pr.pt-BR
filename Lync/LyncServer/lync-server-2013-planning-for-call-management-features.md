# <a name="planning-for-call-management-features-in-lync-server-2013"></a>Planejamento de recursos de gerenciamento de chamadas no Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-12-17_

Os recursos de gerenciamento de chamadas do Enterprise Voice controlam como as chamadas de entrada são encaminhadas e atendidas. O Lync Server 2013 fornece os seguintes recursos de gerenciamento de chamadas:

  - **Estacionamento de Chamada**: permite que os usuários de voz estacionem temporariamente uma chamada e a continuem do mesmo ou de outro telefone.

  - **Retirada de grupo**: permite que os usuários de voz escolham chamadas que estejam tocando para outros usuários de voz atribuídos a grupos de recebimento de chamadas.
    
    <div>
    

    > [!NOTE]  
    > A retirada de grupo é nova com atualizações cumulativas do Lync Server 2013:2013 de fevereiro.

    
    </div>

  - **Grupo de resposta**: encaminha chamadas de entrada para grupos de agentes usando grupos de busca ou perguntas e respostas de IVR (resposta interativa de voz).

  - **Anúncio:**     Reproduz uma mensagem para chamadas feitas para um número não atribuído ou roteia a chamada para outro local ou ambos.

Se você planeja implantar o Enterprise Voice, pode escolher implementar qualquer um ou todos esses recursos de gerenciamento de chamada.
