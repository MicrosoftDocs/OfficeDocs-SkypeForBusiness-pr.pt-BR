# Implantando o Lync Server 2013

 

_**Tópico modificado em:** 2012-10-18_

O processo de implantação do Lync Server 2013 é determinado pela topologia e pelos componentes do Lync Server que decidir instalar, incluindo se você deseja implantar um Pool de Front-Ends ou uma Servidor Standard Edition. Os tópicos nesta seção ajudam a determinar o ambiente no qual deseja implantar e guiam você pelo processo de implantação.
