# Definindo seus requisitos para chamadas de emergência no Lync Server 2013

 

_**Tópico modificado em:** 2015-03-09_

Antes de iniciar uma implantação de E9-1-1 do Microsoft Lync Server 2013, você deve primeiro ser capaz de responder às perguntas detalhadas nas seções a seguir. O planejamento necessário dependerá do tipo de solução E9-1-1 que você optar por implantar: um provedor de serviços E9-1-1 de tronco SIP ou um gateway ELIN (número de identificação de local de emergência). A tabela a seguir identifica as seções desta apostila de planejamento que você precisará consultar para cada uma dessas soluções.
