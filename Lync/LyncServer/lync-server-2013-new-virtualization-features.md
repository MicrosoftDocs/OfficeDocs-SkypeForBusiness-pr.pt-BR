# <a name="new-virtualization-features-in-lync-server-2013"></a>Novos recursos de virtualização no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2013-11-07_

O Lync Server 2013 oferece suporte à virtualização no Windows Server 2012, no Windows Server 2012 R2 e no Windows Server 2008 R2. O suporte no Windows Server 2012 e no Windows Server 2012 R2 inclui suporte para os recursos de virtualização de e/s de raiz única (SR-IOV). Com a SR-IOV, a função virtual de um adaptador de rede físico é atribuída diretamente a uma máquina virtual. Isso aumenta a transmissão e reduz a latência da rede, ao mesmo tempo que reduz a utilização da CPU host necessária para processar o tráfego de rede. Para tirar proveito da SR-IOV, você deve usar um servidor de host com BIOS que ofereça suporte à SR-IOV, além de adaptadores de rede que também ofereçam suporte à SR-IOV.
