# <a name="voice-routing-cmdlets-in-lync-server-2013"></a>Cmdlets de roteamento de voz no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-06-26_

As rotas de voz contêm instruções que dizem ao Microsoft Lync Server 2013 como rotear chamadas de usuários do Enterprise Voice para números de telefone na PSTN (rede telefônica pública comutada) ou um PBX (Private Branch Exchange).

<div>
