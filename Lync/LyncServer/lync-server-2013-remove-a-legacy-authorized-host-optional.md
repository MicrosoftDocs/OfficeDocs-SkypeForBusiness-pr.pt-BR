# Remover um host autorizado herdado no Lync Server 2013 (opcional)

 

_**Tópico modificado em:** 2012-09-05_

Quando você migra o controle de chamada remoto para uma implantação Lync Server, você deve remover entradas de hospedeiros autorizados herdadas (conhecidas como *entradas de aplicativos confiáveis* em Lync Server) para quaisquer gateways SIP/CSTA em sua implantação herdada. Você deve usar as ferramentas administrativas para remover as entradas de hospedeiros autorizados destas implantações, respectivamente.
