# <a name="enable-or-disable-dial-in-conferencing-for-meetings-in-lync-server-2013"></a>Habilitar ou desabilitar a conferência discada para reuniões no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-11-01_

O procedimento a seguir descreve como permitir que um usuário faça parte de uma reunião discada.

<div>

## <a name="to-enable-or-disable-dial-in-conferencing"></a>Para habilitar ou desabilitar a conferência discada

1.  Por uma conta de usuário atribuída à função CsUserAdministrator role ou CsAdministrator, faça o login em qualquer computador na sua implantação interna.

2.  Abra uma janela do navegador e insira a URL do Administrador para abrir o Painel de Controle do Lync Server. Para obter detalhes sobre os diferentes métodos que você pode usar para iniciar o painel de controle do Lync Server, consulte [Open Lync server 2013 Administrative Tools](lync-server-2013-open-lync-server-administrative-tools.md).

3.  Na barar de navegação à direita, clique em **Conferência** e, então, em **Política de Conferência**.

4.  Na lista de políticas de conferência, selecione a política que deseja ativar na conferência discada, clique em **Editar** e, então, em **Exibir detalhes**.

5.  Para permitir que os usuários participen por conexão discada, marque a caixa de verificação **Ativar a conferência via conexão discada PSTN**. Por padrão, os usuários podem discar para reuniões utilizando a 	Rede Telefônica Pública Comutada (PSTN).

6.  Clique em **Confirmar**.
