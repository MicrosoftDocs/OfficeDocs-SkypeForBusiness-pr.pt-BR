# Planejando o Servidor de Chat Persistente no Lync Server 2013

 

_**Tópico modificado em:** 2014-11-04_

É possível usar o Lync Server 2013, Servidor de Chat Persistente para permitir que vários usuários participem de conversas nas quais eles postam e acessam conteúdo sobre tópicos específicos, incluindo texto, links e arquivos. Embora os usuários possam se comunicar em tempo real durante uma sessão, o conteúdo de cada sessão é persistente, o que significa que continua disponível após o fim de uma sessão.

Esta seção descreve as considerações de planejamento em uma implantação do Lync Server 2013, Servidor de Chat Persistente, incluindo a definição de requisitos, a identificação de componentes e topologias com suporte e as recomendações de implantação.
