# <a name="setting-up-permissions-for-archiving-in-lync-server-2013"></a>Configurando permissões para arquivamento no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-10-01_

No Lync Server 2013, as tarefas específicas ainda exigem que os usuários que executam essas tarefas sejam membros de um ou mais grupos específicos. No entanto, você também pode usar o controle de acesso baseado em função (RBAC) para conceder privilégios, atribuindo usuários a funções administrativas predefinidas do Lync Server. Antes de implantar o arquivamento, certifique-se de que os direitos e permissões do usuário adequados estejam no local e que qualquer usuário que você deseja atribuir a uma função RBAC específica tenha sido atribuído a essa função. Para obter detalhes sobre os direitos, as permissões e as funções do usuário para a implantação do suporte para arquivamento, consulte [Deployment Checklist for Archiving in Lync Server 2013](lync-server-2013-deployment-checklist-for-archiving.md), disponível na documentação de planejamento e na documentação de implantação. Para obter detalhes sobre o RBAC, consulte [Planning for Role-Based Access Control in Lync Server 2013](lync-server-2013-planning-for-role-based-access-control.md) na documentação de planejamento.
