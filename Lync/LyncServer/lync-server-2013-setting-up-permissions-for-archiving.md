# Configurando permissões para arquivamento

 

_**Tópico modificado em:** 2012-10-01_

No Lync Server 2013, tarefas específicas ainda exigem que os usuários que as executam sejam membros de um ou mais grupos específicos. No entanto, você também pode usar o RBAC (controle de acesso baseado em função) para conceder privilégios através da atribuição de usuários a funções administrativas predefinidas do Lync Server. Antes de implantar o arquivamento, verifique se as permissões e os direitos de usuário adequados estão implementados e se os usuários que você deseja atribuir a uma função RBAC específica estão atribuídos à função em questão. Para obter detalhes sobre os direitos de usuário, as permissões e as funções para implantar o suporte ao arquivamento, consulte [Lista de verificação da implantação para Arquivamento no Lync Server 2013](lync-server-2013-deployment-checklist-for-archiving.md), que está disponível nas documentações de planejamento e de implantação. Para obter detalhes sobre o RBAC, consulte [Planejamento de controle de acesso baseado em função no Lync Server 2013](lync-server-2013-planning-for-role-based-access-control.md) na documentação de planejamento.
