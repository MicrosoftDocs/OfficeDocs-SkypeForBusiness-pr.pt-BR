# Gerenciando grupos de resposta no Lync Server 2013 durante um desastre

 

_**Tópico modificado em:** 2012-11-01_

O Lync Server 2013 suporta grupos de resposta executadas no pool de backup durante a recuperação de desastres. Esta seção descreve como planejar grupos de resposta durante uma interrupção de energia, como os grupos de resposta trabalham durante a interrupção e o procedimento necessário para os grupos de resposta de fail over e fail back.
