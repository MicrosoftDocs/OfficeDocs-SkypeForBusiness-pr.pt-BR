# <a name="associate-subnets-with-network-sites-for-e9-1-1-in-lync-server-2013"></a>Associar sub-redes a sites de rede para o E9-1-1 no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-10-02_

Cada sub-rede em sua rede que você deseja habilitar para o E9-1-1 deve estar associada a um site de rede específico. A razão para isso é que as informações de sub-rede são usadas para determinar o site de rede em que um ponto de extremidade está localizado.

E9-1-1 não tem nenhum requisito especial para associar sub-redes a sites de rede. Para criar uma associação entre as sub-redes e os sites de rede em sua topologia, siga os procedimentos em [associar uma sub-rede a um site de rede no Lync Server 2013](lync-server-2013-associate-a-subnet-with-a-network-site.md).
