# <a name="managing-call-admission-control-in-lync-server-2013"></a>Gerenciando o controle de admissão de chamadas no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-11-01_
