# <a name="back-up-systems-and-data"></a>Fazer backup de dados e sistemas

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2012-09-18_

Antes de começar a migração para o Lync Server 2013, recomendamos que você execute um backup completo do sistema e documente o sistema existente, incluindo um inventário de contas de usuário que são hospedadas em cada pool, para que você possa reverter para o Lync Server 2010 se ele Se for necessário. Há várias ferramentas e programas disponíveis para fazer o backup e a restauração de dados, configurações e sistemas.

Para obter detalhes e procedimentos, consulte [fazendo backup e restaurando o Lync Server 2010](http://go.microsoft.com/fwlink/p/?linkid=265417) .
