# <a name="clients-for-lync-server-2013"></a>Clientes para Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2014-01-23_

O Lync Server 2013 oferece suporte a vários tipos de software cliente que você pode implantar para os usuários da sua organização, incluindo software cliente instalado no computador, clientes baseados na Web e dispositivos móveis. Este tópico descreve os clientes que você pode usar. Para obter uma comparação detalhada dos recursos fornecidos pelos clientes do Lync Server 2013, consulte [Client Comparison Tables for Lync server 2013](lync-server-2013-desktop-client-comparison-tables.md).

<div>

## <a name="lync-2013"></a>Lync 2013

O Lync 2013 é o cliente completo do Lync Server. A interface de usuário do Lync 2013 foi totalmente reprojetada e inclui recursos integrados recentemente, como chat persistente (o Lync 2010 tinha um cliente separado para funcionalidade de chat), conversas com guias, visualização de vídeo e vídeo com vários participantes. Para obter um resumo das alterações, consulte [What ' s New for clients in Lync Server 2013](lync-server-2013-what-s-new-for-clients.md).

O Lync 2013 Client setup faz parte do programa de instalação do Office na mídia de instalação.

</div>

<div>

## <a name="online-meeting-add-in-for-lync-2013"></a>Suplemento de reunião online para Lync 2013

O suplemento reunião online do Lync 2013 oferece suporte ao gerenciamento de reuniões de dentro do cliente de colaboração e mensagens do Microsoft Outlook. O suplemento de reunião online para o software Lync 2013 é instalado automaticamente com o Lync 2013.

</div>

<div>

## <a name="lync-web-scheduler"></a>Agendador do Lync Web

Lync Web Scheduler é uma ferramenta de gerenciamento e agendamento de reuniões baseada na Web para usuários que não têm acesso ao Microsoft Outlook, ou que estão em um sistema operacional não baseado no Windows. Com o Lync Web Scheduler, os usuários podem criar novas reuniões, modificar reuniões existentes e enviar convites usando seu programa de email preferencial.

</div>

<div>

## <a name="lync-web-app"></a>Lync Web App

Lync Web App é o cliente de conferência baseado na Web para reuniões do Lync Server 2013. Nesta versão, a adição de áudio e vídeo do computador ao Lync Web App oferece uma experiência completa de reunião para qualquer pessoa que não tenha um cliente Lync instalado localmente. Os participantes da reunião têm acesso a todos os recursos de colaboração e compartilhamento e controles de reunião de apresentador.

Se o Lync 2013 não estiver instalado no computador de um usuário e o usuário clicar em um link de reunião em uma solicitação de reunião, o Lync Web App será aberto. Você também pode configurar a página de ingresso na reunião para permitir que os usuários ingressem em reuniões usando versões anteriores de clientes; consulte [Configurando a página de ingresso na reunião no Lync Server 2013](lync-server-2013-configuring-the-meeting-join-page.md) na documentação de implantação.

Devido aos aprimoramentos do Lync Web App, uma versão atualizada do participante do Microsoft Lync 2010 não está disponível para o Lync Server 2013. O Lync Web App é o cliente escolhido para participantes de fora da sua organização. Com o Lync Web App, não é necessária nenhuma instalação de cliente local, embora os recursos de áudio, vídeo e compartilhamento exijam a instalação de um plug-in durante o primeiro uso.

</div>

<div>

## <a name="lync-2013-basic"></a>Lync 2013 Basic

O Lync 2013 Basic é um cliente que pode ser baixado para clientes que têm uma implantação do Lync Server 2013 local e licenciado e clientes que assinam um plano do Microsoft Office 365 que não inclui o cliente completo do Lync 2013. O cliente do Lync Basic inclui presença avançada, contatos, mensagens instantâneas (IM), reuniões do Lync e funcionalidade de voz básica. Recursos não suportados no Lync Basic incluem vídeo com vários participantes, integração com o OneNote, suporte a infraestrutura de área de trabalho virtual (VDI), pesquisa de habilidades, gravação, recursos do Enterprise Voice e tratamento de chamada avançada (por exemplo, encaminhamento de chamadas e chamada de equipe). Para obter detalhes, consulte [Client Comparison Tables for Lync Server 2013](lync-server-2013-desktop-client-comparison-tables.md).

</div>

<div>

## <a name="lync-windows-store-app"></a>Aplicativo Lync da Windows Store

O aplicativo Lync da Windows Store é um aplicativo Lync otimizado para toque, projetado especificamente para o Windows 8,1, Windows 8 e Windows RT. Os usuários podem baixar o aplicativo por meio da Windows Store pesquisando por "Lync". Para obter mais informações, consulte [Client Comparison Tables for Lync server 2013](lync-server-2013-desktop-client-comparison-tables.md), [Lync Windows Store app Requirements for Lync Server 2013](lync-server-2013-lync-windows-store-app-requirements.md)e [Deploying Lync Windows store app in Lync Server 2013](lync-server-2013-deploying-lync-windows-store-app.md).

</div>

<div>

## <a name="lync-2013-for-mobile-devices"></a>Lync 2013 para dispositivos móveis

Os aplicativos móveis do Lync 2013 agora incluem recursos de VoIP (voz sobre IP) e vídeo sobre IP, além de contatos, presença e recursos de mensagens instantâneas. Os usuários móveis podem optar por se comunicar com outras pessoas por meio de mensagens instantâneas, chamadas de voz ou chamadas de vídeo usando o Wi-Fi ou sua conexão de dados da rede celular. Com um único clique do link de reunião em um item de calendário, os usuários móveis podem participar de reuniões de voz e vídeo. Para obter mais informações sobre os aplicativos móveis do Lync 2013, consulte [Planning for Mobile clients in Lync Server 2013](lync-server-2013-planning-for-mobile-clients.md).

</div>

<div>

## <a name="supported-clients-from-previous-releases"></a>Clientes com suporte de versões anteriores

O Lync Server 2013 oferece suporte aos seguintes clientes de versões anteriores do servidor. Você pode tornar determinados clientes anteriores disponíveis aos usuários quando eles ingressarem em reuniões. Para obter detalhes, consulte [Configurando a página de ingresso na reunião no Lync Server 2013](lync-server-2013-configuring-the-meeting-join-page.md) na documentação de implantação.

  - **Lync 2010**   Lync 2010 fornece uma experiência de área de trabalho completa, incluindo im, presença avançada, voz, vídeo, compartilhamento e telefonia. No entanto, nenhum dos novos recursos introduzidos no Lync Server 2013 estará disponível até que o cliente do usuário seja atualizado para o Lync 2013.

  - **Lync 2010 Mobile**   Lync Server 2013 oferece suporte a todos os aplicativos móveis do Microsoft Lync 2010. O Microsoft Lync 2010 Mobile fornece mensagens instantâneas, presença avançada e telefonia para usuários em sua organização que estão se conectando de um smartphone ou um telefone que executa uma edição profissional do Windows Mobile. Você pode instruir seus usuários a instalar o Microsoft Lync 2010 Mobile direcionando-os para o Marketplace de aplicativos para seu celular. Para obter detalhes, consulte "Planning for Mobile clients" na documentação do Lync Server [https://go.microsoft.com/fwlink/p/?LinkID=235955](https://go.microsoft.com/fwlink/p/?linkid=235955)2010 em.

  - **Lync Phone Edition**   o software Lync Phone Edition para telefones IP inteligentes (por exemplo, telefones conectados a USB) não foi atualizado para o Lync Server 2013. O Lync Phone Edition continua a ter suporte para colocar e receber chamadas, presença avançada e recursos de áudio de cliente para conferências.

  - **Lync 2010 Attendant**   o programa integrado de gerenciamento de chamadas do Microsoft Lync 2010 atende à recepcionista para gerenciar várias conversas ao mesmo tempo através de tratamento rápido de chamada, mensagens instantâneas e roteamento na tela.

</div>
