# <a name="deploying-call-management-features-in-lync-server-2013"></a>Implantando recursos de gerenciamento de chamadas no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-12-18_

Os recursos de gerenciamento de chamadas do Enterprise Voice controlam como as chamadas de entrada são encaminhadas e atendidas. O Lync Server 2013 fornece os seguintes recursos de gerenciamento de chamadas:

  - **Estacionamento de chamada:** Permite que os usuários de voz disparkm temporariamente uma chamada e, em seguida, selecione-o do mesmo telefone ou outro telefone.

  - **Retirada de Grupo:** Permite que os usuários respondam às chamadas feitas para outro usuário que é atribuído a um grupo de retirada discando o número do grupo de recebimento de chamada.

  - **Grupo de resposta:** Encaminha as chamadas de entrada para grupos de agentes usando grupos de busca ou perguntas e respostas de IVR (resposta interativa de voz).

  - **Anúncio:** Reproduz uma mensagem para chamadas feitas para um número não atribuído ou roteia a chamada para outro local ou ambos.

Esta seção descreve como configurar esses recursos de gerenciamento de chamada durante uma implantação do Enterprise Voice.
