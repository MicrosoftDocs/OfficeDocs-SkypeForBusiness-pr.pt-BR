# Calendário separado para reunião grande

 

_**Tópico modificado em:** 2012-10-01_

Para cada pool de reunião grande, você deve manter um calendário de reuniões grandes agendadas separado nesse pool. Por exemplo, você pode hospedar uma única conta de usuário no pool de reunião grande e utilizar o Outlook com o Exchange e o Suplemento de Reunião Online para Lync 2013 para manter um calendário separado. Se você utilizar várias contas de usuário para habilitar uma equipe de suporte para criar reuniões grandes, será necessário configurar um calendário separado que agregue todas as reuniões grandes criadas pelos membros da equipe de suporte.

Manter um calendário de reunião grande separado ajuda a prevenir conflitos e garante que somente uma reunião grande esteja ativa por vez.
