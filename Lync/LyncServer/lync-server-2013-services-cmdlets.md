# <a name="services-cmdlets-in-lync-server-2013"></a>Cmdlets de serviços no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2012-06-20_

Os cmdlets de serviços e função de servidor que vêm com o Microsoft Lync Server 2013 permitem que você retorne informações sobre os vários componentes que foram instalados como parte de sua implantação do Lync Server 2013. Além disso, esses cmdlets facilitam a interrupção e o início dos serviços do Lync Server.

<div>

## <a name="services-cmdlets"></a>Cmdlets de serviços

Veja a seguir uma lista de cmdlets relacionados diretamente ao serviço de gerenciamento:

**Aos**

  - <span></span>  
    [Get-CsService](https://technet.microsoft.com/en-us/library/Gg413038(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [Get-CsWindowsService](https://technet.microsoft.com/en-us/library/Gg398803(v=OCS.15))

  - <span></span>  
    [Start-CsWindowsService](https://technet.microsoft.com/en-us/library/Gg398561(v=OCS.15))

  - <span></span>  
    [Parar-CsWindowsService](https://technet.microsoft.com/en-us/library/Gg398426(v=OCS.15))
