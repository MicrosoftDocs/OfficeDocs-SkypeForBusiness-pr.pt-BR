# <a name="server-software-and-infrastructure-support-in-lync-server-2013"></a>Suporte a software e à infraestrutura de servidor no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2012-06-29_
