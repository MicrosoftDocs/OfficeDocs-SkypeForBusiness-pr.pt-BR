## <a name="in-this-section"></a>Nesta seção

  - [Decidindo como implantar o Lync Server 2013](lync-server-2013-deciding-how-to-deploy-microsoft-lync.md) fornece diretrizes para escolher entre esses cenários básicos de implantação do Lync Server 2013.

  - As [decisões iniciais de planejamento do Lync server 2013](lync-server-2013-initial-planning-decisions.md) o orientam pelas perguntas que você deve responder para decidir quais cargas de trabalho e recursos do Lync Server implantar.

  - As [topologias de referência no Lync Server 2013](lync-server-2013-reference-topologies.md) mostram três topologias de exemplo que ilustram um bom design de topologia em três tipos de organização típicas e explica o raciocínio por trás de muitas das decisões no projeto dessas topologias.
