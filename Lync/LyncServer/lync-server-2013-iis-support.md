_**Última modificação do tópico:** 2012-06-29_

Vários componentes do Lync Server 2013 exigem serviços de informações da Internet (IIS). As versões com suporte do IIS incluem o seguinte:

  - IIS 8,0

  - IIS 7,5

Para obter detalhes sobre os requisitos do IIS para os componentes do Lync Server 2013, consulte [requisitos de serviços de informações da Internet (IIS) no Lync server 2013](lync-server-2013-internet-information-services-iis-requirements.md) na documentação de planejamento.
