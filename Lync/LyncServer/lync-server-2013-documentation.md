# <a name="documentation-in-lync-server-2013"></a>Documentação no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2015-05-15_

O modelo MOF é composto por muitas funções de gerenciamento de serviços. A documentação sobre como e quando tarefas são executadas pode ser compartilhada com membros da mesma equipe ou com outras equipes. O método de armazenamento e compartilhamento da documentação pode variar de acordo com o tipo de função. Por exemplo, os procedimentos para administração do sistema podem ser armazenados como documentos do Word, pois eles provavelmente serão impressos e referenciados com frequência. As informações de gerenciamento de configuração podem ser geradas e armazenadas automaticamente em um banco de dados para facilitar a pesquisa e indexação. Algumas documentações podem ser confidenciais e devem ser restringidas.

<div>

## <a name="document-management-systems"></a>Sistemas de gerenciamento de documentos

Um sistema de gerenciamento de documentação atua como um repositório central para documentos e ajuda a garantir que apenas a revisão mais recente de um documento esteja disponível. Você também pode considerar o arquivamento da versão antiga do documento para referência. O Lync Server fornece funcionalidade adequada para essa tarefa.

</div>

<div>

## <a name="databases"></a>Bancos de dados

Várias ferramentas e funções de gerenciamento foram discutidas que são adequadas para o uso de bancos de dados. O processo de gerenciamento de configuração provavelmente usará processos automatizados que armazenam grandes quantidades de dados que exigem indexação e pesquisa. A equipe de suporte pode pesquisar um banco de dados de problemas e resoluções anteriores ao solucionar novos problemas.

É provável que haja diferentes bancos de dados sendo usados para finalidades diferentes. Decida se esses bancos de dados devem ser vinculados ou combinados. Por exemplo, se o Service Desk identificar vários problemas com um tema comum (como novo software causando um problema com um adaptador de rede específico), a equipe de suporte poderá consultar o banco de dados de configuração para prever quantos computadores podem ser afetados.
