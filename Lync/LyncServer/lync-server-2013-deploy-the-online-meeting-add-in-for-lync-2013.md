# Implantar suplemento Online Meeting para Lync 2013

 

_**Tópico modificado em:** 2012-09-30_

Implante o Suplemento de Reunião Online para Lync 2013 para que os usuários possam agendar conferência com suporte para conferência discada.

O Suplemento de Reunião Online para Lync 2013 é instalado automaticamente ao instalar o Lync 2013. Para obter detalhes sobre a personalização do Suplemento de Reunião Online para Lync 2013 para Lync, consulte [Configurando o convite de reunião no Lync Server 2013](lync-server-2013-configuring-the-meeting-invitation.md).
