# <a name="web-server-and-services-cmdlets-in-lync-server-2013"></a>Cmdlets de serviços e servidor Web no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2013-02-18_

Muitos componentes do Microsoft Lync Server 2013 são baseados na Web: esses componentes usam serviços Web ou páginas da Web para executar suas tarefas. Os cmdlets do servidor Web e dos serviços Web permitem que você faça coisas como as configurações do servidor Web de configuração e o gerenciamento de URLs simples. URLs simples tornam mais fácil para os usuários ingressarem em reuniões e conferências e facilitar para os administradores a fazer logon no painel de controle do Lync Server 2013.
