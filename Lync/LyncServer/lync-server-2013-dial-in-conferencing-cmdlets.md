# <a name="dial-in-conferencing-cmdlets-in-lync-server-2013"></a>Cmdlets de conferência discada no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-03-21_

A conferência discada permite aos usuários utilizar um telefone "comum" (ou seja, um dispositivo da rede telefônica pública comutada) para participar da parte de áudio de uma conferência.

<div>

## <a name="dial-in-conferencing-cmdlets"></a>Cmdlets de conferência discada

Se a intenção for não permitir a conferência discada, é possível desabilitar esse recurso usando o cmdlet Set-CsConferencingPolicy e definindo a propriedade EnableDialInConferencing como Falso. Por padrão, todos os usuários podem coordenar reuniões que incluam a conferência discada.
