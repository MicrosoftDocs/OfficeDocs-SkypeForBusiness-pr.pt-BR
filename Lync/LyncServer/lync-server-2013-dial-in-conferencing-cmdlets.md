# <a name="dial-in-conferencing-cmdlets-in-lync-server-2013"></a>Cmdlets de conferência discada no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2012-03-21_

A conferência discada fornece uma maneira para os usuários usarem um telefone "regular" (ou seja, um dispositivo na rede telefônica comutada pública) para ingressar na parte de áudio de uma conferência.

<div>

## <a name="dial-in-conferencing-cmdlets"></a>Cmdlets de conferência discada

Se você não quiser permitir a conferência discada, poderá desabilitar essa funcionalidade usando o cmdlet Set-CsConferencingPolicy e definindo a propriedade EnableDialInConferencing como false. Por padrão, todos os usuários podem hospedar reuniões que incluam conferência discada.
