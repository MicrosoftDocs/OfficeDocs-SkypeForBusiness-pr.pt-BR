# <a name="configure-clients-for-migration"></a>Configurar clientes para migração

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2012-09-24_

Este tópico contém as etapas de implantação de cliente recomendadas que você deve executar antes de migrar para o Lync Server 2013. Essas alterações de configuração devem ser feitas no Lync Server 2010.

<div>

## <a name="to-configure-clients-before-migration"></a>Para configurar clientes antes da migração

1.  Implantar as atualizações mais recentes do servidor, do cliente e do dispositivo (hotfixes) para o Lync Server 2010. Para obter as informações mais atualizadas sobre o Lync Server 2010, consulte atualizar o centro de recursos do Lync <http://go.microsoft.com/fwlink/p/?linkid=232630>em.

2.  No Lync Server 2010, use a filtragem de versão do cliente para permitir somente aos clientes com as atualizações mais recentes instaladas.
