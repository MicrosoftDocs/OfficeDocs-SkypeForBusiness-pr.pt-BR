## <a name="in-this-section"></a>Nesta seção

Para atender aos SLAs de arquitetura de referência e garantir uma transição tranqüila para nossas equipes de suporte, uma abordagem de solução de problemas comum deve ser definida juntamente com um conjunto obrigatório de ferramentas e abordagens de solução de problemas conforme definido no guia de rede do Lync Server [ ](http://go.microsoft.com/fwlink/p/?linkid=390677) .

É altamente recomendável que o System Center Operations Manager seja usado para monitorar a integridade do sistema do Lync Server 2013. Além disso, confira a discussão sobre o KHIs no guia de [rede](http://go.microsoft.com/fwlink/p/?linkid=390677) do Lync Server 2013 e a planilha do Excel para usar com o Lync 2013.
