# Lync Server 2013 híbrido

 

_**Tópico modificado em:** 2016-05-25_

Com as implantações híbridas do Lync, você pode ter alguns de seus usuários do Lync no local e outros no Lync Online, todos compartilhando o mesmo domínio. Isso pode facilitar o fornecimento de serviços Lync a usuários de sua organização em diferentes regiões geográficas ou usuários com conexão remota. Você também pode desfrutar de configurações híbridas do Lync como caminho de migração ao Office 365.

Esta seção descreve as configurações híbridas para o Lync Server 2013
