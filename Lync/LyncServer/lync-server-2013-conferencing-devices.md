# <a name="conferencing-devices-in-lync-server-2013"></a>Dispositivos de conferência no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2014-02-05_

Os dispositivos de conferência trazem recursos do Lync Phone Edition para salas de conferência, permitindo que as pessoas na sala escutem e, dependendo do dispositivo, vejam pessoas em outros locais.

Use os tópicos desta seção para aprender a configurar e gerenciar seus dispositivos de conferência.

<div>


> [!NOTE]  
> Para obter mais informações sobre os dispositivos de conferência com suporte para uso com o Lync Server 2013, consulte os telefones da página da Web <A href="http://technet.microsoft.com/lync/gg278164.aspx">e os dispositivos qualificados para o Microsoft Lync</A>.
