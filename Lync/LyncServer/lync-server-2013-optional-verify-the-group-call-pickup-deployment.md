# <a name="optional-verify-the-group-call-pickup-deployment-in-lync-server-2013"></a>Opcion Verificar a implantação do recebimento de chamadas em grupo no Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2013-01-30_

Depois de habilitar o recebimento de chamadas em grupo para usuários, verifique se a configuração funciona conforme o esperado. No mínimo, verifique o seguinte:

  - Ligue para um usuário que esteja habilitado para recebimento de chamadas em grupo e peça a outro usuário para recuperar a chamada. O outro usuário pode estar no mesmo grupo, em um grupo diferente ou não ter o recebimento de chamadas em grupo habilitado.

  - Ligue para um usuário que esteja habilitado para recebimento de chamadas em grupo e não responda à chamada.
