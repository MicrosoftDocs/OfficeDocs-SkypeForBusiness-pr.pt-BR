# <a name="viewing-status-of-pools-in-lync-server-2013"></a>Exibindo o status de pools no Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2014-05-01_

Para cada pool corporativo e o pool de cada servidor Standard Edition, você pode exibir informações sobre o status do pool e o status de outros componentes usados pelo pool por meio do painel de controle do Lync Server 2013. Recomendamos a implementação do System Center Operations Manager 2012 com o painel de controle do Lync Server para ajudar os administradores a exibir a integridade dos pools em seu ambiente.

<div>

## <a name="view-pools"></a>Exibir pools

1.  Usando o painel de controle do Lync Server, abra o painel de controle do Lync Server.

2.  Clique em topologia e verifique o status de todos os servidores na topologia.
