# <a name="managing-lync-server-2013-archiving"></a>Gerenciando o arquivamento do Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**Última modificação do tópico:** 2012-10-10_

Ao implantar o arquivamento para sua organização, você especifica a configuração inicial durante a implantação. No entanto, pode haver momentos em que você deseja alterar a forma como você implementa o suporte de arquivamento para o gerenciamento cotidiano ou para atender aos novos requisitos em sua organização. Por exemplo, talvez você precise configurar o suporte de arquivamento de forma diferente para um site, pool ou usuários específicos em sua organização. Para usuários hospedados no Lync Server 2013, é possível criar e personalizar políticas e configurações de arquivamento. Se você usar a integração com o Microsoft Exchange, também deverá definir as configurações do Exchange 2013. Esta seção fornece informações e procedimentos para permitir que você faça alterações na sua implantação de arquivamento.
