# Gerenciando a infraestrutura de rede do Lync Server 2013

 

_**Tópico modificado em:** 2014-02-11_

O Microsoft Lync Server 2013 inclui suporte para CAC e bypass de mídia. Para implementar estes recursos, você deve configurar uma rede de regiões, sites, subredes e assim por diante, que permitirão gerenciar a largura de banda em situações onde as transmissões de áudio e vídeo precisam ser restritas. Também é possível usar a tecnologia de rede QoS para oferecer uma melhor experiência ao usuário final para comunicações de áudio e vídeo.

É possível usar o Painel de Controle do Lync Server para configurar e gerenciar CAC, bypass de mídia e QoS. Os seguintes tópicos oferecem as etapas sobre como fazer isso.
