_**Última modificação do tópico:** 2012-10-01_

Os serviços de servidor de chat persistente do Lync Server 2013 usam uma configuração de *pool estendido* para recuperação de desastres. Um pool estendido é um pool que tem computadores distribuídos entre dois data centers físicos, mas que estão dentro de um único site lógico do Lync Server.
