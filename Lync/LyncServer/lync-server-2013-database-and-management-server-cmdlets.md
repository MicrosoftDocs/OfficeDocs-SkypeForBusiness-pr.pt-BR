# <a name="database-and-management-server-cmdlets-in-lync-server-2013"></a>Cmdlets de banco de dados e servidor de gerenciamento no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-10-09_

Os cmdlets do banco de dados e do servidor de gerenciamento são usados para gerenciar os bancos de dados de back-end do Microsoft Lync Server 2013 e seus serviços de gerenciamento de front-end. Você pode usar esses cmdlets para instalar ou desinstalar qualquer um dos bancos de dados usados pelo Lync Server 2013, além de configurar o ponto de controle de serviço do Active Directory para o repositório de gerenciamento central.
