# Definições de Configuração do Lync Phone Edition no Lync Server 2013

 

_**Tópico modificado em:** 2012-10-10_

As definições de configuração executando o Lync Phone Edition se aplicam globalmente ou você pode criar novos conjuntos de definições que se aplicam a um site específico. Os conjuntos incluem segurança de SIP, configurações de trava do dispositivo, entre outros.
