# <a name="mobility-support-in-lync-server-2013"></a>Suporte à mobilidade no Lync Server 2013

_**Última modificação do tópico:** 2012-06-29_

Com o Lync Server 2013 Mobility, os usuários têm acesso à funcionalidade do Lync 2013 em seus dispositivos móveis. Os dispositivos móveis suportados incluem os seguintes:

  - Dispositivos Apple iOS suportados

  - Windows Phone

  - Android

  - Nokia

Para obter detalhes sobre os recursos e requisitos de mobilidade, consulte [Planning for Mobility in Lync Server 2013](lync-server-2013-planning-for-mobility.md) na documentação de planejamento.
