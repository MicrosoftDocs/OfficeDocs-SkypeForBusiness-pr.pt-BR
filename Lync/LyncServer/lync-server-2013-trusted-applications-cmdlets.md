# <a name="trusted-applications-cmdlets-in-lync-server-2013"></a>Cmdlets de aplicativos confiáveis no Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**Última modificação do tópico:** 2012-06-20_

Um aplicativo confiável é um aplicativo desenvolvido por um terceiro que recebe status confiável para ser executado como parte do Microsoft Lync Server 2013, mas que não é uma parte interna do produto. O Lync Server 2013 fornece cmdlets que podem ser usados para configurar e gerenciar aplicativos confiáveis.

<div>
