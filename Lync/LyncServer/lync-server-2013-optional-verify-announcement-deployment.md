# <a name="optional-verify-announcement-deployment-in-lync-server-2013"></a>Opcion Verificar a implantação do comunicado no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2013-02-25_

Após instalar e configurar o Comunicado, você precisará verificar a configuração para garantir que as chamadas a números não atribuídos funcionem conforme o esperado. No mínimo, verifique o seguinte:

  - Ligue para um número válido para sua organização, mas é um número não atribuído.

  - Ligue para o número não atribuído e verifique se o comunicado correto foi reproduzido.
