# Planejando Arquivamento no Lync Server 2013

 

_**Tópico modificado em:** 2012-09-28_

As corporações e outras organizações estão sujeitas à crescente quantidade de reguylamentações do setor e governamentais que requerem a retenção de tipos específicos de comunicação. Se sua organização tem requisitos desse tipo, você pode usar a função de arquivamento em Lync Server 2013 para arquivar comunicações por mensagens instantâneas (IM) e conferências (reuniões) enviadas pelo Lync Server para ajudar a oferecer suporte a alguns de seus requisitos de conformidade.
