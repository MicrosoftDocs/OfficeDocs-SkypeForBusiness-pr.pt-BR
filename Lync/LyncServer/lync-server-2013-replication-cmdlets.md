# <a name="replication-cmdlets-in-lync-server-2013"></a>Cmdlets de replicação no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-10-09_

Os cmdlets de replicação fornecem uma maneira de monitorar e gerenciar a replicação do Lync Server. Você pode usar esses cmdlets para definir as configurações de replicação; para monitorar o progresso da replicação; e forçar manualmente a replicação em um servidor.

<div>

## <a name="replication-cmdlets"></a>Replication Cmdlets

Veja a seguir uma lista de cmdlets relacionados diretamente ao gerenciamento de replicação:

**CRR**

  - <span></span>  
    [Debug-CsInterPoolReplication](https://technet.microsoft.com/library/JJ619185(v=OCS.15))

<!-- end list -->

  - <span></span>
