# <a name="new-monitoring-features-in-lync-server-2013"></a>Novos recursos de monitoramento no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-11-08_

O Lync Server 2013 não tem uma função de servidor de monitoramento separada. Monitoramento é um recurso opcional disponível em todos os Servidores Front-End em uma implantação de Enterprise Edition, e em servidores Standard Edition, que pode ser implementado e configurado para um pool ou site. Novo no Lync Server 2013, você pode habilitar o espelhamento de banco de dados do SQL Server para seu banco de dados de monitoramento.
