# <a name="verifying-backup-prerequisites-in-lync-server-2013"></a>Verificando pré-requisitos de backup no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2013-02-17_

Antes de começar a fazer backup do Lync Server, verifique se você está preparado com o seguinte:

  - Ferramentas de backup. Para obter detalhes, consulte [requisitos de backup e restauração no Lync Server 2013: ferramentas e permissões](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md).

  - Elas. Para obter detalhes, consulte [requisitos de backup e restauração no Lync Server 2013: ferramentas e permissões](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md).

  - Local para armazenar backups. Para obter detalhes, consulte [Configurando um local de backup para o Lync Server 2013](lync-server-2013-setting-up-a-backup-location.md).

  - Mídia para os backups. Para obter detalhes, consulte [Configurando um local de backup para o Lync Server 2013](lync-server-2013-setting-up-a-backup-location.md).
