# No Lync Server 2013, adicionar autenticação Kerberos a outros sites

 

_**Tópico modificado em:** 2010-11-03_

Se você precisar adicionar locais a uma implantação existente já configurada para a autenticação Kerberos, poderá usar uma conta de autenticação Kerberos existente para o novo local ou criar uma nova conta. Para obter detalhes de como criar uma nova conta para um local, consulte [Criar uma conta de autenticação Kerberos no Lync Server 2013](lync-server-2013-create-a-kerberos-authentication-account.md). Para obter detalhes de como usar a mesma conta para um novo local atribuindo uma conta existente a esse local, consulte [Atribuir uma conta de autenticação Kerberos a um site no Lync Server 2013](lync-server-2013-assign-a-kerberos-authentication-account-to-a-site.md).
