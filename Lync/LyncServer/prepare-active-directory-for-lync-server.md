# Preparar o Active Directory para Lync Server

 

_**Tópico modificado em:** 2012-09-17_

Antes de implantar o Lync Server 2013 em um estado de coexistência com o Lync Server 2010, você deve executar algumas tarefas adicionais do Active Directory para configurar o esquema, a floresta e o domínio do Lync Server 2013. As extensões do esquema adicionam classes e atributos ao Active Directory que são exigidos pelo Lync Server 2013. Para obter informações adicionais, consulte o tópico [Preparando Serviços de Domínio do Active Directory para o Lync Server 2013](lync-server-2013-preparing-active-directory-domain-services.md).

**Para preparar o Active Directory para o Lync Server 2013**

1.  No servidor de front-end do Lync Server 2013, execute a Instalação do Lync Server 2013.

2.  Selecione **Preparar Active Directory** .
    
    ![Assistente de Implementação do Lync Server 2013, página de Boas-vindas](images/JJ205265.5f88ae18-9c3c-42ea-a91a-836ecf5d515f(OCS.15).jpg "Assistente de Implementação do Lync Server 2013, página de Boas-vindas")

3.  Conclua as etapas 1 a 5.
    
    ![Assistente de Implantação, Preparação do Active Directory](images/JJ205265.eddd9e94-fa70-453f-8810-b99a2bf0844a(OCS.15).jpg "Assistente de Implantação, Preparação do Active Directory")
