# <a name="prepare-active-directory-for-lync-server"></a>Preparar o Active Directory para Lync Server

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-09-17_

Antes de implantar o Lync Server 2013 em um estado de coexistência com o Lync Server 2010, você deve realizar algumas tarefas adicionais do Active Directory para configurar o esquema, a floresta e o domínio para o Lync Server 2013. As extensões de esquema adicionam as classes e os atributos do Active Directory exigidos pelo Lync Server 2013. Para obter mais informações, consulte o tópico [preparando os serviços de domínio do Active Directory para o Lync Server 2013](lync-server-2013-preparing-active-directory-domain-services.md).

**Para preparar o Active Directory para o Lync Server 2013**

1.  No servidor front-end do Lync Server 2013, execute a instalação do Lync Server 2013.

2.  Selecione **Preparar Active Directory**.
    
    ![Assistente de implantação do Lync Server 2013, página de boas-vindas](images/JJ205265.5f88ae18-9c3c-42ea-a91a-836ecf5d515f(OCS.15).jpg "Assistente de implantação do Lync Server 2013, página de boas-vindas")

3.  Conclua as etapas de 1 a 5.
    
    ![Assistente de implantação, Active Directory Prearation](images/JJ205265.eddd9e94-fa70-453f-8810-b99a2bf0844a(OCS.15).jpg "Assistente de implantação, Active Directory Prearation")
