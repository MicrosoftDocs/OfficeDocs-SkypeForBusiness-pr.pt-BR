# <a name="voice-normalization-rules-cmdlets-in-lync-server-2013"></a>Cmdlets de regras de normalização de voz no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-06-20_

As regras de normalização de voz são usadas para converter um requisito de discagem de telefone (por exemplo, discar 9 para acessar uma linha externa) para o formato de número de telefone E. 164 usado pelo Microsoft Lync Server 2013.

<div>

## <a name="voice-normalization-rules-cmdlets"></a>Cmdlets de regras de normalização de voz

Os cmdlets a seguir podem ser usados para gerenciar regras de normalização de voz.

**Regras de Normalização de Voz**

  - <span></span>  
    [Get-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg398393(v=OCS.15))

  - <span></span>  
    [New-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg398240(v=OCS.15))

  - <span></span>  
    [Remove-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg398501(v=OCS.15))

  - <span></span>  
    [Set-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg398491(v=OCS.15))

  - <span></span>  
    [Test-CsVoiceNormalizationRule](https://technet.microsoft.com/library/Gg399003(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [New-CsVoiceRegex](https://technet.microsoft.com/library/Gg412751(v=OCS.15))
