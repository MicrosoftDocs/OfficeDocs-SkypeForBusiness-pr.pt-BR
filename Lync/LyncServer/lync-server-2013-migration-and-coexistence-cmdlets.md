_**Tópico da última modificação:** 2012-06-26_

O cmdlet [move-CsLegacyUser](https://technet.microsoft.com/en-us/library/Gg413025(v=OCS.15)) fornece uma maneira de mover contas de usuário do Office Communications Server 2007 ou do Microsoft lync Server 2010 para o Microsoft lync Server 2013. Se você precisar mover uma conta de usuário "voltar" (por exemplo, do Microsoft Lync Server 2013 para o Microsoft Lync Server 2010), use o cmdlet [move-CsUser](https://technet.microsoft.com/en-us/library/Gg398528(v=OCS.15)) .
