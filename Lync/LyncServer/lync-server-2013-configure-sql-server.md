# <a name="configure-sql-server-in-lync-server-2013"></a>Configurar o SQL Server no Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<span> </span>

_**Última modificação do tópico:** 2015-01-22_

Para cada banco de dados implantado, você pode usar uma única instância do SQL Server para todos os bancos de dados da implantação do Lync Server 2013 que podem ser colocados em um servidor de banco de dados. Para obter detalhes sobre a colocação de banco de dados, consulte [supported Server em colocação no Lync server 2013](lync-server-2013-supported-server-collocation.md) na documentação de suporte.

Além disso, cada instância do SQL Server deve estar instalada e disponível antes de concluir as etapas no construtor de topologias que definem os bancos de dados ou criar manualmente os bancos de dados com os cmdlets do Windows PowerShell. Para obter detalhes sobre a capacidade de suporte do SQL Server, confira [configuração de hardware para o Lync Server 2013](lync-server-2013-hardware-setup.md).
