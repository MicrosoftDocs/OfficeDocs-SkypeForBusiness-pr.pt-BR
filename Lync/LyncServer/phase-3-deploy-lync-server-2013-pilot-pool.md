# <a name="phase-3-deploy-lync-server-2013-pilot-pool"></a>Fase 3: implantar o pool piloto do Lync Server 2013

_**Última modificação do tópico:** 2012-10-19_

Esta seção aborda as etapas necessárias para implantar um pool piloto do Lync Server 2013. A implantação do Lync Server 2013 requer o uso do construtor de topologias para definir sua topologia e os componentes que você deseja implantar, preparando seu ambiente para implantação dos componentes do Lync Server 2013, publicando seu design de topologia no primeiro servidor front-end e, em seguida, instalando e Configurando o software Lync Server 2013 para os componentes para sua implantação. Quando a instalação for concluída, a implantação do pool piloto do Lync Server 2013 coexistirá com um pool existente do Lync Server 2010.
