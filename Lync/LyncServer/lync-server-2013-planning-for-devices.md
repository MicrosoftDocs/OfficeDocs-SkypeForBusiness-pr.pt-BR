# <a name="planning-for-devices-in-lync-server-2013"></a>Planejamento de dispositivos no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2013-02-28_

O Lync Server 2013 inclui o Lync Phone Edition, software que é executado em dispositivos qualificados e fornece recursos de telefonia tradicionais e avançados, segurança integrada, capacidade de gerenciamento e muito mais. O Lync Phone Edition funciona da mesma maneira que o Lync Server 2013, como no Lync Server 2010. Para obter detalhes sobre o planejamento de dispositivos, consulte [Planning for Devices](http://go.microsoft.com/fwlink/p/?linkid=285880) na Biblioteca TechNet do Lync Server 2010.
