# <a name="reviewing-the-dns-report-in-lync-server-2013"></a>Examinando o relatório de DNS no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2013-02-21_

O relatório DNS, que faz parte do relatório do administrador, detalha todas as entradas recomendadas e conhecidas para o DNS (sistema de nomes de domínio) nas redes internas, de perímetro e externas. Se o designer tiver concluído as edições no diagrama de rede e todos os endereços IP e FQDNs (nomes de domínio totalmente qualificados) forem definidos para seus valores de produção, o relatório DNS fornecerá um recurso de configuração excelente. Este relatório também pode servir como um documento de solução de problemas operacional.

![Relatório de administração de DNS](images/Gg558641.9dd1e810-ddc7-4816-a806-4239baf9ec51(OCS.15).jpg "Relatório de administração de DNS")

Você deve ter sua equipe de gerenciamento de DNS para verificar completamente o relatório de DNS para garantir que não haja erros que possam causar dificuldade durante a implantação ou que possam complicar uma sessão de solução de problemas.
