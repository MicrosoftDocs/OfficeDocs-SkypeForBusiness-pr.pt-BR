# <a name="whats-new-for-devices-in-lync-server-2013"></a>O que há de novo para dispositivos no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-06-22_

O Lync Server 2013 inclui o Lync Phone Edition, software que é executado em dispositivos qualificados e fornece recursos de telefonia tradicionais e avançados, segurança integrada, capacidade de gerenciamento e muito mais. O Lync Phone Edition funciona da mesma maneira que o Lync Server 2013, como no Lync Server 2010. Para obter detalhes sobre os recursos mais recentes relacionados a dispositivos, consulte [What ' s New for Devices](https://go.microsoft.com/fwlink/p/?linkid=256490) in the Lync Server 2010 TechNet library.
