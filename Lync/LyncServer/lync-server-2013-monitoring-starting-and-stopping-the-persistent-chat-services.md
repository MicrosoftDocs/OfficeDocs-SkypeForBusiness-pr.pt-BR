# <a name="monitoring-starting-and-stopping-the-persistent-chat-services-in-lync-server-2013"></a>Monitorar, iniciar e interromper os serviços de chat persistente no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-10-06_

O serviços de chat persistente e os serviços de conformidade de chat persistente fazem parte da topologia do Lync Server 2013 e podem, portanto, ser monitorados, interrompidos e iniciados usando os cmdlets do Windows PowerShell, **Get-CsWindowsService**, **Stop-CsWindowsService**e **Start-CsWindowsService**, respectivamente.
