# Planejamento de monitoramento no Lync Server 2013

 

_**Tópico modificado em:** 2012-09-05_

O serviço de monitoramento no Microsoft Lync Server 2013 oferece uma forma dos administradores coletarem dados de uso, tendência e qualidade do serviço para as sessões de comunicação que ocorrem em sua organização. O monitoramento no Lync Server 2013 não exige mais uma função do servidor separada; ao invés disso, o serviço de monitoramento é integrado em cada servidor de Front-End. No entanto, por padrão, o monitoramento não é habilitado no Lync Server 2013. Este documento ajudará você a determinar se o monitoramento deve ou não ser habilitado na sua organização.
