# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Requisitos de certificado para o servidor de chat persistente no Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-10-03_

Para instalar o servidor de chat persistente, você deve ter um certificado emitido pela mesma autoridade de certificação usada pelos servidores internos do Lync Server 2013 para cada servidor que executa os serviços Web de chat persistente para carregamento/download de arquivos. Certifique-se de que você tem os certificados necessários antes de iniciar a instalação de chat persistente, especialmente se você estiver usando uma autoridade de certificação externa.
