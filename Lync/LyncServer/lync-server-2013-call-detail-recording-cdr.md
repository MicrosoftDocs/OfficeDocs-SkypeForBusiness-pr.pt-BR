# CDR (Gravação de Detalhes das Chamadas) no Lync Server 2013

 

_**Tópico modificado em:** 2012-10-22_

O CDR (registro de detalhes das chamadas) registra informações de uso e diagnóstico sobre atividades ponto a ponto, incluindo serviço de mensagens instantâneas, chamadas VoIP, compartilhamento de aplicativos, transferência de arquivos e reuniões. Os dados de uso podem ser utilizados para calcular o retorno sobre o investimento, enquanto os dados de diagnóstico podem ser usados para solucionar problemas de atividades ponto a ponto e reuniões. Quando você instalar o Lync Server 2013, também instalará um conjunto predefinido de configurações globais do CDR. Use os tópicos desta seção para configurar o CDR.
