# Fase 4: Mover usuários de teste para o pool piloto

 

_**Tópico modificado em:** 2012-09-26_

Você pode mover um único usuário ou grupos de usuários para sua nova implantação do Microsoft Lync Server 2013 utilizando os dois métodos a seguir: Painel de Controle do Lync Server e Shell de Gerenciamento do Lync Server. Os tópicos nesta seção descrevem as tarefas que você deve concluir durante a implantação piloto, assim como antes de mover sua implantação do Lync Server 2013 de uma implantação para uma implantação no nível de produção.
