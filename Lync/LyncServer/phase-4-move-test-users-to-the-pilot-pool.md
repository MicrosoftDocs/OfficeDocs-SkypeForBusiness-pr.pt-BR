# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>Fase 4: mover usuários de teste para o pool piloto

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificação do tópico:** 2012-09-26_

Você pode mover um único usuário ou grupos de usuários para sua nova implantação do Microsoft Lync Server 2013 usando os dois métodos a seguir: painel de controle do Lync Server e Shell de gerenciamento do Lync Server. Os tópicos desta seção descrevem tarefas que você deve concluir durante a implantação piloto, bem como antes de mover sua implantação do Lync Server 2013 de uma implantação piloto para uma implantação no nível de produção.
